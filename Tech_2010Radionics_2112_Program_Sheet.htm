  1. If successful all Zone lights except Zone 4 will turn ON

  2. Press 8 to enter program mode, ON and Zone 1 Led will turn ON

Program Steps 1. Move to Program Group (1 to 13) by pressing Right Panic key

2. Use Number keys to select Location in Group (1 to 8), Zone light shows location

3. To view a Location press Middle Panic key, number lights will flash Value

4. Press Left Panic key to modify Location, Zone light will flash, enter data

5. Press Left Panic key again to save Location, Zone light will stop flashing

  1. Use Number keys to select next Location or Right Panic for next Group

Delete Location Press CLEAR while in a Location

Hex Program While modifying a Location C+2=A, C+3=B, C+4=C, C+5=D, C+6=E, C+7=F

Exit Program Press CLEAR key, all Zone lights except 4 will turn ON, unclip Standby Switch

(System will now send a Reboot (R14) signal)
