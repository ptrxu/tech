DSC PC3000 v7.51 Program Sheet Binary Programming

Power Ratings 475mA Aux output maximum

50mA PGM power

400mA Siren output

Enter Program * 8 3000 (READY, ARMED and SYSTEM Lights should flash)

Exit Program #

Move to location Enter 2-digit location, enter data, # may be required to exit location

Default Panel 1. Power down then up, if board clicks panel is locked out (replace board)

2. If panel does not click leave power ON, short default pins and enter * 8 0000

3. Keypad should show TROUBLE, BYPASS and MEMORY lights flashing

4. Remove short from default pins and program normally

(This does not work on V7.7 3000's (Version info found on large main chip)
