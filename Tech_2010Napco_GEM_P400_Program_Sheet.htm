Napco GEM P400 Program Sheet

Power Ratings 750mA Aux output with 16.5V 20VA transformer

2.25A Siren output

Enter Program 1. FUNCTION + 8 (3 beeps)

2. 4567 + ON/OFF (3 beeps ARMED turns ON, Trouble symbol flashes)

(1 long beep = incorrect Installer Code)

(Will not allow entry into programming if system is reporting, armed or sounding siren)

Exit Program 1. RESET + FUNCTION + 8 (3 beeps)

2. Installer Code + ON/OFF

Program a 1. RESET (3 beeps)

Location 2. 2-digit location (3 beeps, ARMED light turns OFF, READY light turns ON)

3. Enter new Data

4. RESET + next 2-digit location

Clear Location INSTANT

Scroll Location INTERIOR moves within a location like phone number and account number

or will exit some locations (example: INTERIOR will exit 00 Entry/Exit Zones section)

Default Panel 1. Power system down, jumper terminal 15 (PGM) to terminal 3 (ZONE 1)

2. Power up after 10 seconds ARMED, SYSTEM flash

3. Power down, rewire back to normal then power up

(If this does not work system is locked out)

Hex Digits A (10) = *1 B (11) = *2 C (12) = *3 D (13) = *4 E (14) = *5 F (15) = *6
