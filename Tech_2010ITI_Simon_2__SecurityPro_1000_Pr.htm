Default Panel Hold CANCEL, CLOCK SET and MINUTES while powering up (will not work if Phone Lock is ON)

Add Sensor 1. Press ADD under START MENU

2. Press SENSOR/REMOTE until you hear correct Voice Description

(Press OPTION if North, South, East and West is needed)

3. Press DONE, enter 2-digit Group

Common Sensor Groups ---> 01 = Police Panic

10 = Entry Exit

13 = Instant

17 = Interior Follower (Motions)

20 = Delayed Interior Follower

26 = Fire

4. Learn Sensor by the following

Wireless door/window: Remove cover or press "Learn" button

(If hardwired sensor on wireless zone learn while zone is open)

Motion Detector: Trip tamper

Smoke Detector: Remove smoke from base

Keyfob: Press Lock and Unlock at same time

Touchpad: Press BYPASS or both EMERGENCY buttons

5. Keypad should say "Sensor ?? programmed OK"

Hardwire Zones on Panel require 4.7k resistor, not needed if connected to Wireless sensors

Zones Hardwire Zones can only be normally closed sensors

Zone A learned Sensor is automatically added to next available Zone Number

Assignments to assign different Zone enter 2-digit Zone Number AFTER entering Group Number

If learning Hardwire Contact on Wireless Sensor learn with Hardwire Contact OPEN

Delete Sensor Press DELETE + SENSOR/REMOTE until you hear item to delete then DONE

Enable 1. Press ADD button (under START MENU)

Function 2. Press OPTION button (Under MAIN MENU) + 2-Digit Program Location

3 Press DONE (May have to enter digits for Phone Numbers, Accounts etc)

Disable 1. Press DELETE button (under START MENU)

Function 2. Press OPTION button (Under MAIN MENU) + 2-Digit Program Location

Program Lights (Requires Lamp Module) CLICK HERE for Lamp Module

  1. Open Cover and enter Master Code

  2. Press ADD then press LIGHT CONTROL to assign House Code (A to F) then DONE

(This must match Lamp Module dials connected to Light)

  1. Press ADD then LIGHT CONTROL then UNIT# to assign Unit Number

(This must match Lamp Module dials as well)

Unit 1 to 8 most common, controlled by LIGHTS TIME or SENSOR ACTIVATED

Unit 9 to 16 are activated by All On or All Off commands (Keyfobs etc)

Unit 10 Flashes Lamps to Arming Level (1 Flash = Disarm, 2 = Stay, 3 = Away)

  1. Press SENSORS to pick sensor that will activate Lamp Module then DONE

  2. Close cover

Lights Entry/Exit (Requires Lamp Module) CLICK HERE for Lamp Module

Activated 1. Open Simon Cover, enter Master Code

2. Press ADD then LIGHT CONTROL to assign House Code (A to F) then DONE

3. Press ADD + LIGHT CONTROL + UNIT# to assign Unit Number

4. Press ENTRY/EXIT DELAY

5. Close Cover

  1. Set dials on Lamp Module to House Code (Letter Dial) and Unit# (Number Dial)

(Lights turn ON for 4 minutes during Entry and Exit Delay)

Lights Time (Requires Lamp Module) CLICK HERE for Lamp Module

Activated 1. Open Simon Cover, enter Master Code

2. Press ADD then LIGHT CONTROL to assign House Code (A to F) then DONE

3. Press ADD + LIGHT CONTROL + UNIT# to assign Unit Number

4. Press TIME and select Start time with HOURS and MINUTES then DONE

5. Press HOURS and MINUTES to set End time then DONE

6. Close cover

  1. Set dials on Lamp Module to House Code (Letter Dial) and Unit# (Number Dial)

(Activate by pressing LIGHTS TIME ACTIVATED button)

Siren Setup No programming necessary
