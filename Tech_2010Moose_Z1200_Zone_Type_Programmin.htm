Moose Z1200 Zone Type Programming

Digit 1 Zone Type

0 = Null (Not used)

  1. = Burglary

  2. = Fire 24 Hour

  3. = Holdup 24 Hour

  4. = Auxiliary 24 Hour

  5. = Keyswitch (Momentary only)

  6. = CCM Communicator Report 24 Hour

  7. = Tamper (24 Hour Burglary)

  8. = Burglary Chime (Chime always on regardless of panel)

  9. = Verified Fire (Report if 2 alarms within 2 minute window)

  10. (A) = Fire Supervisory

  11. (B) = Activate Universal Output (Use Digit 2 to specify which Output activates)

  12. (C) = Universal Logged

  13. to 15 = For Future Use

Digit 2 Zone Response

0 = Perimeter Instant (Activates Output 0 if Digit 1 = B)

  1. = Perimeter Delay 1 (Activates Output 1 if Digit 1 = B)

  2. = Perimeter Delay 2

  3. = Interior Instant (Bypassed in Stay Mode)

  4. = Interior Follower (Bypassed in Stay Mode)

  5. = Interior Nite (Bypasses when armed in Stay or Nite Mode)

  6. = Interior Delay 1

  7. = Interior Delay 2

Digit 3 Bypassing

0 = Not Bypassable (Fire Zones)

  1. = Bypassable

Digit 4 Troubles/Supervision

0 = Supervised, No Trouble Output (Use for standard Burg Zones)

  1. = 24 Hr Trouble on Open/Alarm on Short (Use for Fire/Smoke Detectors)

  2. = 24 Hr Trouble on Short/Alarm on Open

  3. = Burg Day Alert on Open/Alarm on Open/Short if Armed

  4. = Burg Day Alert on Short/Alarm on Open/Short if Armed

  5. = Non-Supervised (Closed Circuit contacts only, no EOL)

  6. = Non-Supervised (Open Circuit contacts only, no EOL)

  7. = Two-Wire Smoke (Zone 12 only)
