Ademco 5800RP Wireless Repeater Program Guide

Power 9VAC or +12VDC (Non polorized)

Current draw 80ma

Installation (If not using 5883 Receiver including Lynx)

  1. Attach AC and Battery

  2. Leave all dip switches OFF

  3. Thats it!

(If using a 5883 Receiver)

  1. Leave 5800RP powered down

  2. Set Dip switch 1 on 5800RP to ON

  3. Place panel into RF Go-No-Go RF mode

Vista-10/15P/20/20P Installer Code + # + 4

Vista-40/50/100/128 Installer Code + 5

  1. Power up 5800RP, Green and Red LED's should turn ON

  2. Within 1 minute of powerup press and release tamper switch

  3. LED's should flash, Red LED should turn OFF indicating Site ID is learned

  4. Set dip switch 1 to OFF

  5. End Go-No Go test in panel with User Code + 1 (Disarm)

Supervision 1. Dip Switch 2 must be OFF

(Non UL) 2. Learn repeater as wireless sensor type 08, RF type 3, Loop 1

3. Press and release 5800RP tamper to learn

Testing 1. Place 5800RP at desired location, do not permanently mount it yet

2. Remove 5800RP cover, this enters GO/NO-GO test mode and reduces 5800RP signal strength 50%

3. Place panel into GO/NO-GO test mode, this reduces panel signal strength 50%

Lynx Installer Code + # + 8

Vista-10/15P/20/20P Installer Code + # + 4

Vista-40/50/100/128 Installer Code + 5

4. Panel will display a list of all zones

5. Activate each zone (do not activate by tamper)

6. Panel will beep and remove that zone from list

7. End GO/NO-GO test Installer Code + 1

8. Place cover back on 5800RP
