Programming 1. Set dipswitches 4 to 8 in siren to match panel House ID

2. Plug siren into outlet

3. Program a zone in panel as

Zone Type 05 (Trouble by day/Alarm by night)

Input Type 03 (Supervised RF transmitter)

Loop 01

4. Enter serial number from siren or plug siren in, within 15 seconds place magnet by

tamper location until Yellow LED turns off then remove magnet, Yellow LED will flash

LED Indicators Green LED ON = AC power OK

OFF = No AC power

Yellow LED Blink = Wireless RF transmission

ON = RF test

Red LED ON = System Armed or if on constantly bad RF reception (move siren)

OFF = System Disarmed
