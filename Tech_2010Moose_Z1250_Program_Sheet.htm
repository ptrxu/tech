Moose Z1250/Airtech Discovery 3000 Program Sheet

Keypads have 3 function keys below Display, each selects what is displayed above

Enter Program 1. Press MENU (blue triangle key) until Display reads PROG OPTIONS

2. Press function key below "PROG OPTIONS"

3. Enter Installer Code (default is 96321) then press function key under "ENTER"

4. Display MAY briefly display one of these screens

5. Then will enter programming

Exit Program * twice

Program Info Menu driven, display will show M1 for Menu 1, M2 for Menu 2...

Press function key below the following options to activate

PREV moves to previous menu or sub-menu

NEXT moves to next menu or sub-menu

SELECT enters displayed menu

STORE stores data

----> (if displayed) moves to additional data or enters alpha characters

and will appear in for User or Zone alpha text programming

  • moves up to previous menu

  • twice exits programming
