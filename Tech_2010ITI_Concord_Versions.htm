GE/ITI Concord Versions

Information 1. Concord RF which has an integrated receiver built in (most common)

  1. Concord Hybrid which requires an external 60-764-95R RF receiver

Version Info Concord v1 = 1. Could not use 60-820 fixed-english keypad, used 60-728 LED keypad instead

2. All keypads must be addressed with serial number on keypad sticker

Concord v2 = 1. Can use 60-820 fixed-english keypad, 60-728 LED keypad discontinued

2. All keypad/modules auto address on bootup

3. Added Exit Extension, Swinger Shutdown, Recent Closing

Concord v3 = 1. 230 User Codes instead of 59

2. 96 zones instead of 76

3. Event buffer holds 250 events instead of 100

Concord v4 = 1. 6 Partitions instead of 2

2. 16.5 VAC transformer instead of 24 VAC

3. Changes to SIA and CID to add silent police panic reporting

Version Info V1 to V3 = Sticker on main processor chip (Software Label)

V4 and up = Serial Number sticker on board

or

All versions = Factory Code number in user programming

1. Enter user programming with 9 + Master/Installer Code

2. Press A or B key until keypad displays "System Version" then #

  1. Keypad will display "Factory Code" then a 7-digit number
