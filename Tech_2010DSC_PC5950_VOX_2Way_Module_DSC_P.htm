Remote Access (06 Answer Machine Bypass must be set to 01 or 02 or 07 can be programmed for ring count)

  1. Call home, wait 1 or 2 rings then hang up

  2. Call again, PC5950 should beep

  3. Enter User Code, PC5950 should beep 3 times, 1 beep = error

  4. To access 2-way press * then use following commands

  1. = Talk

  2. = VOX Talk and Listen

  3. = Listen

7 = Stay online additional 90 seconds

99 = Disconnect
