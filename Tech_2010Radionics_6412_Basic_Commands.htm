Radionics D6412 Basic Commands

Zones 8 onboard hardwire, up to 40 total

Battery 12 volt rechargeable Gel-Cell

AC Transformer is 18 VAC 20 VA

Partitions 4

Wireless Yes

Downloadable Yes

Panics A B C keys, panics set by Installer

Arm All (Away) 1. Press ALL ON key (CMD should turn ON)

2. if CMD flashes enter User code (All On, Status and CMD should turn ON)

Arm Part (Stay) 1. Press PART ON key (CMD should turn ON)

2. If CMD flashes enter User code (Part On, Status and CMD should turn ON)

Instant Arm 1. Press PART ON or ALL ON key (CMD should turn ON)

(Stay or Away) 2. If CMD flashes enter User code (Part On or All On, Status and CMD should turn ON)

3. Press NO DELAY key

Quick Arm Cmd + 1 = All ON (Away)

CMD + 2 = Part ON (Stay)

Force Away Arm 1. Hold ALL ON for 2 seconds (CMD and any open zone LED's should turn ON)

2. Press ALL ON or again (All On LED should turn ON)

3. If CMD flashes enter User code

Force Stay Arm 1. Hold PART ON for 2 seconds (CMD and any open zone LED's should turn ON)

2. Press PART ON or again (Part On LED should turn ON)

3. If CMD flashes enter User code

Silent Away Arm 1. Press Cmd + 1

2. Hold 1 key for 2 or 3 seconds

3. If CMD flashes enter User code

Silent Stay Arm 1. Press Cmd + 2

2. Hold 2 key for 2 or 3 seconds

3. If CMD flashes enter User code

Arm Partitions 1. Press CMD + 8 + 0 (User Code must have proper authority)

2. If CMD flashes enter User code

Disarm Enter User code

Disarm Partitions 1. Press CMD + 8 + 1 (User Code must have proper authority)

2. If CMD flashes enter User code

Silence Alarm Enter User code

Reset Alarm 1. Press CMD + 4 + 7

2. If CMD flashes enter User code

Reset Smoke 1. Press CMD + 4 + 7

2. If CMD flashes enter User code

Alarm Memory 1. Press CMD + 4 + 0

2. IF CMD light flashes enter User code

3. Keypad will display last zone(s) in memory

Zone flashing fast = Alarm signal was sent

Zone flashing slow = Alarm signal was aborted

4. Press CMD to exit

5. To clear arm and disarm system

Bypass Zones 1. Press BYPASS key

2. If CMD light flashes enter User code

3. Enter 2-digit zone to bypass + * (01 to 40)

4. Enter next zone or CMD key to exit

Un-Bypass Arm then disarm system or repeat above

Door Chime 1. Press CMD + 6 + 1

On/Off 2. If CMD light flashes enter User code

3. If Chime is OFF Zone 1 will flash, press 1 to turn Chime ON

If Chime is ON Zone 2 will flash, press 2 to turn Chime OFF

Door Chime 1. Press CMD + 6 + 2

Tone 2. If CMD light flashes enter User code

3. Keypad beeps and displays current Chime

Zone LED 1 = No Tone (Off)

Zone LED 2 = Short Beep (Default)

Zone LED 3 = Beep until keypress

Zone LED 4 = Beep until closed

4. Press 1, 2, 3 or 4 from above list to select new tone

  1. = No Tone (Off)

  2. = Short Beep (Default)

  3. = Beep until keypress

  4. = Beep until closed

  5. Press CMD to exit

Door Chime 1. Press CMD + 6 + 3

Zone Select 2. If CMD light flashes enter User Code

3. Zone LED's show current Chime zones

LED ON = Chime is ON for that zone

LED FLASHING = Chime can be enabled for that zone

4. Enter Zone to enable/disable chime (1 to 40) then *

5. Enter next zone then * or CMD to exit

Sensor Test 1. Press CMD + 4 + 4

2. If CMD light flashes enter User code (Keypad beep, all Zones LED's will turn ON)

3. Open/activate each zone (Keypad will beep, that Zone LED will turn OFF)

4. Exit test by pressing CMD key (Test ends when STATUS LED stops flashing)

System Test 1. Press CMD + 4 + 1

2. If CMD light flashes enter User code

3. System will test four ways, Zone LED indicate testing in progress in the order below

Press * to skip to next test if needed

Zone 1 LED = Testing Siren output LED ON = Test completed

Zone 2 LED = Strobe output LED Slow Flash = Test underway

Zone 3 LED = Panel Battery LED Fast Flash = Test failed

Zone 4 LED = Phone test

4. To exit press CMD key

Keypad Volume 1. Press CMD + 4 + 9

(If allowed) 2. If needed enter User code

3. Hold * then 1 to increase or 4 to decrease

4. Press CMD to exit

Time/Date 1. Press CMD + 4 + 5

(If allowed) 2. If CMD flashes enter User Code (Zone 1 LED should turn ON)

3. Enter 2-digit Month (01 to 12)

4. Enter 2-digit Day (01 to 31)

5. Enter 2-digit Year (00 to 99)

  1. Enter 2-digit Hour (00 to 23)

  2. Enter 2-digit Minute (00 to 59)

  3. Keypad will beep and exit
