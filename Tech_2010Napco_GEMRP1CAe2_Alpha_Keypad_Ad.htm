Napco GEM-RP1CAe2 Alpha Keypad Address/Quick Program Sheet

Programming of system can only be done by Keypad 1

Keypad Program 1. Move Main Panel JP1 Jumper from Pins 1 & 2 to Pins 2 & 3

2. After 15 seconds keypad will display "XX OUT OF SYSTEM" (XX = current keypad address)

3. Press 11123 then FUNCTION (Keypad shows "Keypad Beep ON")

KEYPAD BEEP ON (Press ON/OFF to toggle ON or OFF, FUNCTION for next)

ENTRY SOUNDER ON (Press ON/OFF to toggle ON or OFF, FUNCTION for next)

KEYPAD ADDRESS XX (Enter 01 to 07 then ON/OFF to save, FUNCTION for next)

Keypad 01 can enter panel programming, 02 to 07 cannot

Addresses must be consecutive 01, 02 03 etc, cannot have missing addresses)

NEW COMPAT# 000 (Press FUNCTION for next, do not alter this options!)

EZM ADDRESS 01 (01 to 06 (Keypad Zones), if not used FUNCTION for next)

ZONE RESPONSE 00 (Press ON/OFF to toggle 750ms or 50ms, usually leave alone)

PROG CTRL MSG 1 (Press FUNCTION to bypass, do not alter this!)

4. Press FUNCTION to loop back to start or RESET to exit

5. Move JP1 Jumper back to pins 1 & 2

Enter Program GEM-P816/1632 Installer Code by default is 456789

GEM-P3200/9600 Installer Code is printed on main processor chip

  1. Installer Code + FUNCTION

  2. Press INSTANT or AWAY until "Activate Program Y/N"

  3. Press INTERIOR or STAY to enter "Easy Menu Programming

  4. When finished press RESET twice to exit or RESET once for "Advanced Programming"

Exit Program Press RESET twice

Programming INTERIOR moves forwards

Keys INSTANT moves backwards

FUNCTION toggles between 4-digit location and 2-digit data

ON/OFF saves

RESET twice exits

0 Blanks a location (unless otherwise noted)

  • + 0 = 0

  • + 1 = B

  • + 2 = C

  • + 3 = D

  • + 4 = E

  • + 5 = F

Enter User Codes (enter 0 to 9, * 0 = Blank Space)

(Codes can be set for either 4 or 6 digits)

  1. Press FUNCTION or MENU to set cursor under User Number (U01)

  2. Enter User Number to program (Example to program User 02 enter 02)

  3. Press FUNCTION or MENU to move cursor under User Code

2. Enter new User Code

3. Press FUNCTION or MENU to move to Area 1 Options and enter new option

  1. Enter new 2-digit Option (Optional)

_ _ = Code Disabled (press * 0 = Blank Space)

_ 1 = Arm/Disarm

_ 2 = Arm Only

_ 3 = Service Code

_ 4 = Access Code

Add 8 to above to allow User Programming

Example: Code of 2222 with Arm/Disarm only would be 2222 BLANK 1

Press ON/OFF to save and move to next User Code or INTERIOR for next display

RF Transmitter Points

  1. Press FUNCTION to set cursor, enter Zone number

  2. Press BYPASS to enter Enroll Mode, LED's will begin flashing

  3. Enter Zone to program (09, 10 etc)

  4. Open Zone loop to program and install the battery

  5. Keypad should beep once indicating learned

  6. Move to next sensor or loop (Door sensors have 2 loops and can be 2 Zones)

  7. Press ON/OFF to save, INTERIOR for next

Keyfob Transmitters (system can use 8 keyfobs, these do not use zones)

  1. Enter Keyfob to program (01 to 08)

  2. Enter Area (Partition) is in (1, 2 or 0 to disable)

2. Enter 6-digit serial number on sticker of keyfob

  1. Enter 1-digit checksum on sticker of keyfob

  2. Enter Aux 1 (Bottom Left Button) 0 = None/Keyfob Disabled

  3. Enter Aux 2 (Bottom Right Button) 1 = Relay Group 1 Toggle

  4. Press ON/OFF to save, INTERIOR for next 2 = Relay Group 2 Toggle

(Arm/Disarm buttons are auto learned) 9 = Keypad Panic
