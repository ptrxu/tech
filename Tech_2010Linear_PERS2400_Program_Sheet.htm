Linear PERS2400 Program Sheet

Interference MAY show receiver failure if less than 20' from 2.4 Gigahertz cordless phone bases

Format Contact ID or 4/2

2-Way Info 2-Way requires Linear VB-4 board that plugs into back of unit.

All Programming is done by Downloading, all the tech can do is add sensors

RA2400 Program 1. Connect Phone Line from Modem to PERS2400

2. Press Test Button, keypad should sound 4 beeps

3. Set RA2400 to connect then press EMERGENCY button for 5 seconds

Compatible DXS-21 & DXS-23 Remote DXS-71 Smoke Detector

Wireless DXS-31 Door/Window DXT-41 & DXT-42 2 Keyfobs

Sensors DSX-62 & DXS-63 Wrist Panics DXT-61A Remote Panic

DXS-54 Motion Detector

Add Sensors 1. Hold CLEAR button for 5 seconds, keypad should beep 5 times

(by Customer) 2. Activate Transmitter, keypad should sound 1 beep indicating sensor learned

(If keypad sounds 1 long beep = Sensor already learned or system full, 16 Max)

Add Sensors 1. Press TEST button in back of Keypad, should hear 4 beeps

(by Installer) 2. Press TEST button again, should hear 5 beeps

3. Activate Transmitter, keypad should sound 1 beep indicating sensor learned

(If keypad sounds 1 long beep = Sensor already learned or system full, 16 Max)

Delete Sensors 1. Press TEST button in back of Keypad, should hear 4 beeps

2. Press and HOLD TEST button again, 6 beeps = All sensors are erased
