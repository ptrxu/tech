LCD Keypad Zone Programming

  1. Enter Installer Programming * 8 + Installer Code

  2. Press *

Press 001 for Zone 001 Description, 002 for Zone 002 etc

(To view all zone descriptions enter 01 to view then # to exit, select next zone etc)

Use < and > to move cursor left and right through label and use number keys to enter data

Keypad setup, Example: Press 1 to show A, press 1 again to show B etc...

  1. = A B C 1 3 = G H I 3 5 = M N O 5 7 = S T U 7 9 = Y Z 9 0

  2. = D E F 2 4 = J K L 4 6 = P Q R 6 8 = V W X 8 0 = Space

To toggle upper/lower case press * use < > to scroll to "CHANGE CASE" and press * again

To toggle special characters press * and use < > to scroll to "SPECIAL CHARACTERS" then press * again

Special Characters

032 = (Space) 037 = % 044 = , 059 = ; 064 = @ 125 = }

033 = ! 038 = & 045 = - 060 = < 093 = ]

034 = " 039 = ' 046 = . 061 = = 094 = ^

035 = # 042 = * 047 = / 062 = > 095 = -

036 = $ 043 = + 058 = : 063 = ? 123 = {

When finished press * use < > to scroll to "SAVE" and press * again, press # to exit
