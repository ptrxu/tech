Alarm.com GE Caddx NX-592E-GSM Module

Information Works with most GE/Caddx NX-4/NX-6/NX-8/NX-8E panels

(certain older v1 panels may not be compatible)

Yellow SIM card = AT&T

White SIM card = T-Mobile

Power Rating 65 mA in PowerSave mode

100 mA in Idle mode

1600 mA max in Connected mode

Mode Notes Module functions under 3 possible modes, troubleshooting must be done in Idle mode

  1. Idle = Normal, AC and battery ok, module is not currently in communication

  2. PowerSave = Module has just powered up or AC power is down or battery is less than 11.5 volts

Entering Phone Test (*4 or *44) switches from PowerSave to Idle mode

  1. Connected = Connected to Alarm.com servers, stays connected for 6 minutes after each message

Entering Installer Programming (*8 + 9713 + 0 + #) switches from Connected to Idle mode

Registration Send alarm or test signal (Test = *44 if enabled)

Diagnostic LED's These readings depend on what mode module is in (Idle, PowerSave or Connected)

LED 1 (Red) Error message in Idle and Connected mode, disabled in PowerSave mode

  1. flash = Module cannot communicate with panel

  2. flashes = SIM card missing (check SIM card holder)

  3. flashes = Error registering with GSM network

  4. flashes = Registered with GSM network but cannot communicate with Alarm.com

  5. flashes = Radio portion of module not working correctly

  6. flashes = Module is fixing an unusual condition with connection to GSM network

  7. flashes = Module cannot retrieve sensor list by installer locked programming

(Location 21 lights 5 and 6 are ON, must be turned OFF by downloading)

  1. flashes = Contact Alarm.com technical support

LED 2 (Yellow) Indicates communication between module and panel in all modes, flashes every

2 seconds in Idle mode or 4 seconds in PowerSave mode

LED 3 (Green) Communication between module and its radio unit and Alarm.com servers,

flashes in unison with LED 2 in PowerSave mode

LED 4 (Green) Signal Strength, flashes 1 to 5 for lowest to highest in Idle mode

disabled in PowerSave mode, flashes 2 seconds ON/OFF in Connected mode

Keypad Display GSM module can display it's information on any alpha keypad (after configuring, see below)

Zone 190 = What module is programmed to report

B = Phone Test M = Panel Programming

E = Alarms N = Tamper

F = System Trouble O = Cancels

G = Sensor Trouble P = Normal Activity

H = Arming/Disarming Q = Modem On Line

J = Sensor Bypass R = Pings (from Alarm.com)

K = AC Power Failure V = Panel Low Battery

L = Phone Failure

Zone 191 = SIM card number

Zone 192 = Module serial number (for registration)

To configure enter keypad zone description programming

  1. Press *92 then Installer Code (9713)

  2. Press 190#, display should show "Zone 190"

  3. Press STAY then * to save then # to exit, this adds a space before the word "Zone"

  4. Follow the same procedure for zones 191 and 192

4. Exit programming by pressing EXIT button

  1. Wait 2 minutes, re-enter keypad programming with *92 then Installer Code (9713)
