Information If House Code doesn't match panel keyfob will work but not give status lights or voice

Loop 4 must be learned to avoid CHECK 9 troubles

2 CR2430 3-Volt Lithium batteries

Set House Code 1. Program house into panel in *24

2. Press top 3 buttons on keyfob at same time, Red and Green LED's will flash

3. Within 5 seconds enter house code by pressing Loop 3 button for 10's digit, Loop 2 button for 1's digit

Example: to program house code 25 press Loop 3 twice, and Loop 2 button 5 times

4. Save by pressing Loop 1 button

5. Unit will flash 10's digit of House code on left LED, 1's digit on right LED and exit with beep

6. Learn each Button as seperate zone like standard 5804 keyfob

Note: Loop 4 must always be learned to avoid Check condition even if button is not used

View House Code Press top 3 buttons buttons at same time, press Bottom button, lights flash House Code

Vista-40 Notes Use Device Program Mode to enable RF Receiver and House Code

Typical Settings Loop 3 = Arm Away (type 21)

Loop 2 = Arm Stay (type 20)

Loop 4 = Disarm (type 22)

Loop 1 = Panic (type 07)

Volume Control 1. Press Loop 1, Loop 2, Loop 3 buttons at same time

(5804BDV only) 2. Within 5 seconds press Loop 3 = Low volume, Loop 2 = Medium or Loop 4 = Hi

3. Press Loop 1 button to save and exit
