AlarmNet 7845iGSM Internet/Cellular Backup

Information Sends using Internet as primary, if unavailable uses GSM Cellular

Cannot register the Internet component by AlarmNet Direct BUT the GSM cellular component can

be registered by AlarmNet direct first. The Internet component will then function when it connects to broadband

Specifications Transformer 9 to 16VAC

Initial units came with 9VAC 15VA (Honeywell part# 1332)

Later units come with 16.5VAC 40VA (Honeywell part# 1361)

Backup Battery 6V 3.1ah (Honeywell part# K14139)

Current Draw 35mA standby, 500mA during transmit

External Antenna GSM-ANT Indoor or GSM-ANT3dB Outdoor

Input Trigger V+ 2V to 13V

V- 0V to 1V

Compatibility Will send Contact ID if connected to compatible panel

Sends Ademco High Speed Express 4/2 if using siren triggers to non-compatible panel

Program 1. Call Dealer Tech Support (1-800-447-9239 ext: 2001) to request an account number

2. Internet installations may require 7720P field programmer

Wiring Original 7845iGSM's shipped with 9VAC 15VA transformer to power only the 7845iGSM

(Wiring method #2)

Later kits shipped with 16VAC 40VA transformer that can power both Vista panel and 7845iGSM

(Wiring method #1)

Do not attempt to power 7845iGSM by itself with 16.5VAC/40VA transformer

Do not attempt to power Vista panel and 7845iGSM by using standard 16.5VAC/25VA transformer

7720P Status SHIFT A = Display software revision

SHIFT B = 1st screen - Display MAC address (Press * for 2nd screen)

2nd screen - Display SCID SIM card information (Press * for 3rd screen)

3rd screen - Display IMEI ID number assigned to GSM (Press * to go back to 1st screen)

SHIFT C = Display current AlarmNet Time/Date

SHIFT D = 1st screen - Connection to Internet (Press * for 2nd screen)

GOOD = Physical link to Internet exists

BAD = Physical link does not exist

2nd screen - 7845iGSM's Network Address (Press * for 3rd screen)

This is the IP address assigned by the network

3rd screen - Display Network Subnet Mask (Press * for 4th screen)

4th screen - Display Network Gateway IP Address (Press * for 5th screen)

5th screen - Display Network DNS Server IP Address (Press * for 6th screen)

6th screen - Encryption Test (Press * for 7th screen)

7th screen - DHCP Status (Press * to go back to 1st screen)

DHCP OK = Network's router auto-assigns the IP address

DHCP OFF = IP address is assigned by Installer

SHIFT E = 1st screen - Display GSM Status (Press * for 2nd screen)

PriRSS = Signal strength in dB

GPRS - Y = GPRS service is available

N = GPRS service is not available

REG - N = Not registered

H = Registered home

S = Searching

D = Registration denied

R = Registered roaming

? = Unknown registration state

2nd screen - Display GSM Status 2 (Press * for 3rd screen)

Cntry = Country Code

Netw = Network Code

LAC = Local Area Code

3rd screen - Display GSM Status 3 (Press * for 4th screen)

Cell = Base Station ID

BaseST = Base Station Antenna Sector

Chan = Control Channel in use

4th screen - 2nd Site Signal Strength (Press * to go back to 1st screen)

SHIFT F = Network Gateway Tests

These must be OK to show link to AlarmNet

SHIFT S = 1st screen - System Status (Press * for 2nd screen)

(if ECP mode) 12345678 = Zone Status

  1. = New Event 5 = Normal X = Not Enabed

(if 4204 Emulation mode) Tm = Tamper Status

(if 2 4204 Emulation mode) P = Line Voltage

+ = OK - = Bad V = Bad, Not Reported

(if Zone Trigger mode) B = Battery Condition

+ = Battery OK

- = Battery below 5.7V

Flt = Radio Faults

OK = Normal

G = No GSM, Fault time expired

g = No GSM, Fault time has not expired

2nd screen - Battery Voltage (Press * for 3rd screen)

3rd screen - Battery Charger Condition (Press * for 4th screen)

4th screen - AC Line Voltage (Press * to go back to 1st screen)

* indicates below acceptable voltage

SHIFT T = Send Test message to AlarmNet

Sends a test message

SHIFT X = Reset CPU (reboot 7845iGSM)

Restarts 7845iGSM

SHIFT 0 = Force Server Update

Updates internal firmware if an update is available

SHIFT Up Arrow = Register 7845iGSM with AlarmNet
