FBI Omni 848 Program Sheet

Keypad Address The following keypads can be used on this system

OMNI-LCD Alpha keypad

OMNI-KP Fixed English keypad

XL7LC Alpha keypad

XK108 LED keypad

If LED keypad is used no other keypad type can be connected to panel

Keypad Address 1. Press CODE and STAY within 30 seconds of power-up

2. Enter address 1 to 8

3. Press * to exit

(By default panel has address 8 enabled for Alpha keypads)

XK108 LED keypads require no addressing

Enter Program 1. Press CODE key + * + 2468 + 1 (P9 appears on fixed english keypads)

2. Press 1 for System Programming menu

Press 2 for Zone Programming menu

Press 3 for RF Programming menu

Press 4 for Zone Description Programming menu

Change Program * + BYPASS + 1 = System Programming menu

Menu * + BYPASS + 2 = Zone Programming menu

  • + BYPASS + 3 = RF Programming menu

  • + BYPASS + 4 = Zone Description Programming menu

Installer Options Code + * + Installer Code + 1 = Installer Programming

Code + * + Installer Code + 2 = Review Programming

Code + * + Installer Code + 3 = Sensor Walk Test (with reduced RF sensitivity)

Code + * + Installer Code + 4 = System Log/Event Buffer

Code + * + Installer Code + 5 = Default System

Code + * + Installer Code + 6 = Clear Tamper Faults

Code + * + Installer Code + 8 = Unattended Downloading (calls downloader)

Code + * + Installer Code + 9 = Online Download (answers downloader)

Move to location Press * + 2 digit location

Next location #

Program location Press # after each digit and move to next location

Example: To program phone number enter 1#8#0#0# etc

Exit Program Press STAY

Default Panel Short "EEPROM RESET" pins on panel and power up

Hex Program CODE+1 = A

CODE+2 = B

CODE+3 = C

CODE+4 = D

CODE+5 = E

CODE+6 = F
