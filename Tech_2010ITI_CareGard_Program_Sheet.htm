Panel Info This is a medical alert system, door/motion sensors will act as a "No activity" sensor

Version info = Turn switch in back of CareGard off then on, panel will speak version

(other ITI sensors may work) 60-906-95 Watch panic

60-807-95R Motion detector

Transformer is hardwired into system

Enter Program 1. Remove front keypad cover (Keypad says "Use Red Number Keys to enter ID")

2. Press 4321 using Red Number Keys (Keypad says "Please select from Start Menu")

If 6-digit User Codes are enabled use 654321

If 5-digit User Codes are enabled use 54321

If 3-digit User Codes are enabled use 321

Exit Program Replace cover

Default Panel Hold CANCEL, CLOCK SET and MINUTES+ while powering up (will not work if Phone Lock is ON)

Add Sensor 1. Press ADD under START MENU "Select from Main Menu"

2. Press SENSOR "Press Button on Sensor"

3. Press tamper or program button to learn "Press button on sensor xx or cancel to quit"

Door/window: Remove cover or press "Learn" button

(Hardwire contacts on wireless sensor learn while zone is open)

Motion: Trip tamper

Smoke: Remove smoke from base

Keyfob: Press Lock and Unlock at same time

Watch panic Press panic button

4. Keypad should learn sensor ("Press DONE to accept Activity sensor xx type Supervised")

5. Press DONE to accept (Zone types cannot be programmed on this system)

Hardwire Zones on Panel require 4.7k resistor, not needed if connected to Wireless sensors

Zones Hardwire Zones can only be normally closed sensors

Zone A learned Sensor is automatically added to next available Zone Number

Assignments to assign different Zone enter 2-digit Zone Number AFTER entering Group Number

If learning Hardwire Contact on Wireless Sensor learn with Hardwire Contact OPEN

Delete Sensor Press DELETE + SENSOR/REMOTE until you hear item to delete then DONE

Enable 1. Press ADD button (under START MENU)

Function 2. Press OPTION button (Under MAIN MENU) + 2-Digit Program Location

3 Press DONE (May have to enter digits for Phone Numbers, Accounts etc)

Disable 1. Press DELETE button (under START MENU)

Function 2. Press OPTION button (Under MAIN MENU) + 2-Digit Program Location

3 Press DONE, "Keypad should say "Option ?? Deleted"
