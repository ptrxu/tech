Ademco 5878 Wall Transmitter

Information Battery: Maxell CR2032, Duracell DL20325 or Varta CR2032

5 years estimated battery life

Comes with 2 serial numbers, serial number 2 is one digit higher than serial number 1

Programming Can be enrolled in Standard or High-Security modes

Standard mode (6128RF receiver/keypad, 5881 receiver, LynxR panels below v13.1)

  1. Enable by holding STAY, 1 and 2 buttons for 3 seconds (LED flashes green)

  2. Learn buttons like standard 5804 keyfob (Sensor type "BR")

High-Security mode (6150RF receiver/keypad, 5881EN, 5883 RF Receivers, LynxR panels above v13.1)

  1. Enable by holding AWAY, 1 and 2 buttons for 3 seconds (LED flashes red)

  2. Place panel in Go-No Go test (Vista = Installer Code + # + 4, Lynx = Installer Code + # + 8)

  3. Sync serial #1 to receiver by holding AWAY, STAY, 1 and 2 until panel beeps

  4. Sync serial #1 in High-Security mode by pressing AWAY or STAY or 1 or 2 once until panel beeps

  5. Sync serial #2 to receiver by pressing 1, 2, 3, 4 until panel beeps

  6. Sync serial #2 High-Security mode by holding 3 or 4 buttons once until panel beeps

  7. Exit Go-No Go mode by entering User Code + OFF

  8. Program 5878 like standard 5804 keyfob (Sensor type "BR")

User Assign (If Vista-10P/15P/20P)

  1. Exit programming

  2. Add a user code to system (02 to 32), many techs use last 4 digits of keyfob serial number

  3. Master Code + 8 + User (02 to 32) + # + 4

  4. Enter 1st keyfob Zone (Example Zone 53) (Keypad should beep)
