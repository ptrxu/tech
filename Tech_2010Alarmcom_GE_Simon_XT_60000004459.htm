Alarm.com GE Simon XT 600-1048XT GSM Module

Information A. Works with all Simon XT panels version 0.0.H and above

B. If Phone Lock is ON Alarm.com cannot change phone and account numbers

C. If Secure Arming is ON Alarm.com cannot arm system

Yellow SIM card = AT&T

White SIM card = T-Mobile

Power Rating 10 mA in PowerSave mode

30 mA in Idle mode

  1. 7A max in Connected mode

(Requires AC Transformer 9 Volts 3.34A (Part # 22-153) that comes in Simon XT kit)

Registration/ 1. Scroll up or down until panel display shows "System Tests" and press ENTER

GSM Test 2. Enter Installer Code (default = 4321)

3. Scroll up or down until display shows "Comm Test" and press ENTER

4. Display should show "GSM Comm Test in Progress"

5. Display should then show results of test

"GSM test report sent OK" = Good phone test

"GSM test not available" = GSM phone test disabled by Alarm.com

"GSM Comm Test, No More Tests" = Only 6 GSM tests allowed within 6 day period

Mode Notes Module functions under 3 possible modes, troubleshooting should be done in Idle mode

Press and hold 5 key to determine what Mode panel is currently in

  1. Idle = Normal, AC OK, battery above 6 volts, module is not currently in communication

  2. PowerSave = Module has just powered up or AC power is down or battery is less than 6 volts

  3. Connected = Connected to Alarm.com servers, stays connected for 6 minutes after each message

Diagnostic LED's These readings depend on what mode module is in (Idle, PowerSave or Connected)

Because of the way the module is mounted the LED's are on the other side of circuit board

LED 1 (Red) Error message in Idle and Connected mode, disabled in PowerSave mode

  1. flash = Module cannot communicate with panel, try re-inserting into tray

  2. flashes = SIM card missing (check SIM card holder)

  3. flashes = Error registering with GSM network

  4. flashes = Registered with GSM network but cannot communicate with Alarm.com

  5. flashes = Radio portion of module not working correctly

  6. flashes = Module is fixing an unusual condition with connection to GSM network

  7. flashes = Module not compatible with control panel (IE: Simon 3 module in a Simon XT)

  8. flashes = Unknown trouble, call Alarm.com

LED 2 (Yellow) Indicates communication between module and panel in all modes, flashes every

2 seconds in Idle mode or 4 seconds in PowerSave mode

LED 3 (Green) Communication between module and its radio unit and Alarm.com servers,

flashes in unison with LED 2 in PowerSave mode

LED 4 (Green) Signal Strength, flashes 1 to 5 for lowest to highest in Idle mode

No flashes = In PowerSave mode

Just exited PowerSave mode

No GSM coverage in the area

2 seconds ON then OFF = in Connected mode
