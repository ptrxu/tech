Moose Z1200/Z1250/Z2000 User Partition/Authority Level Programming

L: = User Authority Level

A: = Primary Area/Partition of this code

E: = Other Areas/Partitons this code will work in

Level

0 = User Code disabled

  1. = Access Control (0 + User Code) to operate outputs

  2. = Arm only

  3. = Arm only/Access Control (0 + User Code) to operate outputs

  4. = Special Arm only (Arms but cannot disarm unless armed by another User Level 4 code)

  5. = Arm with restricted Disarm (Can only arm within scheduled open time window

  6. = Arm with restricted Disarm + Access Control (0 + User Code)

  7. = Arm/Disarm (Use for standard codes)

  8. = Arm/Disarm and Access Control (0 + User Code)

  9. = Arm/Disarm + Access Control + Duress (Use for duress codes)

  10. (A) = Arm/Disarm + Access Control + Bypass

  11. (B) = Arm/Disarm + Access Control + Bypass + Perform System Test

  12. (C) = Arm/Disarm + Access Control + Bypass + System Test + View Event Log history

  13. (D) = Arm/Disarm + Access Control + Bypass + System Test + Event Log + Modify Schedules

  14. (E) = Master User Code (does everything with its assigned partition)

  15. (F) = Grand Master User Code (does everything)
