ITI SX-4B Program Sheet

Format ITI (5-digit Account Number)

Misc Info C5 = Panel Defaulted?

Enter Program Turn Program Switch ON (UP)

Exit Program Turn Program Switch OFF (Down)

Main Panel confirms all programming with "Bouncing Balls" on display

Add Sensor Press STATUS, panel display will cycle through all programmed sensors

before zones cycle through again press new 2-digit sensor number

(New sensor should now show along with the other sensors)

Delete Sensor Press BYPASS + 2-digit Zone number, panel display will cycle through all

programmed sensors and not show the one just deleted

Master Code Turn on Program Switch, press both AUXILIARY Buttons + New Code

Duress Code Turn on Program Switch, press both POLICE Buttons + 99

Entry Delay Press both POLICE Buttons + STATUS Button + 2-digits (0 to 44, Default = 32)

Exit Delay Press both POLICE Buttons + BYPASS Button + 2-digits (0 to 44, Default = 32)

Account Number Can only be programmed by Downloading

Receiver Number Can only be programmed by Downloading

Siren Timeout Can only be programmed by Downloading

House Code (Only after defaulting panel, Please don't do this...really don't)

  1. Assign House Code to Wireless Keypad using switches (see Keypad Picture)

  2. Press BYPASS on keypad, panel will learn House Code

(Display will go blank, panel will beep and show bouncing balls in LED screen)

Feature Number (Can only be programmed by Installer, mostly not used)

A0 = Exit Beeps On/Off

A1 = Not Used, Must be left Off

A2 = Not Used, Must be left Off

A3 = Communicator On/Off

A4 = Low Battery Reports On/Off

A5 = Supervisory Reports On/Off

A6 = Dialer Abort if Alarm Cancelled On/Off

A7 = Remote Siren Yes/No
