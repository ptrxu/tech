Honeywell Total Connect Troubleshooting

Overview Total Connect allows remote access to Vista-15P/20P or LynxR-I panels

Also allows email messages for programmed events (arming/disarming etc.)

Total Control Website = https://services.alarmnet.com/TotalConnect

Compatibility 7845GSMR Cellular Backup

7845IGSM Internet Cellular Backup

7847i Internet Module

7847i-L Lynx Module

7845i, Symphony and original 7845GSM are not compatible

Vista-15P/20P (v5.2 or higher)

LynxR-I (v15.3 or higher)

Configure Email Uses Vist-15P/20P triggered 4204 relay events to send email

Notifications 1. Enable 4204 relays 1 to 4 or 1 to 8 in *79

If AlarmNet Direct is programmed as "4204 Sourced" set up relays 1 to 4 at address 12

If AlarmNet Direct is programmed as "2 4204 Sourced" set up relays 1 to 8 at address 12

(These "phantom" relays cannot conflict with an existing 4204 relay module if installed)

2. Relay 1 = Event 1 on Total Connect web page, Relay 1 2 = Event 2 etc

Example: To send an email if system is armed and disarmed

  1. Program *80 Output Function 1 to close and stay closed Relay 1 if system is Armed Stay (20)

  2. Program *80 Output Function 2 to close and stay closed Relay 2 if system is Armed Away (21)

  3. Program *80 Output Function 3 to Open (Off/Disabled) Relay 1 if system is disarmed (22)

  4. Program Event 1 "Event Text" in Total Connect to say "Security System"

  5. Program Event 1 "Normal State" to say "Was Disarmed"

  6. Program Event 1 "Activated State" to say "Was Armed Away"

  7. Program Event 2 "Normal State" to say "Was Disarmed"

  8. Program Event 2 "Activated State" to say "Was Armed Stay"

When relay 1 activates Total Connect sends "Security System Was Armed Away"

When relay 2 activates Total Connect sends "Security System Was Armed Say"

When relay 1 or relay 2 turns Off total Connect sends "Security System Was Disarmed"

Troubleshooting General

  1. 7845GSM/7845iGSM must be correct version (7845iGSMR or 7845GSMR)

  2. 7845GSM/7845GSM must be wired as ECP device

  3. Vista-15P/20P must be v5.2 or higher

  4. LynxR-I must be v16.0 or higher

  5. A keypad only display appears with Vista-15P indicates AUI address required by Total Connect is

not enabled in panel or in use b another device.

AlarmNet Direct

  1. "Device Address" must be set (Vista-15P/20P/LynxR-I = 3)

  2. "Enable Direct Wire" must be OFF

  3. "Keypad Address" must be set to an AUI address not used by an installed graphics keypad

For LynxR-I this must be set to 1

  1. "Remote Access" must be enabled

  2. "Keypad Type" must be set for "Full Control" to enable all Total Connect options

or for just keypad display set for "Keypad Only" or "Lynx Keypad"

  1. "Multimode" must be enabled in AlarmNet Direct to send email messages

"4204 Sourced" allows 4 different reports to email, "2 4204 Sourced" allows 8

  1. Multimode address must be set correctly (Vista-15P/20P = 12, Lynx 6 or 7)

Vista-15P/Vista/20P

  1. AUI address used by Total Connect must be enabled in field *189

  2. AUI address for remote access must not be used by an installed keypad

(Example: 6270 or 6271C keypad uses AUI address 1, address 2 is for Total connect remote access

As last resort contact AlarmNet Direct about possible database issues
