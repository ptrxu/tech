Ademco 5827BD Bi-directional Keypad

Information Compatible with any panel using a 4281 (5700 series wireless) or 5881/5882 (5800 series wireless

5800TM module is required if using 5881 or 5882 receiver

5800TM module is not required if using 5883 receiver

5800TM module is not required if using 5128RF/6150RF keypad or Lynx

Does not talk, does not display open zones, LED's indicate status (see below)

Battery 1 9-Volt Alkaline in back compartment

User Guide 1. Press * (ON/STAT) to wake from sleep mode

2. SEND/RCV light should flash while 5827BD contacts alarm panel then ARM or READY should turn ON

3. All standard keypad commands can be entered on 5827BD

(Example: User Code + 2 for Away, User Code + 1 for OFF)

LED Indicators Red (ARM) ON steady + 2 beeps = Armed Away or Maximum

ON steady + 3 beeps = Armed Stay or Instant

Blinking + Pulse beep = Armed, Fire alarm in progress or Fire alarm memory

Blinking + Steady sound = Armed, Burg alarm in progress or Burg alarm memory

Blinking + No sound = Disarmed by alarm memory not cleared

Yellow (SEND/RCV) Blinking + Quick beep = Normal key depressions

Blinking + No sound = RF Transmission in progress or reception completed

Green (READY) ON + 1 beep = Disarmed, ready to arm

Blinking + No sound = Disarmed, not ready to arm

Panel Program 1. Enter 2 in *22 (RF System Type)

(Vista-10/15/20) 2. Enter House Code in *24 (01 - 31)

(Vista-10P/15P/20P) Enter House Code in *24 (01 - 31)

Keypad Program 1. Press * (Yellow light starts blinking)

2. Hold 1 and 3 for 3 seconds (Red & Green Lights start alternately blinking)

3. * 8 + Master Code + # (Enters Master Code into keypad)

4. * 1 + Master Code + # (Enables Quick Arm)

5. * 9 + House Code + #

6. * 57 + # if using 5700 series receiver

* 58 + # if 5800 series receiver

7. * + # exits programming
