Door Chime Sound Programming

  1. Enter * 8 + Installer Code + *

  2. Enter 2-digit Zone Number (01 to 32)

  3. Press 1 to 4 to select door chime

  4. Repeat step 3 for each zone

  5. Press # to exit back to panel programming

  1. 2 3 4

4 Beeps "Bing-Bong" "Ding-Dong" Alarm Tone
