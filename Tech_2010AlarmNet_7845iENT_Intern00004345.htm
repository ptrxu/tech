Commands Requires a Honeywell/Ademco 7720P Programmer

SHIFT 1 (A) = Display software revision (SPACE for next screen)

SHIFT 2 (B) = Display MAC address/Network settings (SPACE for next screen)

SHIFT 3 (C) = Update Firmware

Enter Y to update to new code if needed or N to exit

SHIFT 4 (D) = 1st screen - Connection to Internet (Press * for 2nd screen)

Yes = Physical link to Internet exists

No = Physical link does not exist

2nd screen - 7845i-ENT's Network Address (Press * for 3rd screen)

Indicates if an IP address has been assigned to 7845i-ENT

3rd screen - Encryption Test (Press * to go back to 1st screen)

SHIFT 5 (E) = 1st screen - 7845i-ENT's Network Address (Press * for 2nd screen)

This is the IP address assigned by the network

2nd screen - Display Network Subnet Mask (Press * for 3rd screen)

3rd screen - Display Network Gateway IP Address (Press * for 4th screen)

4th screen - Display Network DNS Server IP Address (Press * to go back to 1st screen)

SHIFT 6 (F) = Restart network services

SHIFT 7 (S) = System Status (Press * for 2nd screen)

ECP = ECP Mode Enabled

TM = Tamper Status

X - not enabled

  1. = New Event (Tamper)

5 = Normal

Flt = Radio Faults

OK = Normal

I = No network , fault time expired

i = No network, fault time not expired

SHIFT 8 (T) = Send Test message to AlarmNet

Sends a test message

SHIFT 9 (X) = Reset 7845i-ENT

Restarts 7845i-ENT

SHIFT Up Arrow = Register 7845i-ENT with AlarmNet
