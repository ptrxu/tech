  3. Enter your install access code (default is 0123).

  4. Enter the set output configuration feature number

  1. Enter output number and press #.

  2. Press # to accept the displayed value or enter the

desired value and press #.

  1. Repeat steps 4 and 5 for all desired outputs.

  2. When finished, press * to exit menu.
