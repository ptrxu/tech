Ademco 5800TM Transmitter Module

Information Provides support for bi-directional keyfobs (5804BD, 5804BDV, 5827BD 5827BDV)

Not required with 5128RF or 6150RF keypads

Addressing Vista-32 and above Address 28 = Cut Red Jumper (W1)

Program in #93 as an "Alpha Console" Address 29 = Cut White Jumper (W2)

Address 30 = Cut both Jumpers

Vista-40 Partition 1 = Cut Red Jumper (W1)

#93 programming is not required Partition 2 = Cut White Jumper (W2)

Vista-10/SE, Vista-15, Vista-20/SE, Via-30/PSE No programming or cutting jumpers required

Vista-10P, Vista-15P, Vista-20P Address 28 = Cut Red Jumper (W1)
