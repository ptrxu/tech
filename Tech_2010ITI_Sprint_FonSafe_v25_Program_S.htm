ITI Sprint FonSafe v2.5 Program Sheet

V2.5 are most common, have internal AC Transformer, can only use FonSafe specific ITI Sensors

Most programing is done at house, communicator programming done by CS-4000 Receiver

Enter Program PROGRAM + Master Code

If "Phone is Wrong" sounds system cannot be programmed while connected to phone line

If tech cannot program sensors connect with CS4000 and use INSTALLMODE ON then UPD

Goto Location Press digit of field desired then press digit of subfield desired

Example: To learn sensor press 2 then 1 then 1

Exit a Location RESET (moves back one menu)

Exit Program RESET

Send to CS4000 1. At CS4000 type ANS 1

  1. Press PROGRAM + Master Code + 8 then press ENTER on CS4000

or if the above doesn't work

2. TEST key + Master Code + 3 + open Zone, press RESET to end

(Apparently zone must be violated to send a phone test signal)

  1. Do not hang up phone, if 1st connection fails press ANS 1 again

CS-4000 Programming Commands

ACCOUNT <??-???> Sets Account Number

ACCESS <???> Sets Master Code

ALARMQ Views last 6 events of Alarm history

ARMLEV <ZZ> <X> Set Zone Arm Level (ZZ=Zone, X=Arm Level 1, 2 or 3)

BYPASS <01 to 32> Bypasses Sensor

CPUTIME SET Sets FonSafe to CS4000 Time

CUSTCODE <????> Customer Code/Home ID Code (no idea what this is)

DELAY <1 to 32> <ON/OFF> Programs Sensor as Delay/Instant

DELETE <01 to 32> Deletes Sensor

DESCRIPTION <ZZ> <XX> Add Voice Description (ZZ=Zone, XX=Description (0-29)

ENTRY <30 to 60> Sets Entry Delay

EXIT <30 to 60> Sets Exit Delay

INSTALLMODE ON Enable local access to zone programming options

INSTALLMODE OFF Disables local access to zone programming options

PHONE <1800 etc> Sets Primary Receiver Number

PHONE2 Secondary Receiver Number

PLEVEL <1, 2, 3> Arming Modes 1 = OFF, 2 = HOME, 3 = AWAY

RESTORE <1 to 32> Restores Bypassed Zones

SIREN <10 to 20> Sets Siren Timeout

STIME <HHMM> Sets daily supervisory reports (Optional)

TTONE <ON/OFF> Sets Panel for Tone/Pulse

UPD Updates changes made to programming to panel

ZONES List all Sensors

Interrogator Programming

From Panel 1. Press PROGRAM + Master Code

2. Press 7 + 5 to enable Interrogator support (it should say "System IO is ON")

3. Press RESET to exit FonSafe programming

Remote (if you don't get this right you must wait 15 minutes before trying again)

Program 1. Assume the tech has no Butt Set, its easier to program like this anyway

2. Call panel on regular phone, let ring twice

3. Hang up, wait 10 seconds and call back, FonSafe picks up on 2nd ring

4. Enter Master Code at prompt, panel will announce menu options

5. Press * * 0 FonSafe may say "Goodbye" (if it hangs up FonSafe isn't setup right)

6. Interrogator should begin beeping (Should hear BEEP after next commands)

7. Press * then * + # + 10 + 1234 + # (Enters Interrogator Programming)

8. Press * + # + 39 + 2 + # (Configures Interrogator to FonSafe)

9. Press * + # + 40 + 2 + 0 + # (Sets up trip action for FonSafe)

10. Press * + # + 44 + 0 + # (Sets up 0 to Talk, 4 to Listen)

  1. Press 99 to exit and disconnect

Local Program 1. Connect butt-set to Program Post and Diode on Interrogator (See diagram)

2. Buttset must be clipped to loop of wire that extends from top of diode down to circuit board

3. Power up panel and Interrogator

4. Move programming jumpers to programming positions

(Move vertical jumper down. Move horizontal jumper to left)

5. Enter Programming by * + # + 10 + 1234 + # (see program sheet on bottom of page)

6. Press 99 to exit then move vertical jumper Up, horizontal jumper Right

Program Sheet (Usually programming here is never needed, leave alone)

  • + # + 10 + 1234 + # Enters Interrogator Programming

  • + # + 11 + # + 0 or 1 Turn Aux Output OFF (0) or ON (1)

  • + # + 12 + # Retreives Account Number in DTMF format (default 1234)

  • + # + 30 + ???? + # Set Interrogator Login Password

  • + # + 31 + 0 or 1 + # Set Dialing format for Pulse (0) or DTMF (1)

  • + # + 32 + ??? + # Phone Number (up to 20 digits, for pause hold 7 for 5 seconds)

  • + # + 33 + 0 or 1 + # Off site Accessing OFF (0) or ON (1)

  • + # + 38 + ???? + # Interrogator 5-digit Account Number

  • + # + 39 + ? + # Set Control panel type connected to Interrogator

  1. = (Default) 2 = FonSafe

  • + # + 40 + X + Y + # Trip Input and Trip Action

X = Trip Input Y = Trip Action

0 = Falling edge 0 = Instant ON, Activation Beeps

1 = Rising edge (SX-V) 1 = Instant ON, no Activation Beeps

2 = ITI Trip 2 = Callback, answer on 1st ring

(Caretaker Plus, Com 2000) 3 = Dial out after trip detect

4 = Callback, No On-site phone ring

  • + # + 41 + > + # Interrogator Aux output option Disabled (0) or Enabled (1)

  • + # + 42 + X + # Auxiliary Output Time/Recording Save Time

X = Aux Output Time 0 = 5 seconds

  1. = 10 seconds

  2. = 5 minutes

  3. = 10 minutes

  • + # + 44 + ? + # Switches hot-key assignments (0 or 1 default is 0)

Default is 0 = 0 Listen 4 Talk 7 Extend 9 Terminate

Switched is 1 = 1 Talk 3 Listen 7 Extend 9 Terminate

  • + # + 49 + # Clears Memory, defaults Interrogator
