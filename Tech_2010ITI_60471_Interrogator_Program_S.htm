Remote (if you don't get this right you must wait 15 minutes before trying again)

Program 1. Assume the tech has no Butt Set, its easier to program like this anyway

2. Call panel on regular phone, let ring three (or let ring 8 times)

3. Hang up, wait 10 seconds and call back

4. Interrogator picks up and should begin beeping (Should hear BEEP after next commands)

5. Press * then * + # + 10 + 1234 + # (Enters Interrogator Programming)

6. Press * + # + 39 + 2 + # (Configures Interrogator to Commander)

  1. Press * + # + 40 + 2 + 0 + # (Sets up trip action for Commander)

  2. Press * + # + 44 + 0 + # (Sets up 0 to Talk, 4 to Listen)

  3. Press 99 to exit and disconnect

Program Sheet (Usually no programming below is needed)

  • + # + 10 + 1234 + # Enters Interrogator Programming

  • + # + 11 + # + 0 or 1 Turn Aux Output OFF (0) or ON (1)

  • + # + 30 + ???? + # Set Interrogator Login Password

(Default = 1234)

  • + # + 31 + 0 or 1 + # Set Dialing format for Pulse (0) or DTMF (1)

(Default = DTMF)

  • + # + 32 + ??? + # Phone Number (up to 20 digits, for pause hold 7 for 5 seconds)

(Default = none)

  • + # + 33 + 0 or 1 + # Off site Accessing OFF (0) or ON (1)

(Default = ON)

  • + # + 34 + 0 to 7 + # Sets Unit ID number 0 to 7 (Caretaker Plus and SX-V only)

(Default = 0)

  • + # + 35 + ?? + X + # Determine which sensor activates which microphone

?? = Sensor Number (01 to 63)

X = Which microphone (1 to 3) for microphone 1 to 3)

(Default = none)

  • + # + 36 + ?? + # Delete sensor mapping done by above command

?? = Sensor Number (01 to 63)

  • + # + 38 + ???? + # Interrogator 5-digit Account Number

(Default = none)

  • + # + 39 + ? + # Set Control panel type (1 to 2) connected to Interrogator

  1. = Caretaker Plus (these settings not applicable to SX-V)

  2. = Commander 2000/Concord Express/LifeGard/Sprint FonSafe

(Default = 2)

  • + # + 40 + X + Y + # Trip Input and Trip Action

X = Trip Input Y = Trip Action

0 = Falling edge 0 = Instant ON, Activation Beeps

1 = Rising edge (SX-V) 1 = Instant ON, no Activation Beeps

2 = ITI Trip (FonSafe) 2 = Callback, answer on 1st ring

(Caretaker Plus) 3 = Dial out after trip detect

(Commander 2000) 4 = Callback, No On-site phone ring

(Concord Express)

(Default Trip Input = 2)

(Default Trip Action = 0)

  • + # + 41 + > + # Interrogator Aux output option Disabled (0) or Enabled (1)

(Default = 0)

  • + # + 42 + X + Y + # Auxiliary Output Time/Recording Save Time

(Default = 0)

V1.5 * + # + 42 + X + Y + + Z + # Auxiliary Output Time/Recording Save Time/CS Hold Time

X = Aux Output Time Y = Recording Save Time

0 = 5 seconds 0 = 10 minutes)

1 = 10 seconds 1 = 20 minutes

2 = 5 minutes 2 = 1 hour

3 = 10 minutes 3 = 5 hours

(Default = 0)

Z = Central Station Hold Time (v1.5 and higher only)

0 = 1 minute

1 = 2 minutes

2 = 3 minutes

3 = 4 minutes

(Default = 0)

  • + # + 43 + ??? + # Sets House Code (001 to 255, must match Control panel)

(Default = None)

  • + # + 44 + ? + # Switches hot-key assignments (0 or 1 default is 0)

Default is 0 = 0 Listen 4 Talk 7 Extend 9 Terminate

Switched is 1 = 1 Talk 3 Listen 7 Extend 9 Terminate

(Default = 0)

  • + # + 49 + # Clears Memory, defaults Interrogator

Press 99 to exit Important! Move vertical jumper Up, horizontal jumper Right
