Moose Z1200 Output Programming

0 = Not Used

  1. = Burg Alarm

  2. = Fire Alarm

  3. = Holdup Alarm

  4. = Auxiliary Alarm

  5. = Tamper 24Hr Burg

  6. = Ready (Ready to Arm)

  7. = Armed (System is Armed)

  8. = Violation (Burg, Fire, Tamper or Panics)

  9. = Lamp (If a key is pressed during Entry/Exit and Alarm times)

  10. = Trouble Indicator (On until Trouble cleared)

  11. = Audible Trouble (On until Trouble silenced at keypad)

  12. = Chime Enabled Indicator

  13. = Chime Output (On 2 seconds when chime zone activated)

  14. = Switched Power Interrupt (Active for 20 seconds during Fir/Smoke reset)

  15. = Telephone Line Fault (Not available on the Z1200)

  16. = Pre-Alarm (Active during Entry Delay)

  17. = Exit Alert (Active during Exit Delay)

  18. = Access (A valid Passcode is entered after 0 command)

  19. = Communication Failure

  20. = Universal 0 (A Universal 0 Zone has been violated)

  21. = Universal 1 (A Universal 1 Zone has been violated)

  22. = Special Alarm (System is armed with Level 4 User Code)

  23. = Fire Trouble (Restores with correction of Zone fault)

  24. = For Future Use (Not available on the Z1200)

  25. = Fire Bell Ringback (Active for 1 second after kissoff of a closing signal)

  26. = Auto Arm Warning (Active for 10 minutes before Auto Arming)

  27. = Instant Armed

  28. = Armed and Exit Time Expired

  29. = Fire Supervisory

  30. = Duress

  31. = Ground Start (Triggers a Telco Ground Start circuit)

  32. = Battery Test Screen (Battery Test is active)

101 to 144 = Zones 1 to 44 Violated
