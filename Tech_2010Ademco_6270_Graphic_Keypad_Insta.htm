Ademco 6270 Graphic Keypad Installation Guide

Panel Info This keypad should not be only keypad on system, it is recommended to also install a standard keypad

The 6270 will always reboot after exiting installer programming , this is normal

Compatible with the following panels

Vista-15P v3 and above Max 2 keypads

Vista-15PMT v3 and above Max 2 keypads

Vista-20P v3 and above Max 2 keypads

Vista-20PMT v3 and above Max 4 keypads

Vista-20PS Max 2 keypads

Vista-128BP v2.4 and above Max 3 keypads

Vista-128FBO v1.6 and above Max 3 keypads

Vista250BP v2.4 and above Max 3 keypads

Vista-250FBP v1.5 and above Max 1 keypad

Power Info 210mA Backlight On Sounds On

190mA Backlight On Sounds Off

180mA Backlight Off Sounds Off

Keypad keeps rebooting = Not enough power

One keypad should run okay but 2 usually requires an external power supply

Installation 1. Enable 6270 in panel programming (Vista-15P/20P section *189 enter 10 for 1 6270 or 11 if 2 6270's

Single 6270 2. Power up 6270 like normal

Installation 1. Enable 6270 in panel programming (Vista-15P/20P section *189 enter 10 for 1 6270 or 11 if 2 6270's

Multiple 6270's 2. Power up one keypad

3. At main menu press SECURITY

4. Press MORE CHOICES

5. Press SETUP

6. Press ADVANCE SETUP

7. Enter panel Installer code, if this does not work try 4140

8. Press CENTRAL STATION

9. Press ECP ADDRESS (each keypad defaults to address 1)

10. Press UP or DOWN arrow to select ECP address 2 then OK

11. Press Keypad may reboot, if not press HOME to return to main screen

12. Power panel down, connect 2nd keypad and power up panel

"ECP Error" Incompatible panel, 6270 ECP address not configured correctly or *189 not enabled in control panel
