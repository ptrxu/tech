Alarm Light (Red) (1) OFF Key

ON - System is in Alarm User Code + 1 to disarm or silence alarm

OFF - Normal, no Alarms in memory

(2) Away Key

Trouble Light (Red) User Code + 2 to Arm all sensors

ON - System has a Trouble

OFF - Normal, no Troubles (3) Stay Key

User Code + 3 to Arm all but Motions

Away Light (Red)

ON - System is Armed to Away mode (4) Max Key

OFF - System is Disarmed or Armed Stay User Code + 4 to arm all sensors in instant mode

Stay Light (Red) (5) Test Key

ON - System is Armed to Stay mode Tests sensors and/or phone communication

OFF - System is disarmed or Armed Away

(6) Bypass Key

Instant Light (Yellow) Used to Bypass zones

ON - System is Armed in Instant mode

OFF - System is not Armed in Instant mode (7) Instant Key

User Code + 7 to arm all sensors except motions

Power Light (Green)

ON - AC Power is OK (8) Code Key

OFF - AC Power is off Used to change User Codes

Ready Light (Green) (9) Door Chime Key

ON - System is Ready to Arm User Code + 9 to turn ON/OFF Door Chimes

OFF - One or more Zones are open

# Key (Quick Arm)

Press # + 2 to Arm AWAY

Press # + 3 to arm STAY

Panic Keys

Press * & # to send Police Panic
