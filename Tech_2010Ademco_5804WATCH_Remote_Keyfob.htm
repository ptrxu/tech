Operation Press Red SECURITY button then press function desired

Lynx Program Lynx must be put into Go/No-Go test first

  1. Exit programming

  2. Enter Installer Code + # + 8 (Keypad will sound siren for 1 second)

  3. Press SECURITY button on watch

  4. Hold all 4 loop buttons at same time (Watch display shows "Sync sent")

  5. Lynx should beep

  6. Program buttons like a normal keyfob

Panel Program (With standard panel or RF keypad programming)

  1. When asked to learn serial number press red Security button then hold all 4 buttons at once

  2. After 3 seconds watch display will say "Sync Sent" and panel will beep

  3. Learn keyfob normally

  4. If using 6150RF keypad enable high security mode in keypad programming first

Watch Program 1. Hold Red SECURITY button for 8 seconds, Loop 1 programming begins

(This shows Loop 1 is disabled)

2. Press START/STOP to cycle through available commands (see list at bottom)

(This shows Loop 1 is set for AWAY Arming)

3. Press MODE to save selected command and move to next

4. Some commands support a delay from 0 to 15 seconds, cycle selections with

START/STOP then press MODE to save and move to Loop 2

(This shows Loop 1 is set for no sending delay)

  1. After programming all 4 loops display will show message duration, this is time in

seconds signal will send, press START/STOP to cycle selections then MODE to save

(bad areas may need longer time to send, good areas less)
