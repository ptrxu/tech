DSC X-10 Programming (Using X-10 PowerFlash)

Note: Requires PowerFlash X-10 module wired to PGM1 or 2

Program Keyfob to control X-10 Lamp Modules

  1. Program PGM in location 009 as 21 (*73 Command Output)

  2. Program Keyfob Button as 19 (*73 Command Output 3)

Programming Panel-Controlled X-10 Module (No Keyfob)

  1. Program PGM as 01 = Light active during alarm

05 = Light active when panel armed stay/away

06 = Light active when panel ready light is On

08 = Light active for 3 minutes after arm/disarm occures

10 = Light active during alarm and latched like strobe

If this doesn't work move Lamp module close to PowerLine as possible and try again
