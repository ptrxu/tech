AlarmNet 7847i-L LynxR-I Internet Backup

Information Internet only

Can be registered without using 7720P programmer like the 7845GSM cellular backup

Works only with LynxR-I v17 and above panels

Works with Total Connect

Turn off "Power Save" option if having downloading issues

Power 12VDC to 14VDC supplied by LynxR-ISIA

56mA current

Total Connect Enable as

Programming Device Address = 3

Allow Local Programming = Yes

Enable Power Save = Yes

Remote Access = Yes

Keypad Address = 1

Keypad type = Lynx

MultiMode = Depends if Email Notifications are wanted

Disabled = No Notifications

4204 Sourced = Up to 4 Email Notifications

2 4204 Sourced = Up to 8 Email Notifications

MultiMode Address = 7

If using 4204 Sourced enable in *86 and program X-10 devices 9 to 12 in *80

If using 2 4204 Sourced enable in *86 and program X-10 devices 9 to 16 in *80

Check 103 or 1. 7847i-L not communicating with LynxR-ISIA

Fault 103 2. 7847i-L not communicating with AlarmNet

3. Cellular backup is not registered

Commands Requires a Honeywell/Ademco 7720P Programmer

SHIFT 1 (A) = Display software revision

SHIFT 2 (B) = Display MAC address/Network settings

SHIFT 3 (C) = Display Time and Date (Greenwich Mean Time)

SHIFT 4 (D) = 1st screen - Connection to Internet (Press * for 2nd screen)

Yes = Physical link to Internet exists

No = Physical link does not exist

2nd screen - 7847i Network Address (Press * for 3rd screen)

Indicates if an IP address has been assigned to 7847i

3rd screen - Display Network Subnet Mask (Press * for 4th screen)

4th screen - Display Network Gateway IP Address (Press * for 5th screen)

5th screen - Display Network DNS Server IP Address (Press * for 6th screen)

6th screen - Encryption Test (Press * to go back to 1st screen)

SHIFT 5 (E) = Not Used

SHIFT 6 (F) = Performs diagnostic tests to AlarmNet or private LAN

These must be OK to show link to AlarmNet

SHIFT 7 (S) = System Status (Press * for 2nd screen)

12345678 = Zone Status

X = not enabled

  1. = New Event (Alarm)

5 = Normal

TM = Tamper Status

X - not enabled

  1. = New Event (Tamper)

5 = Normal

Flt = Radio Faults

OK = Normal

I = No network , fault time expired

i = No network, fault time not expired

SHIFT 8 (T) = Send Test message to AlarmNet

Sends a test message

SHIFT 9 (X) = Reset 7847i

Sends a test message

SHIFT 0 = Force Server Update

Enter Y to send firmware info to AlarmNet

SHIFT Up Arrow = Register 7847i with AlarmNet
