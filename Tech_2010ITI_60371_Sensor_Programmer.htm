ITI Programmer (SX-V, Caretaker, Commander Panels only)

Information 2 types of programmers exist

Part # 60-371 that could to Octal and Decimal modes for programming Commander 8/17 and SX-V

(Commander 8/17 required Decimal mode, SX-V required Octal mode)

Part # 60-101 that could only do SX-V panels, could not switch to Decimal mode

Press SENSOR NUMBER twice to switch between Decimal and Octal format

Read Sensor 1. Press ON, programmer will say HELLO

2. Connect pigtail to pins on sensor, if a long tone sounds pigtail is connected backwards

3. Press READ, current sensor programming will be displayed

If SLEEP appears battery is low, sensor programming has been erased

If FAIL appears sensor is dead

Program Sensor 1. Press HOUSE CODE, house code display will flash, enter new 3-digit house code then ENTER

(After pressing ENTER display should say OK, if FAIL appears sensor is likely bad)

2. Press SENSOR NUMBER, sensor number display will flash, Enter 2-digit zone number then ENTER

3. Press SENSOR TYPE, sensor type display will flash, Press number key of sensor type then ENTER

(Example: If door or window press 1, if motion sensor press 2)

  1. Disconnect from sensor when done

Caretaker Wiring "Pigtail" must be removed and connected directly to board,

programmer cannot be used on that panel
