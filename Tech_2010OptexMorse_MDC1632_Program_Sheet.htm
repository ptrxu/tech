Optex/Morse MDC 8/16/32 Program Sheet

Enter Program 1. Enter 0000 + PROGRAM, display will show "ENTER YOUR COMPANY PIN"

2. Enter 4-digit Installer Code (default is 9999), display will show"ENTER RESPONSE"

3. All main menus will now scroll across screen, select by entering 1-digit main menu

4. Press 1-digit sub menu

(Example: to program Receiver 1 phone number

1. Press 5 for "Receiver"

2. Press 1 for "Receiver 1"

3. Press 1 again for "Telephone", program phone number then press ENTER

4. Press CLEAR to exit back to "Receiver 1" then CLEAR to scroll main menus

Enter Data Enter new Values then press ENTER

Exit Location Press CLEAR

Exit Program Press CLEAR until you are back to "System Ready"

Zone Program 1. From main menu listing press 3

2. At "Set Zone Number" press 1 to 16 then ENTER

3. Press CLEAR to select Zone options (1 for Alarm Outputs, 2 for Loop Types etc)
