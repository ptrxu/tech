Silent Knight SK5107 Basic Commands

Operator Code 1111

Silence Alarms SILENCE + User Code

Clear Memory CLEAR

Clear Wrong Codes CLEAR

Silence Troubles SILENCE + User Code

Display Trouble 6 + ENTER + User Code

Dialer Abort 3 + ENTER + Installer Code

System Test TEST + ENTER + User Code

Phone Test 0 + ENTER

Set Time 9 + ENTER + User Code + 4 digit military time (00:00 to 23:59)

Display Codes (on Main Panel)

0 to 16 = Zone (with Alarm, Alarm Memory or Trouble LED)

E0 = Trouble with Dialer

E1 = Trouble with External Printer

E7 = Trouble with EEPROM Memory

F0 = Keypad Power Trouble

F1 to F7 = Trouble with a particular Keypad

A1 to A4 = Trouble with a particular Bell Output

P1 = Trouble with Smoke Power

P2 = Trouble with Accessory Power

P3 = Earth Ground Fault to Circuit Ground

P4 = Earth Ground Fault to Power

Po = Printer out of Paper

dL = Communicator lost data it was trying to transmit

dF = Communicator Failure

dc = Low Battery

Ac = Low AC Power

L1 = Phone Line 1 Fault

L2 = Phone Line 2 Fault

Keypad Display -0 = Fire Drill -5 = Zone Test

-2 = Walk Test -6 = Hex Programming Mode

-4 = Downloading -8 = Set Date Mode

-9 = Set Time Mode
