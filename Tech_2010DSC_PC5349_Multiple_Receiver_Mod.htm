DSC PC5349 Multiple Receiver Module

Specifications Frequency: 900mhz and 433mhz

Current Draw: 180mA with 2 receivers

Information No downloading of wireless sensors can be done if this module is used

Wait 2 minutes after exiting programming to update wireless receivers

Do not use the PC5130 module with this

Do not enable RF Jam detection

Do not enable Identified Wireless Key option

Compatibility PC5132-900

WLS910 Keypad is NOT supported

Programming 1. Delete all 900MHz devices from old receiver

2. Power down and disconnect 900MHz receiver

  1. Connect 433MHz receiver directly to panel

  2. Program new device into zone slot deleted in step 1

  3. Perform placement test on new sensor

  4. Power down and disconnect 433MHz receiver

  5. Connect PC5349 and both receivers

  6. Apply power, wait at least 2 minutes for receiver mapping to take place

  7. Test all wireless zones

  8. If adding more 433MHz sensors repeat from step 1
