DSC NT9005 Envoy Program Sheet Binary Programming

Report Format SIA, Contact ID, 4/2, 3/1

Enter Program 1. * 8 5555, Program Icon will flash

2. Press 1 to enter Flash Programming or 2 for Advanced Programming

Flash (Quick) Programming

Flash programming is divided into 4 groups

Press STAY (Forward) to move to next group or AWAY (Back) for previous group

"Sn00" = Learn Sensors

"Ph00" = Enter Phone Number

"Ac00 = Enter Account Number

"PL00" = Placement Test (to confirm sensors)

Enroll Sensors 1. Display shows "SN00"

2. Enter 6-digit serial number of sensor

3. Keypad will beep and 2-digit zone used then 6-digit serial number

4. Press 1 to accept sensor (display will show "done")

or 2 to return to "SN00" to re-enter serial number

5. Press FORWARD (STAY key) to return to "SN00" prompt for next sensor

or FORWARD (STAY key) until "PL00" appears

6. Activate each zone to confirm, keypad gives 1 beep for good placement or 3 for poor

a. Panel must receive 3 good placement tests in a row to confirm

b. Test door sensors by opening zone

c. Test motion detectors by removing from backplate (wait 30 seconds between tests)

d. Test smoke detectors by removing from backplate (wait 30 seconds between tests)

  1. Panel will beep and sound siren for 2 seconds if successful

  2. Press FORWARD (STAY key) to return to "Sn00" prompt or # to exit programming

Hardwire Sensors Loop 1 (Y1) and Loop 2 (G2) can be any zone number (01 to 32), see locations 030 and 101 to 132

Enter Phone/Acct 1. Press STAY until display shows "Ph00"

2. Enter Central Station Phone Number then #

3. Display will show "Ph" and scroll phone number

4. Press 1 to accept (display will show "done")

or 2 to return to "Ph00" to re-enter phone number number

5. If 1 is pressed display will show "Ac00" for Account number

6. Enter 4-digit account number

7. Display will show "Ac" and scroll account number

8. Press 1 to accept (display will show "done")

or 2 to return to "Ac00" to re-enter account number

Exit Program #

Move to location Enter 3-digit location

Default Panel 1. Remove all wires from BLACK and GREEN terminals

2. Jumper between the two terminals

3. Power up and wait until beeps sound

or

  1. Enter Installer Programming * 8 5555

  2. Enter 999 + Installer Code + 999

Enable Download * + 6 + Master Code + 5 (Enables for 6 hours)
