Ademco Universal Voice System (UVS)

General Info 2-way will trigger on any alarm that sounds siren

Alarms that do not sound siren will not activate 2-way on many panels

DSC Notes UVS will not work if connected to R1 and T1 of DSC panel, connect to T and R

Trigger by connecting Falling Voice Trigger to PGM1 and set PGM1 to 13 Kissoff Output

Ademco Notes Vista panels must have *91 set to 4 if available

Troubleshooting 1. Check if UVS is getting 12VDC power

2. Check if UVS is activating by listening in

Panel should take 30 seconds to send signal

UVS should stay online for 90 seconds afterwards

3. Check if phone lines are wired properly

4. Check programming of panel (if Vista-15P/20P)

5. Make sure no cancel or restoral signals are programmed in panel

6. Make sure SW1 dip switches 1 and 8 are ON
