ITI LifeGard Program Sheet

Format ITI or 4/2 (Always use ITI)

Power Rating 290ma max (Sirens, motions etc)

60-483 Slimline Interior Siren 60ma

13-046 Exterior Siren 100ma

Interrogator 2-Way Module 55ma Inactive up to 300ma Active

Hardwire Standard Motion 20ma (average)

Hardwire Microwave Motion 30 to 50ma (average)

Hardwire Glassbreak 25ma to 35ma (average)

Enter Program From Panel - Power system down, lower battery tray, power up system

From Keyfob - Lower battery tray, enter 4321 from keyfob

Exit Program Raise battery tray and tighten screws

Answer Download Press TEST for 3 seconds

Remove HELP Panic cover to expose programming keys

Clear Memory POLICE + POLICE + CANCEL + CANCEL

Add Keyfob ACTIVITY LEVEL + ACTIVITY LEVEL + Keyfob (1 to 4)

Delete Keyfob TEST + TEST + Keyfob (1 to 4)

Learn Sensor ACTIVITY LEVEL + Sensor Group (00 to 29) + Sensor Number (01 to 17)

Delete Sensor TEST + Sensor Number

Exit Sensor Program STATUS (it will say "Invalid, Try Again")

Account Number FIRE + FIRE + ACTIVITY LEVEL + Last 5 digits of Account

(If not using J or B account numbers enter 0 + Last 4 of Account)

House Code FIRE + FIRE + House Code (001 to 254)

Primary Phone FIRE + FIRE + TEST + Receiver Number

Delete Phone FIRE + FIRE + TEST + 1

Format STATUS + CANCEL + CANCEL + (00, 01 or 03)

00 = ITI 01 = 4/2 2300Hz 03 = 4/2 1400Hz

Siren Timeout CANCEL + CANCEL + (02 to 15)

Add Upper Sensor TEST + Sensor (79 to 98 except 86 to 88, 92 to 97)

Delete Upper Sensor ACTIVITY LEVEL + Upper Sensor (79 to 98 except 86 to 88, 92 to 97)

Pill Reminder 2 STATUS + POLICE + POLICE + 2 + 0000 to 2400 (time in hours)

Set Feature CANCEL + CANCEL + ACTIVITY LEVEL + (20 to 24, 27, 30, 34 or 35)

F20* ON = Interrogator Module Enabled OFF = Disabled

F21 ON = Battery power only OFF = AC Power

F22 ON = Tone Dialing OFF = Pulse Dialing

F23 ON = Open/Closing Reporting Enabled OFF = Disabled except in Event Buffer

F24 ON = Hardwire Zone Input Normally Open OFF = Normally Closed

F25 Not Used

F26 Not Used

F27* ON = Panel Sounds disabled OFF = Panel Sounds enabled

F30 ON = Low Bat reports Immediately, OFF = Reports according to STIME

F32** ON = Wireless Fixed Panics not supervised in Away OFF = Fixed Panics supervised always

F34 ON = External Output for Strobe OFF = Output for Siren

F35 ON = System only switchable between DAY & NIGHT OFF = System Normal

F43 ON = Demo Kit Enabled OFF = Disabled

F44 ON = Exterior Siren Constant (not pulsed) OFF = Exterior Siren Pulsed

F45 ON = Zone 1 set for Keyswitch Arming OFF = Zone 1 Normal

F46 ON = Automatic Daylight Savings time correction OFF = No correction

F47 ON = UL Fire Trouble Beeps restart 4 hours OFF = Beeps restart after 10 hours

  • = Must be ON if using Interrorgator Module

  • = Version 5.1 and later
