Ademco 6150V/6160V Voice Keypad Programming

Information 6150V can be used on all panels except very old Vector, 4160 and 4180

6160V can be used on all panels except 4110, very old Vector, 4160 and 4180

Power Rating 6150V - 60mA standby 160mA active (Backlight, sounder and voice on)

6160V - 60mA standby 190mA active (Backlight, sounder and voice on)

Keypad Address 1. Remove power from keypad then power back up

2. Hold 1 & 3 for 3 seconds within 60 seconds, "ADDR = 31" or "31" should appear

3. Within 10 seconds enter 00 to erase old address

4. Enter new address (01 to 31) then press *

5. Display shows "VOICE CHIME" or "C" then * Press 0 to disable Voice Chime

Press 1 to enable Voice Chime

6. Display shows "ADDITIONAL CON" or "A" Press 0 if only keypad is in system

Press 1 if multiple voice keypads in system

7. Press * to save and exit

View Address Press 1 & 3 at any time then * to exit

Voice Chime # + 0 + 2 + 4 to Enable or Disable (Keypad should say "ON" or "OFF")

Detailed Status # + 0 + *

Show Zone Faults * *

Record Message 1. Press # + 0 + 1 (LED turns ON)

2. Record message up to 2.5 minutes

3. Press 1 to end (LED begins FLASHING)

Play Message 1. Press # + 0 + 3

2. Recorded message will play (LED turns OFF)

Adjust Volume Press # + 0 + 2 then 3 to raise or 6 to lower
