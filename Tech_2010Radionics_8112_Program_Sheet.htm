Radionics D8112 Program Sheet Click HERE for D5200 Programmer

Power Ratings 1A Aux output

2A Siren output

Formats Modem II, BFSK, 4x2 1400 or 2300Hz

Modem Account numbers can be 4 digits, all other formats must be 3 digits

Download Info Panel will pick up depending on how many rings AnsArmRings and AnsDisRings are set

Answer Download Press Command 43 to cause panel to answer, may require entering access code

Program Info Most common version is 8112G2

Command 59 shows panel revision

Programming only by downloading, 5200 Programmer or 5100 "Wand" Programmer

Plug programmer in and momentarily short terminal 32 to 29 to enable program mode

Having many devices (keypads, OctoPopits, etc) interferes with programming

Disconnect all devices from Terminal 31 then load program again

When finished programming must Disable/Restart

Receiver Info If Radionics receiver C1 to C6 use Modem II Format with 4-digit Account

If Virtual Receiver use BFSK with 3-digit Account

This is not a complete, it only shows the basics for MAIN and ZONEX sections

MAIN

PRIMARY CENTRAL ST. Variables/Defaults

  1. Off I NO I Yes/No Yes = System Disabled, No = System Enabled

  2. Local I NO I Yes/No Yes = Communicator OFF, No = Communicator ON

  3. CS I _7 _I 1 to 6 1 = Silent Knight Low Speed Double Round

2 = Ademco Low Speed Double Round

3 = Silent Knight High Speed Double Round

If Virtual Receiver use 6 4 = Ademco High Speed Double Round

If Rad Receiver use 7 5 = Radionics Single Round with Parity Digit

6 = Radionics BFSK Superfast Single Round

7 = Radionics Modem II

Blank = Sescoa, Franklin Hi Speed Double Round

  1. SgRd I NO I Yes/No Yes = Single Round with Checksum (Format 5 only)

No = Disabled

  1. Ack14 I NO I Yes/No Yes = 1400Hz Ack tone, No if Modem II Format used

  2. Ack23 I YES I Yes/No Yes = 2300Hz Ack tone, No if Modem II Format used

  3. P I _1 _I _8 _I _0 _I _0 _I_ _I _ _I _ _I 8-digits 0 to 9, C to D, Blank (Phone Prefix)

C = 3 second pause D = 7 second Dial Tone pause

  1. Ph I _? I _? I _? I _? I _? I _? I ? I 7-digits 0 to 9, C to D, Blank (Rest of Phone Number)

ADDITIONAL PHONE Variables/Defaults

CALL ROUTING Variables/Defaults

  1. Route I _ _I 0 to 7, Blank

BLANK = No Alternate Phone Number, all reports to Primary

  1. = Primary = All Reports, Alternate Duplicate Reports

  2. = Primary = All Reports, Alternate = Reports from Designated Zones

  3. = Primary = All Reports, Alternate = Alarms from Designated Zones

  4. = Primary = All Reports, Alternate = Backup Primary for Alarms

  5. = Primary = All Reports, Alternate = Backup Primary for ALL Reports

  6. = Primary = Supervisory Reports, Alternate = Alarm Reports

  7. = Primary = Supervisory Reports, Backs up ALL Reports
