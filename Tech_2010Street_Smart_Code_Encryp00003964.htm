Street Smart Code Encryptor CE2-Y (Yellow) Installation Instructions

Compatible Panels (see Compatibility Chart)

Wire directly to keypad bus, Arm/Disarm and Panic (Red) buttons auto configure

Red Button relay mode and middle button can be wired to activate PowerFlash,

Garage Door and Zones configured as Panic

Non-Compatible Panels (see Compatibility Chart or if reciever doesn't recognize panel)

Wire directly to keypad bus, Arm/Disarm button will not function but middle

can be used as Arm/Disarm and Red button can be used as Panic only if panel supports

Zone Panic and Arm/Disarm functions, Red button also can be used to open garage door

and activate a PowerFlash

Ademco Must use have 6128 keypads or higher or Panic Button will not work

Notes (however use Relay 3 to activate zone programmed as 24 Hour Police Panic)

Current Code Encryptors will not work with Vista-15P/20P due to addressing of keypads

DSC 1575 Connect Brown to PGM2 with inline 2.2k, Program PGM2 for momentary keyswitch (26)

Brown/White Wire to Aux +

DSC Notes If using PC5501Z-433mhz Keypad with RF Receiver turn off RF Jamming (804 section 90)

Radionics Requires special version (CE2Y-RAD) which may no longer be available

2112/2212 Set panel for 3-digit User Codes only and disable "Passcode Required" option

Program 1. Wire harness to alarm panel (see wiring diagram)

2. Plug harness into Code Encryptor, LED flashes 1 time on power up then flashes showing panel type

  6. Ademco Addressable

User Code (Do not use Master Code, program a secondary code for Keyfob)

  1. Press and HOLD program button on Receiver, LED turns ON three seconds then OFF

  2. RELEASE program button, LED will flash rapidly

  3. Enter four-digit User Code

  4. LED should stop flashing if User Code learned

Left keyfob works with its own receiver, Right keyfob uses its own receiver

Left keyfob cannot work with Right keyfob receiver and vice-versa

Add 1. Disarm panel and enter 78738 from keypad. System will arm in "STAY" mode

Keyfob 2. Press big button until system disarms. It should take a four presses

or (use this instead)

  1. Press and release button on receiver. LED will turn ON

  2. Press big button on new remote 3 times if 4 button remote press top left key

  3. LED on receiver should turn OFF (If LED stays ON, remote not learned)

Delete (If deleting Keyfobs all keyfobs will be deleted)

Keyfob 1. Disarm panel, enter 76278 from keypad

2. Keypad should "beep" or lights will turn off to confirm delete

or (use this instead)

  1. Press and hold program button

  2. LED will turn ON for four seconds, then go OFF then ON again

  3. Release program button

Default 1. Press and hold program button

Receiver 2. LED will turn ON for four seconds, then go OFF then ON again

3. Release program button

Deactivate 1. Unplug wire harness from Code Encryptor

Panic 2. Press and HOLD program button

Button 3. While HOLDING program button, plug harness in, LED will turn ON

  1. Wait until LED turns OFF

  2. Once LED has turned OFF release button

Activate (If panic was previously disabled)

Panic 1. Unplug wire harness from Code Encryptor

Button 2. Press and HOLD program button

3. While HOLDING program button, plug in Code Encryptor harness

The LED light located on the front will turn ON

4. Release program button

Channel 2 Every time button 2 is pressed Gray wire sends a 500ma (-) output for 3 minutes

Time Output You can use this output to drive entry/exit illumination

Every time customer enters or exits house this output can trigger a relay or X-10

Powerflash module to interface with driveway, porch or lighting

Output activates automatically when button 2 is pressed and will shut off in 3 minutes

Channels 1 Data outputs

Channel 2 Relay N/O, Comm (10amp)

Channel 3 selectable: Form C Relay (N/O, N/C, Comm) 5amp

(Output can be configured from pulsed to latching, 75 or 150 second timed output

Channel 3 - Keypad panic data output (Programmable On or OFF)
