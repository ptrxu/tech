DSC WS4979 Wallplate Touchpad

Information Emulates a 4-button keyfob

Compatible with all DSC 433 mhz receivers

Cannot do a placement test on this sensor

Panic by default requires holding button 3 seconds, to disable see below

Batteries: 2 CR2025 button type 3V Lithium, estimated battery life 5 to 8 years

Remove top cover by prying open with screwdriver in bottom slot to replace batteries

Range: 500 feet open air, no farther than 100 feet recommended

Temperature: 32 to 120 degrees

Programming 2 keyfob slots are required if using all 4 buttons

Function keys 1,2 and 4 have one ESN serial number

Function key 3 uses ESN number +1

Enable/Disable 1. Press all 4 buttons until LED flashes

Disarm button 2. Press Disarm button, LED flashes 1 time if enabled, 2 times if disabled

Enable/Disable 1. Press all 4 buttons until LED flashes

Panic Delay 2. Press Away button, LED flashes 1 time if delay is enabled, 2 times if delay is disabled
