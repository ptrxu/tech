GE Concord v4 Program Sheet

Information Must be programmed with 60-746 or ATP-1000 Alpha Keypad wired to keybus terminals or 60-791 cable

XS 000 - RF Receiver Tamper Zone = Turn 06012 Receiver Failure OFF

IA000 - Equipment Failure Condition = Turn 06017 Ground Fault OFF

Monitronics Concord 4 Express v4.0 to 4.02 = 32 zones, 1 partition, 16 user codes

Monitronics Concord 4 Express v4.10 and above = 32 zones, 2 partitions, 16 user codes

Standard Concord 4 = 96 zones, 6 partitions, 229 user codes

Power Ratings 1A Aux output with 16V 40VAC transformer

100mA for smoke detectors on zone 8

  1. A Output 1

300mA Output 2

Enter Program 1. Enable program keypad with 8 + Installer Code + 02 or use any existing alpha keypad

(Standard) 2. Press 8 + Installer Code + 00, keypad says "SYSTEM PROGRAMMING"

(Do not power up with Programmer connected to avoid Bus Errors later)

Enter Program (Quick Menu only allows programming of phone and account numbers and sensors)

(Quick Menu) 1. Enable program keypad with 8 + Installer Code + 02 or use any existing alpha keypad

  1. Press 8 + Installer Code + 03, keypad says "ACCOUNT NUMBER"

Move to Location From any menu (like SECURITY or GLOBAL) enter 4 or 5 digit "Shortcut"

(you cannot do this while IN a location)

Exit Program 1. Press * until "SYSTEM PROGRAMMING" appears

2. Press A or B keys until Keypad shows "EXIT PROGRAMMING" press #

  1. Remove Programmer Keypad from connector on circuit board

"Call For Service" Unknown trouble, to remove see below

  1. Press D key & 6 at same time

  2. Press # + 250 + # + *

Enable 1. Press 9 + Master Code ("SYSTEM MENU" then "USER CODES")

Downloading 2. Press * + 41 + 2 + # ("Display shows "DOWNLOADING ON")

  1. Press * + 7 + # to exit

Send to Press 8 + Master or Installer Code + 7 + 0

Downloader

Sensor Test 1. Press 8 + Master Code + 3

2. Keypad shows "Armed to Sensor Test", beeps every 60 seconds

3. Open Door or Window, move by motion

Good wireless sensor test will beep keypad and show "OK"

Bad wireless sensor test will sound long low-pitched beep and not show as tested

  1. Press * to view any untested sensors

  2. Press 1 + Master Code to end

Rescan Bus 8 + Installer Code + 01

Address Keypad Does not have to be done with this version of panel

All modules and keypads are auto-addressed

Delete Unit ID (If programming keypad is accidentlly learned into panel see Bus Devices section)
