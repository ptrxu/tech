ITI 60-736-95 Supervised Wireless Siren

Compatibility Caretaker +

Commander 2000

Concord (Supervised)

Simon 2 (Supervised)

Will NOT work with Simon 3

UltraGard

Siren Status Red LED + Trouble Beeps

OFF = No AC Power

  1. Flash = Siren House Code is 0

  2. Flashes = Low Siren Battery

  3. Flashes = Low Siren Battery, Siren House Code is 0

  4. Flashes = No Signal from panel

  5. Flashes = No Signal from panel, Siren House Code is 0

  6. Flashes = No Signal from panel, Low Siren Battery

  7. Flashes = No Signal from panel, Low Siren Battery, Siren House Code is 0

Status Volume 1. Press and hold siren TEST button for 10 seconds, siren should beep

2. Press TEST button for 1 second for next volume level

3. Continue until desired volume level is set

4. After 5 seconds of inactivity siren will exit programming

Installation 1. Add battery pack to siren

(Unsupervised) 2. Set House Code in panel (001 to 254 or A to O)

3. Plug in siren

4. Press STATUS on panel keypad, siren should beep once and LED flashes 5 times

(If needed reset siren house code to 0 by holding red test button for 20 seconds)

Concord (Concord must have 60-755 X-10 carrier card and 60-762 or 60-779 X-10 transformer)

Installation 1. Set panel partition house code (000 to 254)

(Supervised) Location 0401 if partition 1

Location 0411 if partition 2

2. Set SWS Supervision Code (location 0703)

For 1 partition systems this must match panel house code

For 2 partition systems the house codes and SWS codes must all be different

3. Press and hold red TEST button on siren until 3 beeps are heard (about 20 seconds)

4. Enter LEARN SENSORS (location 080) and set at sensor type 33

5. Plug siren in, it should beep indicating it received panel house code.

Red TEST button on siren should flash 10 times indicating it is in supervised mode

6. Panel display should indicate sensor (siren) has been learned

If this doesn't work hold red TEST button 20 seconds to reset house code and try again

Simon 2 (Must be Standard or 2-way Simon (not Basic) and must have 22-109 AC transformer)

Installation 1. Set panel house code (A to O)

(Supervised) 2. Press and hold red TEST button on siren until 3 beeps are heard (about 20 seconds)

3. Unplug siren

4. Learn siren as a sensor into panel (sensor type 33)

4. Plug siren in to learn, siren should beep and flash 10 times and learn into Simon

If this doesn't work hold red TEST button 20 seconds to reset house code and try again
