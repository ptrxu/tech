Optex/Morse 824R Program Sheet

the following is for the 824R, the 824 may be different

Enter Program 9999 + + 7 Keypad should say PRG

Exit Program # twice

Move to Location Enter 2-digit location

Previous Location STAY

Exit Location # (Moves to top of menu, just enter next location)

Entering Data Enter data + to store and advance to next field

Default Panel 1. Installer Code + + 7 Keypad should say PRG

  1. Press 99, display will read TST8

  2. Press 0, display will read DEF

  3. Press , display will read CODE

  4. Press 99999 + and wait 15 seconds

  5. Display should flash TIME and is defaulted

Each location has submenues, like 56 then 56A 56B etc, keypad shows Location + Value

Press to advance to next location

Press # to move to top of programming to choose any Location you want

Enter data then press to save

Zone Program

00 Zone 1 Zone/Loop Type ??

  1. st digit = Zone Type 2nd digit = Loop Type

0 = E/E 1 0 = Normally Open

  1. = E/E 2 1 = Mormally Closed

  2. = Instant 2 = Normally Open/Close*

  3. = Interior 1 3 = N.O. w/EOL (2.2k required)

  4. = Interior 2 (motions) 4 = N.C. w/EOL (2.2k required)

  5. = Interior 3

  6. = 24 Hour

  7. = Fire Zone

  8. = Day Zone

00A Zone 1 Loop Response 05 (00 to 99 ms)

00B Zone 1 Alarm Report Code 31 (Doesn't matter if SIA?)

00C Zone 1 Trouble Report Code 00 (Disabled, use 76 if Smoke Detector)

00D Zone 1 Bypass Report Code 00 (Disabled)

00E Zone 1 Restore Report Code 00 (Disabled)

00F Zone 1 Features (ON = bar ON, OFF = Bar OFF)

(Look at keypad picture, see the bars under AWAY and BYPASS? toggle below like DSC)

Light Default Press Bar ON Bar OFF

AWAY I_ ON__I 1 ON = Telephone Reporting Enabled OFF = Local Alarm

STAY I_ ON__I 2 ON = Audible Alarm OFF = Silent

INST I_ OFF_I 3 ON = Pulse Siren Output OFF = Disabled

BYPASS I_ OFF_I 4 ON = Zone Activates PGM Output 1 OFF = Disabled

ALARM I_ OFF_I 5 ON = Zone Activates PGM Output 2 OFF = Disabled

TROUBLE I_ ON__I 6 ON = Zone Activates Walk Test OFF = Disabled

FIRE I_ OFF_I 7 ON = Monitor (Chime) Mode Enabled OFF = Disabled

BLANK I_ OFF_I 8 ON = Silent/Disarmed, Audible Armed OFF = Disabled

00G Zone 1 Features (ON = bar ON, OFF= Bar OFF)

Default Option Press Bar ON Bar OFF

AWAY I_ ON__I 1 ON = Keypad 1 Audible Entry Buzzer OFF = Disabled

STAY I_ ON__I 2 ON = Keypad 2 Audible Entry Buzzer OFF = Disabled

INST I_ ON__I 3 ON = Keypad 3 Audible Entry Buzzer OFF = Disabled

BYPASS I_ ON__I 4 ON = Keypad 4 Audible Entry Buzzer OFF = Disabled

ALARM I_ OFF_I 5 ON = Keypad Displays Alarm OFF = Disabled

TROUBLE I_ OFF_I 6 ON = Zone Swinger Shunt Allowed OFF = Disabled

FIRE I_ ON__I 7 ON = Zone Bypass Allowed OFF = Disabled

BLANK I_ OFF_I 8 ON = Alarm Output to Radio OFF = Disabled
