Caddx NX-408E/416E/448E Wireless Receiver

Information NX-4/NX-6 wireless zones can overlap hardwire zones, the hardwire zones will automatically be disabled

NX-8/NX-8E wireless zones can overlap hardwire zones but onboard zones must be disabled (Location 37)

NX-8/NX-8E claim blocks of 8 so if a sensor is learned into zone 9 then zones 10 to 16 must also be wireless

NX Panels can have multiple receivers

NX-8E = 8 receivers maximum

NX-8 = 4 receivers maximum

NX-6 = 3 receivers maximum

NX-4 = 4 receivers maximum

And up to 48 zones

NX-408E = 8 Zones maximum

NX-416E = 16 Zones maximum

NX-448E = 48 Zones maximum

Enroll Module 1. Enter Programming with * 8

2. Enter Installer Code (Default = 9713)

3. Press 0 # to enter Panel Programming

4. Press EXIT twice to Exit Programming

5. All modules will be enrolled (Service light turns ON for 12 seconds)

Enter Program 1. Enter panel programming with * 8

2. Enter Installer Code (Default = 9713)

3. Enter Module Address + # (Default = 35)

Go to Location Location + #

Enter Data Enter data then * to save and move to next section of a location

  • also steps through programming while viewing a location

  • also exits if at end of a location

# exits at any time

Example: To change Supervision Time

  1. Press 195 + # (Moves to location 195)

  2. Enter 12 + * (Changes Burg Supervision from 24 to 12 hours)

  3. Enter 5 * (Changes Fire Supervision from 4 to 5 hours and exits)

Delete Sensor 1. Enter panel programming with * 8

2. Enter Installer Code (Default = 9713)

3. Enter Module Address + # (Default = 35)

4. Enter Zone to delete then # (Zones 1 to 192)

5. Turn light 1 OFF then * to save

6. Press # to exit

7. Press EXIT twice to exit programming
