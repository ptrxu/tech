Ademco 5828/5828V Remote Keypad Program Guide

Information Keypad and display shuts down after 15 seconds if on battery power until a button is pressed

Display only shuts down after 15 seconds if on DC power supply

For best RF transmission use BOTH battery and DC power supply

Older 5828's had a problem with bent antennas reducing RF reception

Does not work with original Lynx

Original 5828's worked with LynxR/R-EN and unofficially 6150RF keypads

Later versions only worked with LynxR/R-EN and did not work with 6150RF

Latest 5828's will only work with LynxR/R-EN and 6160RF keypad/receivers

Power 3 1.5 Volt AA Alkaline batteries and/or 5.0 Volt 220ma DC transformer part: 5828DM
