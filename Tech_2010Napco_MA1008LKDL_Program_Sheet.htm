Napco MA1008LKDL Program Sheet

Power Ratings 450mA Aux output with 16.5V 19.2VA transformer

500mA Aux output with 16.5V 40VA transformer

  1. 2A Siren output

How to Identify Check the inside wiring diagram for exact model

Enter Program 1. Hold 8 key for 3 seconds, keypad will beep

  1. Enter 4 5 6 7 8 9, display should show dash on display

(Will not allow entry into programming if system is reporting, armed or sounding siren)

Program Location 1. Press [S] key display will show 3 dashes

2. Enter 3-digit location (use [S] for any 0)

  1. Value currently programmed in location will display

  2. Scroll Value UP with 4 key, scroll Value DOWN with 7, blank location with 5

  3. When display shows desired Value press 2 to move to next location

(Or S + 3-digit location to move to elsewhere)

Program with LCD LCD layout is slightly different

  1. scrolls left 2 scrolls right 5 scrolls data up 9 scrolls data down

Exit Program 1. Hold 8 key 3 seconds, keypad will flash and beep

  1. Enter 4 5 6 7 8 9, display should return to normal

Lockout Bypass (Make sure panel is not trying to communicate)

  1. Run jumper to pins on enpty chip socket marked X (see wiring diagram)

  2. Press 9 for 3 seconds, then 8 for 3 seconds and enter 456789

  3. LED display should show --- , remove jumper and program normally

(wire MUST be removed to view or change locations)

Zone Page

If Fire Trouble occures after exiting program re-enter and set 083 to 5, power down then back up

IF something screws up power down then up, you have 30 seconds to get into programming

To program REPORT OPTIONS in locations 166 to 185 see below Diagram

Add location together and enter sum in location

Example: Enable Reporting for Zones 1 to 6, Keypad Panic, Fire, Duress, Fire Trouble, AC Fail and Low Bat

  1. Press B/A then 1 6 6 to move to Location 166

  2. Add ZONE 1 (1) + ZONE 2 (2) + ZONE 3 (4) + ZONE 4 (8) = 15 or F, enter the F in location 166

  3. Press 2 key to move to 167

  4. Add Zone 5 (1) + Zone 6 (2) + Keypad Panic (8) = 11 (B), enter the B in location 167

  5. Press 2 key to move to 168

  6. Add FIRE ZONE (1) + AMBUSH (2) + FIRE TROUBLE (4) = 7, enter the 7 in location 168

  7. Press 2 key to move to 169

  8. Add AC FAIL (2) + LOW BAT (4) = 6, enter the 6 in location 169
