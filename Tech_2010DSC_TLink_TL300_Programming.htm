Program Notes Most programming will involve locations 001, 003, 007 and maybe 011, all others can be left default

Enter Program Enter DSC panel programming (* 8 5555 or * 8 1555 or * 8 5010)

Enter 851

Move to location Enter 3-digit location then enter data, # may be required to exit location

(Programs exactly like a DSC panel)

Exit Program #

Default Panel 1. Power T-Link down

2. Connect jumper from PGM1 to IN1

  1. Apply power, wait 20 seconds then power down and remove jumper

or

  1. Go to location 999

  2. Enter 00 to default module

  3. Enter 55 to reboot module

Status LED 1 Blink = Network Absent

2 Blinks = Account is set to default FFFFFF

3 Blinks = T-Link is not receiving heartbeat from central station receiver

  1. Blinks = Not receving polls from panel through PC-Link Interface

  2. Blinks = One of T-Link Inputs (IN)) is in alarm condition

  3. Blinks = T-Link failed to communicate with Receiver #1

  4. Blinks = PC5108 Module is not responding to T-Link

  5. Blinks = PC5108 Tamper has been activated

  6. Blinks = T-Link failed to communicate with Receiver #2

  7. Blinks = System was armed by key switch zone

  8. Blinks = T-Link is being programmed remotely

  9. Blinks = T-Link is being programmed locally

  10. Blinks = T-Link unable to connect to Receiver #2 on power-up
