ITI Caretaker Program Sheet

Format 4/2 see bottom of page for all signals

Programming can only be done with telephone, NOT keypad

Enter Program Toggle Program Switch to right, phone will say "Program Memory"

OR * + Master Code + # + 7 + <Service Code>

Entering programming with Service Code disconnects person on other end of phone

Entering programming with dip switch puts person on other end of phone on hold

Exit Program Dip Switch Method flip switch back to OFF

If using Service Code enter * 99

If some Dufus got into Keypad instead of Panel programming * 99

House Code 8 switch module right side of panel, add to get house code

Left = OFF - Right = ON

Switch 1 = 1

Switch 2 = 2

Switch 3 = 4

Switch 4 = 8 Example: House Code of 067 = 1, 2 & 7 ON, all others OFF

Switch 5 = 16

Switch 6 = 32

Switch 7 = 64

Switch 8 = 128

Locations (Enter location + Value)

* 01 4-digit Master Code

  • 02 Two-digit Duress Code (Usually make 99)

  • 03 Entry Time (06 to 58 seconds)

  • 04 Exit Time (06 to 58 seconds)

  • 05 to 16 Zone Programming (See Program Sensors below)

  • 17 Trouble Annunciation (OFF = One Time, ON = 24 Hours)

  • 18 Programs Hand-held Keypad (if applicable)

  • 19 Non ITI Pir Lockout (ON/OFF)

  • 20 Receiver Number (ex: 1 800 246 8910#)

For .8 second pause use hardwired touchpad with

hidden button located under first A in "Caretaker" logo

  • 21 Secondary Receiver Number

  • 22 4-digit Account Number (For 3-digit accounts press # to finish)

  • 23 Freeze Alarm Activation Point (Default = 42 F 00 = OFF)

  • 24 ****** Police Panic (ON/OFF)

  • 25 Siren Timeout (In Minutes)

  • 26 4-digit Service Code

  • 27 Off Premise Double Call Access (ON/OFF)

  • 28 Communicator Speed (Toggles between 10 & 20 Baud)

  • 29 Touch-Tone Sensitivity (ON = 100 ms, OFF = 45 ms)

  • 99 Exits program if Service Code used (Or turn Program Switch OFF to exit)

Add Sensor Remove 4 wire connector from ITI programmer

Connect sensors to main Panel in while programming

(Not required if using Service Code)

(Example: * 05 + 10 adds Zone 10 as Smoke, Panel says "Sensor 23 Program OK")

To read what is programmed press * + <Group number>

* 05 + Zone Number = Fire/Smoke Sensors

* 06 + Zone Number = Perimeter Instant Sensors

* 07 + Zone Number = Delay Zones

* 08 + Zone Number = Interior Instant Zones (will honor Entry Delay)

* 09 + Zone Number = Instant Motion Zone

* 10 + Zone Number = Delay Motion Zone

* 11 + Zone Number = Auxiliary Normally Open Zone

* 12 + Zone Number = Auxiliary Normally Closed Zone

* 13 + Zone Number = Audible Supervised Panic

* 14 + Zone Number = Audible Non-Supervised Panic (Supervised for Low Bat only)

* 15 + Zone Number = Silent Non-Supervised Panic (Supervised for Low Bat only)

* 16 + Zone Number = 24 Hour Zones

* 18 Programs Wireless Keypads (No value, must be hooked to panel to program)

Note: Hardwire Sensors must learn while open and use Zones 01 to 10 only

Hardwired (If available)

Hardwire Zone module plugs into bottom of board using Zones 1 to 12

Zones 1 to 10 are NC, 11 and 12 are NC or NO and require EOL

  1. Enter programming using Service Code (not dip switch method)

  2. Open then Close then Open zone to learn in

  3. Program zone type from above list

Change Zone # 1. Add sensor to new zone number in programming

2. Exit programming, do sensor test, new zone number is added, old removed

Delete Sensor 1. Plug Sensor into main board with programmer pigtail

(Not required if using Service Code)

2. Enter * + Group of Sensor to Delete (05 to 16) + Sensor Number (2-digits)

3. Keypad should say "Sensor xx is OFF"

Example: to delete Zone 10 Instant door enter * 06 # 10

Signals "1" = Fire Zones (Sensor Group 05)

"2" = Panics/Duress (Sensor Group 13 14 15 and ******* Panic)

"3" = Intrusion Zones (Sensor Group 06 07 08 09 10 16)

"4" = Auxiliary Zones (Sensor Group 11)

"5" = Auxiliary Panic (Sensor Group 12)

"6" = Freeze Detection (Energy Saver)

"7" = Trouble (Sensor or System Troubles)

"8" = Phone Test

"9" = System Restore/Cancel
