Alarm.com GE Simon 3 60-971 Weblink/Skynet Module

Information 1. All registration is done at the alarm.com web page www.alarm.com

2. Uses Skytel or Weblink radio paging networks, is not cellular

3. At this time Monitronics does not support Alarm.com monitoring

4. Allows monitoring and control of Simon 3 using a web page

5. Panel monitoring routed through Alarm.com to our receivers

6. Supported by Simon 3 v3.5 and later

  1. Requires special Simon 3 backplate and battery harness supplied by alarm.com

Installation On powerup the panel auto-recognizes the alarm.com module and installs it as

Zone: 24

Zone Type: 28

Serial Number: 000000A (I think)

Description: "Remote Module"

Equipment 60-971 Skytel version

60-971W Weblink version
