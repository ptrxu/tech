Moose Z1200/Z1250/Z2000 Alpha Description Programming

Program 1. Press function key under ARROW until flashing cursor is within " "

2. Press number keys to display letters, keep pressing until desired letter found

(Example: to program an S press 7 four times)

3. Press function key under ARROW to move to next character

4. To delete press * to move quotation mark to cursor

  1. Press NEXT to finish, * to exit
