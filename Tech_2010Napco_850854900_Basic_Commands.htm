Napco 850/854/900 Basic Commands

System is Chip Programmable only, it cannot be downloaded or programmed from keypad

Zones 850 - 8 zones

900 - 9 zones

Battery 12 volt rechargeable Gel-Cell inside main panel

AC Transformer is 16.5 volts, screwed into outlet somewhere in house

Wireless No

Downloadable No

Panic Button Police Panic only, * & #

Zone Status STATUS light blinks if zone open, press 3 key for 3 seconds to display

Arm 4-digit User Code (MUST have steady STATUS light)

Disarm 4-digit User Code (If wrong code entered wait 5 seconds, try again)

Silence Alarm 4-digit User Code

Clear Alarm Press 9 for 3 seconds

Flashing P Customer tried arming without steady STATUS light, disarm and try again

Bypass Zone Press [S] + zone (1, 2, 3 etc), BYPASS light turns on

Group Bypass (if enabled) Many interior zones can be bypassed by pressing [S] twice

Alarm History Hold [S] key for 3 seconds to view last alarm

Smoke Reset Press 9 for 3 seconds

Door Chimes Press 5 for 3 seconds

Instant Doors Press 4 for 3 seconds before or after arming, STATUS light flashes rapidly

Phone Test 1. Press 6 for 3 seconds to start, keypad will begin beeping

2. If test is okay beeping will stop, failed test sounds a steady tone

3. Press 9 for 3 seconds to stop steady tone in case of failure
