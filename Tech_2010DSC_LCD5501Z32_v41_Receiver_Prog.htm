Program 1. Enter 902 in panel programming and wait 3 minutes to enroll module to panel

2. Enter location 804

3. Enter 2-digit sub-location and enter data

(6-digit entry is required on all but PC501 v1.0 which requires 5-digit entry)

(Use "ESN" Number from sticker on sensor, if sensor only has SN number enter 0 as 1st digit)

  1. st Digit of serial number = Transmitter Type

  2. = Door/Window

  3. = Motion Detector

  4. = Smoke Detector

  5. = Pendant Panic

  6. = Keyfob

ESN to SN (If WS5010 v1 and 2 or PC5010 v1)

Conversion 1. The 1st digit of ESN number is also 1st digit of SN

2. Open up calculator in Microsoft Windows

3. Click on VIEW at top of calculator and select SCIENTIFIC

4. Click on HEX button at top left of calculator

5. Input the 2nd, 3rd and 4th characters or numbers of the ESN number

6. Click on DEC button at top left of calculator

  1. These are the last 4 digits of the SN number

Example: ESN 26DA4A would be SN 21754

  1. If 0 is 2nd digit use 3rd and 4th digits into calculator then add 00 to beginning of the result

Example: If ESN is 6039E9

  1. The 1st digit (6) of ESN number is also 1st digit of SN

  2. Click on VIEW at top of calculator and select SCIENTIFIC

  3. Click on HEX button at top left of calculator

  4. Enter 3rd and 4th digit (39)

  5. Click DEC button, the result should be 57

  6. The SN number will be 60057

Hex Program A = * 1 * B = * 2 * C = * 3 * D = * 4 * E = * 5 * F = * 6 *

If combining press * at end of hex string (example BC would be *23*
