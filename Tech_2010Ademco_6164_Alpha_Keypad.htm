Ademco 6164 4-Zone Alpha Keypad Program Sheet

Information Emulates a 4-zone 4229 Relay/Zone expander.

Can be used on any Ademco panel that compatible with a 4229 (Vista).

Keypad can be used on Vista-40 and higher panels but zone expander cannot be used.

CPU LED (Yellow) CPU failure of keypad

Tamper switch reports as 4229 case tamper if Zone/Relay option is enabled

Tamper switch reports as a keypad tamper if Zone/Relay option is disabled

Power Ratings 190mA (Backlight On, Sounder On)

115mA (Backlight On, Sounder Off)

55mA (Backlight Off, Sounder Off)

Relay is rated for max 1A, 24V AC/DC

Enter Program 1. Press 1 and 3 together for 5 seconds within 60 seconds of power up

2. Programming begins at "Keypad Address", * saves and moves to next location

Exit Program Press # at end of last program location (Language)

Location Display Default Description

  1. Keypad Address "CON ADDRESS = XX" 31 Enter 01 to 31 then * for next

Vista-10SE/Vista-20SE = 31

(Enter 00 then 01 to 31) Vista-15P/20P = 16 to 23

Vista-40 = 01-15

Vista-50P and above = 01-30

  1. Expander Enable "ZONE EXPANDER" 0 1 Toggles ON/OFF then * for next

"OFF [ 1 = ON]"

  1. Zone Expander Addr "EXP ADDRESS = XX" 01 Enter 00 to 15 then * for next

Vista-10SE/Vista-20SE = 01

Vista-10P/15P = 07 to 08

Vista-20P = 07 to 11

  1. Zone Type "ZONE TYPE" EOLR 1 Toggles between EOLR and

"EOLR [ 1 = DBALAN]" Double Balanced then * for next

  1. Tamper "TAMPER" OFF 1 Toggles tamper ON/OFF then * for next

"OFF [ 1 = ON]"

  1. Display "DISPLAY" ON Enter option then * for next

"1=ON 2=KEY 3=DIS" 1 = Display is always active

2 = Blanked 45 seconds after keypress

3 = Blanked after armed until disarmed

  1. Sounds "SOUNDS" All 1 Toggles All or Alarms only then * for next

"ALL [ 1 = ON]"

  1. Language "LANGUAGE?" ENG Press # to select NO and exit keypad programming

"* = YES # = NO" or press * to select language

"DEU ENG ESP FR" # Toggles between languages then * to save and exit

"IT NL RUS" DEU = German

ENG = English

FR = French

IT = Italian

NL = The Netherlands?

RUS = Russian
