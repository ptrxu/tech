DSC EV-DW4927 Shock Sensor

Information Battery: CR123A Lithium

Shock sensor and door switch

Pre-warn capability by audio sounder

Test by hitting window with palm of hand

Jumper Jumper 1 ON = Normal

Jumper 1 OFF = Test mode

Jumper 2 ON = Pre-Warn capability enabled

Jumper 2 OFF = Pre-Warn capability disabled

Jumper 3 ON = North American RF receiver

Jumper 3 OFF = EU (Europe) RF Receiver

Adjustment 1. Remove jumper 1

2. Gently hit window with open palm, gradually increase pressure

Quick beep and red LED flash = Pre-warn threshold was reached

Long beep and green LED flash = Sensor activation

  1. Replace jumper 1 (not sure if a dial exists to adjust)

Programming Serial number for reed switch is same as shock sensor +1

Enter as 2 wireless zones in 804 if using both sensors
