Telular TelGuard TG-1 General Information

Information 1. Designed for cellular only accounts, TG-1 acts like a receiver and reads output from panel dialer

2. Can interpret 4 or 10-digit Contact ID

3. Can interpret 4 to 10-digit SIA

4. Dialer capture on panels using SIA require

4. Sends exactly the same signals as panel

5. Can be configured to send to same phone and account as panel or a specific account and phone

(configured by web page)

Programming (Usually never required)

  1. Connect handset or plug house phone into J7

  2. Connect power to the TG-1, handset/phone should sound 2 beeps

  3. Press # then * to enter program mode, handset/phone should sound 2 beeps

  4. Enter address below, handset/phone should sound 2 beeps

  5. Enter new 1 or 2 digit value, handset/phone should sound 2 beeps

  6. When finished exit programming with *, handset/phone should sound 2 beeps
