Ademco 6151 Fixed English Keypad with 1-Zone Expander

Information Will not work with 4110, 4140, 4150, 4160, 4180 and Vector panels

Emulates a 1-zone 4229 Zone expander

Can be used on any Ademco panel that compatible with a 4229 (Vista)

Keypad can be used on Vista-40 and higher panels but zone expander cannot be used.

Tamper switch reports as 4229 case tamper if Zone expander is enabled

Tamper switch reports as a keypad tamper if Zone expander is disabled

Power Ratings 80mA (Backlight On, Sounder On)

50mA (Backlight On, Sounder Off)

35mA (Backlight Off, Sounder Off)

Enter Program 1. Press 1 and 3 together for 5 seconds within 60 seconds of power up

2. Programming begins at "Keypad Address", * saves and moves to next location

Exit Program Press # at end of last program location (Language)

Location Description Default Options

  1. 31 Keypad Address 31 Enter 01 to 31 then * for next

Vista-10SE/Vista-20SE = 31

(Enter 00 then 01 to 31) Vista-15P/20P = 16 to 23

Vista-40 = 01-15

Vista-50P and above = 01-30

  1. E0 Zone Expander Enable 0 = OFF Enter option then * for next

0 = Off

  1. = On

3. 107 Zone Expander Address 07 Enter 00 to 15 then * for next

Vista-10SE/Vista-20SE = 01

Vista-10P/15P = 07 to 08

Vista-20P = 07 to 11

4. A1 Zone Type 1 = EOLR Enter option then * for next

0 = Standard EOLR

  1. = Double Balanced

5. b0 Tamper Button Enable 0 = Off Enter option then * for next

0 = Tamper disabled

  1. = Tamper enabled

6. c1 LCD Display 1 Enter option then * for next

  1. = Display is always active

  2. = Blanked 45 seconds after keypress

  3. = Blanked after armed until disarmed

7. d1 Keypad Sounds 1 = All 1 Toggles All or Alarms only then * to exit
