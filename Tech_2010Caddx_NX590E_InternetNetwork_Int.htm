Caddx NX-590E Internet/Network Interface

Information Works with all GE Caddx NX-series panels

Used for IP signals to Osborne-Hoffman Network Receiver

Used for remote control by GE's Premise Connect Web Page

Program Sheet from vE document, NX-590E v1.6

Power 12VDC from panel

90mA standby, 120mA during communication

Enter Program 1. * + 8

2. 9713 (default, if 6-digit Code option is enabled use 971300)

3. 79 + #

Move to Location Location + #

Enter Data * saves data and moves to next segment, # exits at any time

Example: Toggle Options

  1. Press 0 + # (Moves to location 0)

  2. Press 1 through 8 (Toggles options 1 to 8 On or Off)

  3. Press * (Saves data and moves to 2nd segment)

  4. Press 1 to 8 (Selects option 1 to 8)

  5. Press * to save and exit

  • saves data and moves to next location within a segment, # exits at any time

Example: Toggle Options

  1. Press 91 + # (Moves to location 91)

  2. Enter 1st digit

  3. Press * (Saves data and moves to 2nd digit)

  4. Enter 2nd digit

3. Press * (Saves data and moves to 3rd digit)

4. Enter 3rd digit

  1. Press * to save and exit

Exit Program 1. Press # to exit any program location

2. Press EXIT key twice

Note: Panel may blip the siren for 1 second

Premise Connect 915 #
