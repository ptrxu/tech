Enter Program (All programming must be done from Keypad addressed as #1)

(Entering programming cannot be done if panel is trying to communicate)

  1. Press FUNCTION to enter system menu

  2. Press FUNCTION until display reads ACTIVATE PROGRAM

  3. Press YES (INTERIOR), system returns to READY

  4. Enter Installer Program Code (printed on sticker on circuit board)

  5. Display should read "PROGRAM 2" then "ADDR = 000 DATA = "

Jump to Location 1. Press 1 or 2 key to move cursor until above 1st program location digit

2. Press 4 to scroll digit UP or 7 to scroll digit DOWN until desired digit (0 to F) is found

3. Press 2 key to move cursor right to 2nd program location digit

4. Press 4 to scroll digit UP or 7 to scroll digit DOWN until desired digit (0 to F) is found

5. Press 2 key to move cursor right to 3rd program location digit

  1. Press 4 to scroll digit UP or 7 to scroll digit DOWN until desired digit (0 to F) is found

  2. Press 0 to read what is in location ( . in a location = Blank)

Program Location 1. Press 1 or 2 key to move cursor until above Left Data digit

2. Press 4 to scroll digit UP or 7 to scroll digit DOWN until desired digit (0 to F) is found

3. Press 2 key to move cursor right to Right Data digit

4. Press 4 to scroll digit UP or 7 to scroll digit DOWN until desired digit (0 to F) is found

5. Press ON/OFF to save (display will briefly flash)

Previous Location Press INTERIOR key

Next Location Press INSTANT key

Blank Location Press 5

Exit Program Press RESET, may need to enter Installer Code if changed during programming
