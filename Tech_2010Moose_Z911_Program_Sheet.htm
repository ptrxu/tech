Enter Program 1. Close (Hook) SW-1 switch

2. Press 9 + 98765 (Keypad should sound 3 beeps)

  1. Press 0 (Keypad should sound 3 beeps again)

  2. Enter 3-digit location (Do not press #)

OR

  1. Power down by black switch bottom left of circuit board

  2. Close (Hook) SW-1 switch and power up (No lights on Keypad)

  3. Press 0, panel beeps indicating Program Mode

  4. Enter 3-digit location (Do not press #)

(Unhook SW-1 when finished to prevent accidental entry by customer)

Account Number This panel can only send a 3-digit account number, Data Entry needs to issue accounts that start with 0

(Example: D3 0833 will work however D31833 will not)
