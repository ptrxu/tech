Electronics Line Summit 3208 Basic Commands

Zones 8

Downloadable Yes

Battery 12-volt rechargeable Gel-Cell inside main panel

AC Transformer is 15-volts plugged in somewhere in house

Wireless No

Partitions up to 4

Panic Buttons Police # + MENU

Fire 0 + MENU

Medical *E + MENU

Arm Away Press AWAY or enter User code

(Partition systems use SELECT 2 + 4 + (1 to 4 for Partition 1 to 4) + User Code)

Arm Stay Press STAY

(Partition systems use SELECT 2 + 3 + (1 to 4 for Partition 1 to 4) + User Code)

Arm Instant Press STAY or AWAY then *

(Partition systems use SELECT 2 + 1 + (1 to 4 for Partition 1 to 4) + User Code)

Disarm Enter User Code

(Partition systems use SELECT 1 + (1 to 4 for Partition 1 to 4) + User Code)

Cancel Alarm Press SELECT + 4 + 5 + User Code (Stops all Alarm Communication)

Silence Alarm Enter User Code or press SELECT + 4 + 4 + User Code

Reset Display Arm then Disarm

Bypass Zone Press SELECT + 3 + 1 + Zone (1 to 8) + User Code

Unbypass Zone Arm then Disarm or press SELECT + 3 + 2 + Zone (1 to 8) + User Code

(If this does not work try SELECT + 3 + 9 + User Code

View Memory (LCD Keypad Only)

  1. Press SELECT + 6 + 1 + User Code (PROGRAM will being Flashing)

  2. Press MENU to scroll to next event

  3. After last event system returns to normal

Clear Memory (LCD Keypad Only)

Press SELECT + 6 + 3 + User Code (PROGRAM will turn ON and OFF)

Force Bypass Press SELECT + 2 + 2 (Arms if Zones are Open)

Smoke Reset Press SELECT + 4 + 3 + User Code

(If this does NOT reset smoke detector set for service, smoke is bad)

Door Chime On Press SELECT + 3 + 3 + User Code (Not available on some systems)

Door Chime OFF Press SELECT + 3 + 4 + User Code (Not available on some systems)

Battery Test SELECT + 4 + 2 + 6

Phone Test SELECT + 4 + 2 + 3 + User Code

Sensor Test SELECT + 4 + 2 + 1

Siren Test SELECT + 4 + 2 + 2 (Sounds 1 second Siren Test)

System Test SELECT + 4 + 2 + 5 (Tests Panel Circuitry)

Set Time (LED Keypads)

  1. SELECT + 4 + 1 + User Code

  2. Enter 6-digit Date (7/2/2000 would be 070200)

  3. Enter 4-digit Military Time (1:30PM would be 1330)

  4. System automatically exits

Set Time (LCD Keypads)

  1. SELECT + 4 + 1 + User Code

  2. Enter 4-digit Military Time (1:30PM would be 1330)

  3. Enter 6-digit Date (7/2/2000 would be 070200)
