AT&T 8000 Basic Commands

Zones 32 (0 to 31) all are Wireless

Frequency 40.66 to 40.70 Mhz

Batteries 12 volt rechargeable Gel-Cell in main panel, all sensors are 9-volt Alkaline

Downloadable Only to Dialer, NOT to disarm or program User Codes and Zones

Panic Button Police, press 2 square keys by ALARM logo on keypad

Arm Home 3-digit code + HOME (System beeps once, Main Panel shows -H)

(4 beeps = one or more zones open)

(4 beeps on close may mean hardwire zone open)

Arm Away 3-digit code + AWAY + OPEN (System beeps once, Main Panel shows -A)

Disarm 3-digit code + OFF (System beeps once, Main Panel shows -0)

Silence Alarm 3 digit code + OFF

Reset Alarm Press CLEAR DISPLAY on Main Control Panel

Reset Trouble Press TEST/DEMO on Main Control Panel

Bypass Open zone then arm, System will beep 4 times and auto-bypass zone

(Bypassed zone will be active again if zone is closed later)

Alarm Memory Press MESSAGES, last alarm will display, press CLEAR DISPLAY to clear

Finding 1. Enter User Code then OFF at keypad (Panel beeps once, shows - 0 )

Open Zones 2. At main panel press TEST/DEMO quickly (Display shows 8. 8. )

3. Quickly press all 4 buttons at same time (Display should show ".C.C")

4. Activate sensor, beeps indicate signal strength, Display shows Zone

3 Beep = Sensor Okay

2 Beeps = Sensor wireless reception is marginal

0 Beeps = Sensor failure or Sensor Battery is dead

5. Scroll through tested zones with MESSAGES button

  1. Exit by pressing TEST/DEMO again (Display shows - 0 )

Reset Smoke 3-digit code + OFF

Panel Displays -A = Armed Away

-H = Armed Home

-0 = Off

-0 = System is in User Test Mode

- - = No State Selected

FI = Unidentified Fire Transmitter

AA = System is in 5 minute wait mode after an alarm

CC = Main Panel Trouble

00 = Panel Low Battery

8.8. = Installer Test Mode

Nothing = No power to panel
