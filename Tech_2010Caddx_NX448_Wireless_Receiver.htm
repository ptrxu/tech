Caddx NX-408/NX-416/NX-448 Wireless Receiver

This is the original NX wireless receiver, most installations use the NX-E series receivers

The original NX-408/NX-416/NX-448 cannot be used with NX-8E panels

Information NX-4/NX-6 wireless zones can overlap hardwire zones, the hardwire zones will automatically be disabled

NX-8/NX-8E wireless zones can overlap hardwire zones but onboard zones must be disabled (Location 37)

NX-8/NX-8E claim blocks of 8 so if a sensor is learned into zone 9 then zones 10 to 16 must also be wireless

NX Panels can have multiple receivers

NX-8E = 8 receivers maximum

NX-8 = 4 receivers maximum

NX-6 = 3 receivers maximum

NX-4 = 4 receivers maximum

And up to 48 zones

NX-408E = 8 Zones maximum

NX-416E = 16 Zones maximum

NX-448E = 48 Zones maximum

Enroll Module 1. Enter Programming with * 8

2. Enter Installer Code (Default = 9713)

3. Press 0 # to enter Panel Programming

4. Press EXIT twice to Exit Programming

5. All modules will be enrolled (Service light turns ON for 12 seconds)

Enter Program 1. Enter panel programming with * 8

2. Enter Installer Code (Default = 9713)

3. Enter Module Address + # (Default = 35)

Go to Location Location + #

Enter Data Enter data then * to save and move to next section of a location

  • also steps through programming while viewing a location

  • also exits if at end of a location

# exits at any time

Example: To change Supervision Time

  1. Press 0 + # (Moves to location 0)

  2. Enter 12 + * (Changes Burg Supervision from 24 to 12 hours)

  3. Enter 5 * (Changes Fire Supervision from 4 to 5 hours and exits)

Learn Sensor 1. Enter Programming with * 8

2. Enter Installer Code (Default = 9713)

3. Enter Module Address + # (Default = 35, is determined by board dipswitches)

  1. Press 50# and turn lights 1, 4 and 5 ON then *

  2. Press 51# and enter starting zone bank (0 to 5) + *

0 = Zone 1 to 8 (NX-4/NX-6/NX-8 Panels)

  1. = Zone 9 to 16 (NX-6/NX-8 Only)

  2. = Zone 17 to 24 (NX-8 Only)

  3. = Zone 25 to 32 (NX-8 Only)

  4. = Zone 33 to 40 (NX-8 Only)

  5. = Zone 41 to 48 (NX-8 Only)

  6. Press 49# and enter 1st zone number to learn within above specified block then *

(3 beeps = zone is not within specified block in step 5 or a sensor is already learned in that position)

  1. Activate Tamper on each sensor, if Keyfob press ARM and DISARM

Keypad should chime after learning each device

3 beeps = Sensor already learned or a sensor already in that location

7a. To ABORT and go back to step 5 press 0# 0*

  1. Once all sensors are learned press EXIT once to leave wireless receiver programming

  2. Press 0# to enter panel programming and program sensor type

  3. Press EXIT twice to leave panel programming

Delete Sensor 1. Enter panel programming with * 8

2. Enter Installer Code (Default = 9713)

3. Enter Module Address + # (Default = 35)

4. Enter Zone to delete then # (Zones 1 to 48)

5. Turn light 1 OFF then * to save

6. Press # to exit

7. Press EXIT twice to exit programming

Sensor Trouble 1. Enter panel programming with * 8

2. Enter Installer Code (Default = 9713)

3. Enter Module Address + # (Default = 35)

4. Enter Zone (1 to 48) then #

5. View Lights 5, 6 and 7 for trouble

5 ON = Xmitter Not Lost

6 ON = Low Battery

7 ON = Last Signal Good

6. Press # to exit

7. Press EXIT twice to exit programming

Defaulting 1. Enter panel programming with * 8

2. Enter Installer Code (Default = 9713)

3. Enter Module Address + # (Default = 35)

4. Press 910#

5. Press EXIT twice to exit programming
