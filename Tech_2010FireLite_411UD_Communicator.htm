Fire-Lite 411UD Communicator

This is a 3-channel dialer that connects to a fire panel that has no communicator

The 411 can only be programmed by a PRO-411 DACT Programmer, the 411UD can be downloaded

Enter Program 1. Press MODE (Display goes blank)

2. Within 10 seconds enter 7764 (Display will show numbers entered)

3. Press ENTER/STORE key (Display shows "00_ ", Trouble LED blinks)

("00_ " Is 1st location in programming and data in that location)

Program Keys = Selects Operating Mode

= Go to next Address (Location)

= Go to previous Address

= Press once to go to Address 00

= Press twice then enter Address to go to then

= Saves Data and goes to next Address

Enter Data Enter data in each location (0 to F) + ENTER/STORE to save and move to next location

Exit Program MODE + 6676 + ENTER/STORE (Returns to Normal Mode)

Initiate Download MODE + 3696 + ENTER/STORE
