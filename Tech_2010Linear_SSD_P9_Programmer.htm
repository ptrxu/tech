Remote Connection

  1. At first Password prompt type PASS then ENTER

  2. At next password prompt type 01 then ENTER

  3. Display shows "File maintenance" press ENTER

  4. Arrow up to "modify", press ENTER

  5. Select file 1 (for 2-way Systems) or 2 (Standard Systems, no 2-way) then ENTER

  6. at "1700 Master file?" press "Y"

  7. At "Select Step" select 1 then press ENTER

  8. Press ENTER until you get to "Phone Number 1" enter Receiver Number, press ENTER

  9. At "Account Number" enter 3 or 4 digit Account Number (depends on format) then ENTER

  10. Press ESC , at "Save?" press Y

  11. Arrow up to "Remote Comm" then press ENTER

  12. Arrow up to "Write Comm" then press ENTER

  13. Type Customer phone number then press ENTER

  14. Enter Access Code (try 3193, needed if Dialer locked out in previous programming)

  15. Programmer will dial phone, ring twice, hang up and call again, panel should answer

  16. At "Enter File Number" Enter 1 or 2 (depending on system to be programmed)

Download will start then show "Set Time & Date?" press ESC to disconnect

Local Connection

  1. Do above Remote Connection steps, instead of "Remote Com" use "Local Com"

  2. Plug in RJ Cord from panel directly into back of P9 Programmer

  3. Arrow up to "Write Comm" then press ENTER

  4. Display will say "Press button on Dialer" press Red button on TOP of Dialer Main board

  5. Display asks "1 - 8?" select template to download, Programmer will being downloading

  6. Press ESC when "Set Time & Date" prompt to disconnect

(Locked out Dialers allow downloading if using Local Connection method)
