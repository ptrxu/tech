CEI Capricorn EZ-8 Basic Commands

Zones Don't know yet, all are wireless

Battery 12 volt Gel-Cel (Non-replaceable)

Keyfob (Non-replaceable)

AC Transformer is 16 VAC 750mA

Door/Window 12V Lithium battery

Smoke Detector 9 Volt

Downloadable No

Panic Buttons Press both "E" keys at same time

Arm Away Press AWAY

or press AWAY + User Code

Arm Stay Press STAY

or press STAY + User Code

Arm Night Press STAY twice

or press STAY + User Code + STAY again

Disarm Enter User Code

Clear Alarm Press 4 & 9 keys at same time to silence protest beeps

Press 5 & 0 keys at same time to clear alarm display

Silence System Press 4 & 9 keys at same time to silence protest beeps

Bypass Zone Hold 3 key until 2nd beep, enter User Code + 2-digit Zone (01 to 52)

(Display should show b + 2 digits for zone bypassed, clears if disarmed)

Reset Smoke Hold 4 key until 2nd beep

Door Chimes Press 2 & 7 keys at same time + User Code

Test System 1. Press 1 & 6 keys at same time + User Code (all 3 lights should begin flashing)

2. Press "E" keys to test siren

3. Open all sensors, panel will beep twice to confirm activation

4. Press 1 & 6 keys at same time to end system test

User Code 1. Press 3 & 8 keys at same time + User Code

(only 1 Code) 2. Keypad will sound steady tone

3. Enter new 4-digit User Code within 10 seconds
