Troubleshooting 1. X-10 module on wrong "leg" of AC wiring and can't communicate with Panel

2. Wrong House Code or Unit ID

Garage Opener (Requires X-10 Universal Module) CLICK HERE for Universal Module

(* on Keyfob) 1. Open Cover and enter Master Code

2. Press ADD then LIGHT CONTROL to assign House Code A to F (usually leave at A) + DONE

(This must match Universal Module dials connected to garage door)

3. Press ADD then LIGHT CONTROL then UNIT# until correct Unit ID sounds

(Must match X-10 Universal Module dials as well)

4. Press SENSORS until correct sensor is heard (Keyfob)

  1. Press DONE to save and close cover

(Universal Module switches must be set for Momentary and Relay Only)

Program Lights (Requires Lamp Module) CLICK HERE for Lamp Module

  1. Open Cover and enter Master Code

  2. Press ADD then press LIGHT CONTROL to assign House Code (A to F) then DONE

(This must match Lamp Module dials connected to Light)

  1. Press ADD then LIGHT CONTROL then UNIT# to assign Unit Number

(This must match Lamp Module dials as well)

Unit 1 to 8 most common, controlled by LIGHTS TIME or SENSOR ACTIVATED

Unit 9 to 16 are activated by All On or All Off commands (Keyfobs etc)

Unit 10 Flashes Lamps to Arming Level (1 Flash = Disarm, 2 = Stay, 3 = Away)

  1. Press SENSORS to pick sensor that will activate Lamp Module then DONE

  2. Close cover

Lights Entry/Exit (Requires Lamp Module) CLICK HERE for Lamp Module

Activated 1. Open Simon Cover, enter Master Code

2. Press ADD then LIGHT CONTROL to assign House Code (A to F) then DONE

3. Press ADD + LIGHT CONTROL + UNIT# to assign Unit Number

4. Press ENTRY/EXIT DELAY

5. Close Cover

  1. Set dials on Lamp Module to House Code (Letter Dial) and Unit# (Number Dial)

(Lights turn ON for 4 minutes during Entry and Exit Delay)

Lights Time (Requires Lamp Module) CLICK HERE for Lamp Module

Activated 1. Open Simon Cover, enter Master Code

2. Press ADD then LIGHT CONTROL to assign House Code (A to F) then DONE

3. Press ADD + LIGHT CONTROL + UNIT# to assign Unit Number

4. Press TIME and select Start time with HOURS + & - and MINUTES + & - then DONE

5. Press HOURS and MINUTES to set End time then DONE

6. Close cover

  1. Set dials on Lamp Module to House Code (Letter Dial) and Unit# (Number Dial)

(Activate by pressing LIGHTS TIME ACTIVATED button)

PowerHorn (Requires PowerHorn Module) CLICK HERE for PowerHorn Siren

Siren Setup No programming necessary

Set Letter Dial on PowerHorn 1 Letter higher (if Simon is set at A then use B)

Set Unit Number Dial on PowerHorn (1 to 15) for Siren sounds only

or Unit Number 9 for Siren, Arming, Status and Trouble Beeps
