ITI SX-V Program Sheet

Format ITI only, can be programmed by CS-4000 Receiver

Do NOT power down system or programming will be lost, use switch on bottom right to turn off

On Powerup CPU will show RR for 2 minutes then CS in display window and beep every 60 seconds

Programming With wireless Keypad

Enter Program 1. Program House Code into Wireless Keypad with ITI Programmer (1 to 254)

2. In SX-V move Program Switch 2 UP to enter program (See wiring Diagram)

3. Press BYPASS on Remote Keypad to program House Code into Control Panel

(Control panel will show P, sensor number display will go blank for few seconds)

Exit Program Move Switch 2 DOWN

Press and HOLD BOTH FIRE, AUX or POLICE keys listed below, keypad will give initial beep then beep again

Programming by LCD Keypad

Enter Program 1. Program House Code into Remote Keypad with ITI Programmer (1 to 254)

2. Move Program Switch 2 UP to enter program (See wiring Diagram)

3. After several seconds keypad should say "Program Mode"

Move to Location Press STATUS to move forward, BYPASS moves backwards through Selections

Enter a Location Press COMMAND, enter Data, COMMAND again to save (Keypad blinks "OK")

Exit a Location Press FIRE to move back up to top of Menus (see below list)

Exit Program Move Switch 2 DOWN

Program Fields (Move to field, COMMAND enters, STAY or BYPASS toggles, FIRE exits)

PROGRAM SENSORS (programs sensor text, below example shows Zone 45 as Front Door)

Press BYPASS or STATUS until "Program Sensor" appears

Press COMMAND to enter (Display reads "PRG 01 - Sensor In ")

Press STATUS (Display reads "PRG ")

Enter Sensor to program (Display reads "PRG 45 ")

Press COMMAND (Display reads "A 00 ")

Enter 1st word or character from list below (Display reads "A 62 Front ")

Press COMMAND to safe and move to next (Display reads "B 00 ")

Enter 2nd word or character from list below (Display reads "B53 Door ")

Continue until finished then press FIRE (Display reads "Add 45 Front Door ")

Press COMMAND to save (Display reads "Add 45 Done ")

ADD SENSORS 1. Press BYPASS or STATUS until "PROGRAM SENSORS" appears

2. Press COMMAND to enter (Display reads "Add 00 Buddy Rep")

3. Press STATUS (Display reads "ADD")

4. Enter Sensor (00 to 99)

5. Press COMMAND to save (Display shows "ADD xx DONE")

6. Press FIRE to exit

DELETE SENSORS 1. Press BYPASS or STATUS until "DELETE SENSORS" appears

2. Press COMMAND to enter (Display reads "Delete 00 Buddy Rep")

3. Press STATUS (Display reads "DEL")

4. Enter Sensor (00 to 99)

5. Press COMMAND to save (Display shows "DEL xx DONE")

6. Press FIRE to exit

PROGRAM FEATURES Enable or Disable F00 to F17 (Toggle ON/OFF with BYPASS / STATUS)

F01 Tamper Polarity OFF = CPU Tamper is NC

ON = CPU Tamper is NO

F02 Exterior Siren Delay OFF = Exterior Siren activates with Interior Siren

ON = Exterior Siren waits 15 seconds

F03 Local Alarm OFF = System dials Monitoring Station

ON = Local System Only

F04 Low Battery Report OFF = Low Battery Reporting Enabled weekly

ON = Low Battery Reports only once until repaired

F05 Supervisory Report OFF = Supervisory Reports daily until repaired

ON = Supervisory Reports weekly until repaired

F06 Dialer Abort OFF = System reports Alarm then Cancel

ON = System aborts call if disarmed in 15 to 20 seconds

F07 Open Sensor Display OFF = Open sensors are not displayed

ON = Open sensors are displayed

F10 Dealer Sensor Test OFF = Standard Level 9 Sensor Test Enabled

ON = Siren sounds when data round is received

F11 Interior Siren Sound OFF = Interior Siren sounds Status and Alarms

ON = Interior Siren sounds only Alarms

F12 Restoral Reporting OFF = Restoral Reporting Disabled

ON = Restoral Reporting Enabled

F13 Not Used

F14 Hourly Phone Test OFF = Hourly phone line Testing disabled

ON = System checks phone line every hour

F15 Sensor Tamper OFF = Sensor Tamper Reporting Enabled

ON = Sensor Tamper Reporting Disabled

F16 Trouble Beeps OFF = Troubles sound 6 beeps every minute

ON = Trouble Beeps sound only if sensor is opened

F17 Direct Bypass OFF = Bypasses only unbypassed by Arming/Disarming

ON = Access Code + Bypass + Sensor unbypasses only

CPU TIME Set panel time (Military) (COMMAND to save, FIRE exits)

ACCESS CODE Enters Master Code (COMMAND to save, FIRE exits)

DURESS CODE Enter 99 (COMMAND to save, FIRE exits)

ACCOUNT NUMBER Last 5 of Account (no letters) (COMMAND to save, FIRE exits)

PHONE NUMBER 7 to 11 Digits (COMMAND to save, FIRE exits)

ENTRY DELAY 04 to 60 seconds (COMMAND to save, FIRE exits)

EXIT DELAY 04 to 60 seconds (COMMAND to save, FIRE exits)

HOUSE CODE 001 to 254, must match sensors (COMMAND to save, FIRE exits)
