Enter Program Move Slider Switch in Control Panel to Right "Program"

If using Programmer plug into program terminal on panel

If keypad programming enter 9713 (default)

LCD Keypad Press C 0 0, press 2 to select Panel Programming

Go to Location Location + #

Entering Data Enter data + * + # to save and move to next location

Example: Master Code in 0 to 3 is entered as 1* # 2 * # 3* # 4* #

If using Programmer save with 220# and unplug from panel
