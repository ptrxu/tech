How to Identify Grey Westinghouse keypad logo = PC5010 v1 with Westinghouse default programming

Blue Westinghouse/Westar logo = WS5010 v2 proprietary panel

Supported PC5132 Wireless Receiver (v2.1 or above)

Modules PC5400 Printer Module

PC5908/5928 Intercom

PC5580 Escort does not not work due to no partitioning (it uses WS5580 Escort)

PC5010 Keypads will not work (they might for a while but screw up later)

700mA Siren output

50mA PGM 1 output

300mA PGM 2 output

Report Format SIA only, hard coded into panel, report codes are pre-programmed
