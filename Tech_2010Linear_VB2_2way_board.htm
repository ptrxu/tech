Linear VB-2 2-way board

Information Compatible with DVS1200, DVS2400, DVS2408, Dual 824

Plugs into main board and provides panel and 2-way voice

2-way begins in VOX mode, if 1 or 3 keys are pressed will exit VOX mode for rest of session

VB-3 is panel voice only, will not do 2-way

Commands 1 = Talk

3 = Listen

  1. = Disconnect

Any other key = Extend 90 seconds
