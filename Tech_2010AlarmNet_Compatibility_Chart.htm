Note For panels that use a siren pulse for fire enable "Lynx Panic" option in AlarmNet Direct

This option causes the 7845GSM to look for a 1 second On/Off pulse. Also set zone 3

as "Report only when armed" to prevent a siren burst from activating Zone 3 (Medical/Duress)
