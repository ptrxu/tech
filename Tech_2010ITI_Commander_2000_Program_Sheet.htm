ITI Commander 2000/3000/SecurityPro 3000 Program Sheet

Format ITI or 4/2 (Always use ITI)

Unlock &cmem b609 (1st 4 digits)

plock <Installer Code> %43214321

Power Rating 290ma max (Sirens, motions etc)

60-483 Slimline Interior Siren 60ma

13-046 Exterior Siren 100ma

Interrogator 2-Way Module 55ma Inactive up to 300ma Active

Hardwire Standard Motion 20ma (average)

Hardwire Microwave Motion 30 to 50ma (average)

Hardwire Glassbreak 25ma to 35ma (average)

ADT Notes ADT's version has an incompatible format

Default Press Police until beep, press Auxiliary until beep

Installer Code 63211

Exit Program Press 99

Enter Program 1. Loosen screws under keypad

  1. Lower bottom tray

  2. Press 4321, all lights should flash, if not Installer Code has been changed

(Download with CS-4000 Receiver to change code back to default)

Exit Program Raise battery tray and tighten screws (Lights stop flashing)

Defaul Panel POLICE + POLICE + MEDICAL + MEDICAL ("Memory Goodbye")

House Code FIRE + FIRE + [001 to 254] (Use only with X-10 devices)

Duress Code POLICE + POLICE + [00 to 99] (use 99) ("xx OK!")

Siren Timeout MEDICAL + MEDICAL + [2 to 9] minutes ("xx OK!")

Entry Delay POLICE + POLICE + STATUS + [08 to 88 sec] ("xx OK!")

Exit Delay POLICE + POLICE + BYPASS + [08 to 88 sec] ("xx OK!")

Account Number FIRE + FIRE + STATUS + Last 5 of Account (0 as 1st digit if 4-digit account)

(A to F can only be added by CS-4000 Receiver)

Receiver Number FIRE + FIRE + BYPASS + [up to 18 digits] (for pause press FIRE FIRE)

(If you cannot change this Phone Lock is enabled)

Read Phone FIRE + FIRE + BYPASS ("Phone 1 8 x x etc")

Delete Phone FIRE + FIRE + BYPASS + 1 ("Phone OFF")

Report Format COMMAND + AUX + AUX + [00 to 03] 00 = ITI (Default)

  1. = 4/2 1400 Hz

  2. = 4/2 2300 HZ

Installer Code AUX + AUX + BYPASS + New Code twice (Also enables Phone Lock)

Hear All Sensors Press STATUS

Do not use "SAW" sensors with AA batteries, use only older "Crystal" 3.5 Lithium Sensors!

Learn Sensor 1. Press STATUS + Group [00 to 99] (Keypad says "Sensory Level xx")

2. Enter Zone Number [01 to 17] (Keypad says "Sensor xx")

Common Groups

  1. = Audible Panic 17 = Motions

03 = Silent Panic 20 = Delayed Motion

10 = Entry Exit 26 = Fire

13 = Instant

3. Learn sensor by doing the following

Door Sensor Remove cover

(hardwired contacts on wireless sensors learn while zone is open)

Glassbreaks Make a lot of noise (keys, clap etc)

Motions Remove cover, may require pressing button

Heat Sensor Rub hands until warm, place on sensor 30 seconds

Smoke Hold Test button 30 seconds until Panel beeps

Keyfobs Hold LOCK and UNLOCK buttons 3 seconds

Touchpads Press BYPASS button for 3 seconds

Hardwired Zn 18 If normally closed sensor learn with door open (close then open)

If normally open sensor learn with door closed (open then close)

4. If learned correctly keypad will say "Sensor OK" and go to next sensor

(Panel continues to learn in same group, to change group go back to step 1)

Delete Sensor Press BYPASS + Sensor [01 to 99] (Keypad says "Sensor xx Goodbye!")

Learn Keyfob 1. Learn like standard Sensor using Group 01 (Audible Panic) or 03 (Silent Panic)

or

  1. STATUS + STATUS + [1 to 4] + press LOCK & UNLOCK on keyfob

Delete Keyfob 1. BYPASS + Sensor [01 to 17] (if learned by Step 1 above)

or

2. BYPASS + BYPASS [1 to 4] (if learned by Step 2 above)

F26 ON = COMMAND 1 disarms OFF = Code Required (Default=OFF)

F27 ON = Panel Sounds disabled OFF = Sounds enabled (Default=OFF)

CS-4000 Commands

ACCT or ACCOUNT <XX-XX> Account Number (0 as 1st digit if 4-digit account)

ACCESS <4-digits> Programs Master Code

ALARMQ Views last 6 events of Alarm history

ANS 1 Connects CS-4000 to Panel

BATTLIFE <1 to 30> Battery Life? in Hours

BYPASS or BYP (00 to 16) Bypasses Sensors

DELETE or DEL (00 to 16) Deletes Sensors

DURESS <00 to 99> Hostage/Duress Code (usually 99)

ENTRY <8 to 88> Entry Delay (seconds)

EXIT <8 to 88> Exit Delay (seconds)

GROUP <zone> <zone type> Changes Zone type (Commander 17 2.6 only)

Freeze GROUP <zone> 01 L

Fire GROUP <zone> 02 D

Delay GROUP <zone> 04 L

Instant GROUP <zone> 05 L

Interior Inst Follower GROUP <zone> 06 L

Interior Follower GROUP <zone> 06 I

Silent Police Panic GROUP <zone> 12 A

Audible Medical Panic GROUP <zone> 01 B

HOUSE <1 to 255> Views House Code

INIT <00 to 16> Initializes Sensors

MACCESS <2 to 6> <4 digits> Programs Secondary Codes

(Commander 8 only has 1 secondary Code)

OPTION <20 to 27> <ON or OFF> Turns Functions ON/OFF (xx = 20 to 27)

F20 ON = Enable Interrogator OFF = Disables

F21 ON = DC Power Monitored OFF = Not Monitored

F22 ON = Dialing Format OFF = Pulse

F23 ON = Retain O/C only OFF = Retain All Events

F24 ON = Hardwire Loop NO OFF = Normally Closed OFF

F26 ON = Quick Disarm Enabled OFF = Disabled

F27 ON = Panel Alarm Mute OFF = Alarm Output Normal

Note: F20 and F27 must be ON for Interrogator

PLOCK ???? %XXXXXXXX Changes Installer Code

???? = Old Installer Code

XXXX = New Installer Code Twice

PFORMAT <0 to 2> Report Format (0=ITI, 1=4x2, 1400hz, 2=4/2, 2300hz)

PTFREQ <1 to 255> Automatic Phone Test in Days

PHONE <7 to 4 digits> Primary Receiver (D = Pause)

PHONE2 <7 to 4 digits> Secondary Receiver (Commander 17 only)

PLEVEL <1 to 3< Changes Arming Level

PLEVEL 1 = Disarm

PLEVEL 2 = Arm Home

PLEVEL 3 = Arm Away

RECALL Causes panel to disconnect then call back to CS receiver

REL Releases Panel from CS-4000 receiver

SIREN <1 to 9> Siren Timeout in Minutes

STIME <hh:mm> Supervisory Report Time

UPD Sends updated programming to panel

ZONES Lists all Zones including Panics & Supervisory
