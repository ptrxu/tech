Enter Program Press CODE key + * + 4612 + 1

or

Code key + * + 2468 + 1

Options Code + * + Installer Code + 1 = Enter Programming

Code + * + Installer Code + 2 = System Log View

Code + * + Installer Code + 3 = Unattended Download (Answer download request)

Code + * + Installer Code + 4 = On-Line Download (Send to downloader)

Move to location Press * + 2 digit location

Program location Press # after each digit and move to next location

Example: To program phone numberenter 1#8#0#0# etc

Exit Program Press STAY
