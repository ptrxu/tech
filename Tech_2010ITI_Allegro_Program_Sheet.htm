Program Keys

Program Info 1. Use LOCK/UNLOCK and */# keys to move to a location

2. For phone and account numbers enter data then # to save

3. # also toggles ON/OFF or SIA/CID

Default Panel 1. Enter programming

2. Press ARM twice until "CLEAR MEMORY"

3. Press # then enter Installer or Dealer Code

Learning Dialer 1. Remove dialer cover

2. Enter Panel programming

3. Press DISARM twice until "DEVICES" then #

4. Press # when "ADD" appears, display shows "01 - TRIP"

5. Press tamper on dialer 3 times, hold down on 3rd press

6. LED should flash 3 times, release tamper

7. Press tamper on dealer 2 times, hold down on 2nd press

8. LED should flash 2 times, release tamper

9. Press and hold tamper on dealer until LED flashes 1 time then release

  1. Dialer will then learn into system
