AlarmNet LynxDM-GSM Cellular Backup Information

Specifications Transformer 9VAC 15VA (Honeywell part# 1332) that comes with LynxR-I

External Antenna Required and included in kit

Current Draw 35mA standby, 500mA during transmit

Backup Battery 6V 3.1ah (Honeywell part# K14139)

Input Trigger V+ 2V to 13V

V- 0V to 1V

Information 1. Program for ECP Mode

2. Disable LynxDM-GSM Low Battery and AC fail reporting

3. Use keypad pigtail (Part #4632-4) if connecting to ECP header

4. Do NOT connect battery

Trouble Display bF = Panel cannot communicate with LynxDM-GSM or unregistered LynxDM-GSM

Check 103 or (Displayed by 6128/6150 Fixed-English Keypads, event buffer will display exact trouble listed above)

Fault 103 1. Module not communicating with control panel

2. Module front cover tamper

3. Primary cellular communication path failure (No cellular service)

4. Secondary cellular communication path failure (no alternate cell tower)

5. Cellular backup is not registered

  1. Cellular backup low battery

  2. Cellular backup battery charger failure

  3. Cellular backup AC failure

Cellular/Internet (Sent by the Cell/Internet unit in Ademco High Speed Express format)

Signals 01 = Fire Alarm/Fire Panic

02 = Burg Alarm/Police Panic

03 = Medical Panic

08 = Cover Tamper/Restore

C1 = AC Failure

C2 = Low Battery

C4 = Watchdog Failure or Reboot

C5 = Status Restore (Communication trouble restore)

T90 = 24 Hr Test/Module Tamper

U00 = Reboot
