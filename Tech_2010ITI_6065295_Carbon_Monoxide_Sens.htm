Caddx NX-497/ITI 60-652-95 Carbon Monoxide Sensor

Information Contains internal crystal sensor with no battery, uses 110VAC outlet for power

Green LED ON = Normal

Red LED ON = Alarm

Compatibility CareTaker +

Commander 2000

Concord (all versions)

Simon (all versions)

UltraGard

Programming 1. Unplug sensor from AC outlet

2. Program as type 34 Gas sensor

3. To learn plug sensor in, wait 5 seconds then press TEST 5 to 7 seconds

Testing Test button only functional within 67 seconds after power-up

Unplug sensor for 10 seconds then plug back in and press test button
