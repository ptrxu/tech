DSC Links 1000 v3 Programming

LINKS 1000 MUST be programmed by external programmer to operate

Phones Wiring Must be first in line, Tip & Ring to Links, T1 & R1 from Links to Panel

Wiring Links 1. Wire Zone from panel to SYS TBL on Links

2. Wire Zone common to TMP on Links

3. Jumper from TMP to AUX-GND on Links

4. 5.6k Zone Resistor from TMP to SYS TBL on Links

5. Wire PGMin to PGM 1 (Never PGM 2)

Note: Older Links label this terminal as S PGM

6. Jumper 5.6k Resistor from AUX+ to PGM 1 on panel

Programming Panel

WSS5010 1. Program Zone definition for "Links Supervisory" (10)

2. Program PGM 1 as 16

3. Turn on lights 1 and 7 in 361 to enable Links as backup

4. Turn on light 4 in zone attributes to allow bypassing in event of trouble or failure

PC1555/5010 1. Program Zone 8 for Links Supervisory in 001 as 09

2. Program PGM 1 as 16 to enable Links

3. Turn on lights 1 in 361 to report to Links as backup

4. Enter Links Trouble Report Codes in section 320 (zone # Links is connected)

5. Program Call Directions in 361 to 368

6. Turn on light 4 in zone attributes to allow bypassing in event of trouble or failure

Testing 1. Make sure LED L3 (Top) or L4 (2nd from top) Signal Strength LED is ON

(If L5 or L6 is on either raise Links higher or mount longer antenna)

2. Make sure LED1 (Middle LED) is flashing once every 10 seconds

3. Remove RJ Cord from Block, send an alarm signal

4. After 1 minute relay should click, LED1 is ON steady, Links should send

Trouble LED L1 2 flashes = AC Trouble to Links

3 flashes = Low Battery

4 flashes = Loss of Cellular Communication (Raise Links or use longer Antenna)

6 flashes = Tamper Trouble

7 flashes = Internal Trouble

8 flashes = PGM trouble (most common trouble)

(Check PGM programming and wiring (Usual cause)

(Check for Resistor across AUX+ and PGM 1)

No Signal Still? 1. Hook handset to panel, send signal, if nothing happens check panel programming

2. Call Links Number (printed on box), "Customer not in service" = Links not activated

(If Links is activated message should be "Customer has left the vehicle")
