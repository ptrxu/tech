Program Keys A Key moves from View Data Mode to Modify Data Mode

(View Mode) B Key moves to previous Group (1 to 14)

C Key moves to next Group (1 to 14)

  • Key moves to next location within a Group

CLEAR Key exits keypad programming (Unhook Program Latch to completely exit)

Program Keys A Key moves from View Data Mode to Modify Data Mode

(Modify Mode) B Key scrolls backwards through list of options, Press for YES/NO toggling

C Key scrolls forward through list of options and characters

  • Key saves data and moves one position to the right

CLEAR Key exits Modify Data Mode and goes back to View Data Mode
