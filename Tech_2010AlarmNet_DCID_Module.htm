Honeywell AlarmNet LynxR-EN DCID Dialer Capture Module

Information 1. Provides primary reporting using Telco dialer capture, designed for LynxR-EN using Contact ID

2. Original 7845GSM/IGSM will not work, must have ECP bus connector at bottom left on circuit board

  1. Comes with own enclosure or will fit in LynxR-EN in space to left of battery

  2. Accepts 4 or 10-digit Contact ID however AlarmNet will only send 4-digit Contact ID

  3. Does not power 7845GSM/IGSM, must use 6V 3.1aH battery and 16VAC transformer

  4. Be sure to switch yellow to green wires between N4632-4 pigtail and K14580LF cable

  5. Honeywell/Ademco panels must delete field *40 and remove any pauses in phone number

  6. DCID-EXT module will not provide dial tone in the event of a 7845GSM fault.

  7. Current use: 40mA standby, 80mA active
