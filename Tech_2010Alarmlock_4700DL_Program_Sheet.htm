Power Ratings 500ma total (including Aux Power, Fire, Keypads)

Keypad 40ma

Standard Motion 20ma (average)

Microwave Motion 30 to 50ma (average)

Glassbreak 25ma to 35ma (average)

Smoke Detector 40 to 50ma (average)

Enter Program CODE + */P + 4600 (Keypad sounds several beeps if incorrect)

Enter location */P + 2 digit location

Program location After every digit press #/P to move to next location

Example: To program phone number enter 1#8#0#0# etc

Exit Program Press INTERIOR

Default Panel Short "EEPROM RESET" pins together & power up (panel CAN be locked out)

Hex Program CODE+1 = A, CODE+2 = B, CODE+3 = C, CODE+4 = D, CODE+5 = E, CODE+6 = F
