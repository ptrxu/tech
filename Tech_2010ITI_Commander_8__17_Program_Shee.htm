ITI Commander 8/17 Program Sheet

Format ITI only

Enter Program Back bottom screw under keypad out until lights begin flashing

Exit Program Turn bottom screw under keypad in until flashing stops

Defaul Panel POLICE + POLICE + MEDICAL + MEDICAL ("Memory Goodbye")

House Code FIRE + FIRE + (001 to 254)

Duress Code POLICE + POLICE + (00 to 99) (Hostage = 1st 2 digits of code + 99)

Siren Timeout MEDICAL + MEDICAL + (2 to 9)

Entry Delay POLICE + POLICE + STATUS + (8 to 88 seconds)

Exit Delay POLICE + POLICE + BYPASS + (8 to 88 seconds)

Account FIRE + FIRE + STATUS + [xx-xxx] (0 as 1st digit if 4-digit account used)

(A to F can only be added by CS-4000)

Receiver Number FIRE + FIRE + BYPASS + <up to 14 digits> (For pause press FIRE FIRE)

Add Sensor STATUS + Sensor (2 digits) ("Sensor xx Okay")

Erase Sensor BYPASS + Sensor (2 digits) ("Sensor xx Goodbye")

Hear All Sensors Press STATUS

Program Sensor to zone number below to define what zone type is (1 sensor per zone)

CS-4000 Commands

ACCT or ACCOUNT <XX-XX> Account Number (0 as 1st digit if 4-digit account)

ACCESS <4-digits> Programs Master Code

ANS 1 Connects CS-4000 to Panel

BATTLIFE <1 to 30> Battery Life? in Hours

BYPASS or BYP (00 to 16) Bypasses Sensors

DELETE or DEL (00 to 16) Deletes Sensors

DURESS <00 to 99> Hostage/Duress Code (usually 99)

ENTRY <8 to 88> Entry Delay (seconds)

EXIT <8 to 88> Exit Delay (seconds)

GROUP <zone> <zone type> Changes Zone type (Commander 17 2.6 only)

Freeze GROUP <zone> 01 L

Fire GROUP <zone> 02 D

Delay GROUP <zone> 04 L

Instant GROUP <zone> 05 L

Interior Inst Follower GROUP <zone> 06 L

Interior Follower GROUP <zone> 06 I

Silent Police Panic GROUP <zone> 12 A

Audible Medical Panic GROUP <zone> 01 B

HOUSE <1 to 255> Views House Code

INIT <00 to 16> Initializes Sensors

MACCESS <2 to 6> <4 digits> Programs Secondary Codes

(Commander 8 only has 1 secondary Code)

PTFREQ <1 to 255> Automatic Phone Test in Days

PHONE <7 to 4 digits> Primary Receiver (D = Pause)

PHONE2 <7 to 4 digits> Secondary Receiver (Commander 17 only)

PLEVEL <1 to 3< Changes Arming Level

PLEVEL 1 = Disarm

PLEVEL 2 = Arm Home

PLEVEL 3 = Arm Away

RECALL Tells panel to Disconnect and call back

REL Releases Panel from CS-4000

SIREN <1 to 9> Siren Timeout in Minutes

STIME <hh:mm> Supervisory Report Time

ZONES Lists all Zones including Panics & Supervisory
