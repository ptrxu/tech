Ademco 6128RF/First Alert FA210RF Keypad/Receiver

Information Keypad operates as 5881M Receiver, max 16 transmitters

8 keyfobs can be added to keypad itself

Sensors cannot be closer than 18 inches to keypad

Does not support Encrypted keyfobs

Has internal 5800TM module for bi-direction keyfobs

Does not support 5828 wireless keypads

Can be added to older Ademco 4110XM, 4110DL panels to allow keyfobs

Later versions of the 6128RF may program like the 6150RF

Backlight Info All versions of 6128 and 6148 do not have backlight screen

Enter Program 1. Press 1 & 3 for 5 seconds within 30 seconds of power up

2. Keypad will flash "- - 0 0" (if anything else press * to exit)

3. Enter location to program (1 to 9)

4. Enter new data then * to save and exit back to "- - 0 0"

  1. Enter next location to program

Exit Program Press * until "- - 0 0" then *

Default Keypad 1. Press 1 & 3 for 5 seconds within 30 seconds of power up

2. Keypad will flash "- - 0 0" (if anything else press * to exit)

3. Press 9, keypad will flash EE

4. Press 1 to default

5. Keypad will beep 3 times display "- - 0 0", Press * to exit

Keyfob Loop Configuration

  1. Press 4, display shows "Ln"

  2. Enter Loop number to program (1 to 4)

  3. Enter action for this loop then * to accept and move back to "Ln" prompt

  4. Press * to move back to "d2" then * to exit back to "- - 0 0"

  1. = Disarm Default

  2. = Arm Away Loop 1 = Relay toggle On/Off

  3. = Arm Stay Loop 2 = Disarm

  4. = Arm Maximum Loop 3 = Arm Away

7 = Arm Instant Loop 4 = Arm Stay

# + 99 = Police Panic

# + 7 = Manually Start Relay a Action (Vista-20SE use # + 7 + Device)

# + 8 = Manually Stop Relay a Action (Vista-20SE use # + 8 + Device)

# + 71 = Activate Relay as programmed in Control (Vista-40 and above only)

# + 72 = Activate Relay as programmed in Control (Vista-40 and above only)

# + 72 = Activate Access Control Relay for Partition (Vista-40 and above only)
