DSC WLS900 Program Sheet

Enter Program 1. Press #, keypad should say "Enter Access Code"

2. Press * 8, keypad should say "Enter Installer Code"

3. Press down Arrow key for until display shows "Edit Installer Program Y/N?"

4. Press Up Arrow Key for Yes (Programming will be like DSC PC1500/1550)

Exit Program #

Enter location Enter 2-digit location, # may be needed to exit some locations

Add Modules (Remove batteries of Sensor before enrolling)

Commands Display shows

  1. Enter programming "Edit System Components Y/N?"

  2. Press Up Arrow Key (Yes) "Add Components to System Y/N?"

  3. Press Up Arrow Key (Yes)

  4. Display shows 0U = Number of WLS Door Sensors

"0U 0M 0S 0A 1K" 0M = Number of WLS904 Motion Sensors

"Serial # [ ]" 0S = Number of WLS906 Smoke Detectors

  1. Enter Module or Sensor Serial Number 0A = Number of Wireless Sirens

  2. Press Up Arrow Key (Yes) 1K = Number of Keypads

(Sounders will sound 2 short beeps)

  1. Install batteries in sensor

  2. Sounder will sound 2 short beeps

  3. Mount Module, sounder will beep twice again

  4. Display will be updated to show new components, press # to exit

Add Keypad (Remove batteries of Keypad before enrolling)

(Max 2) Commands Display shows

  1. Enter programming "Edit System Components Y/N?"

  2. Press Up Arrow Key (Yes) "Add Components to System Y/N?"

  3. Press Up Arrow Key (Yes)

  4. Display shows 0U = Number of WLS Door Sensors

"0U 0M 0S 0A 1K" 0M = Number of WLS904 Motion Sensors

"Serial # [ ]" 0S = Number of WLS906 Smoke Detectors

  1. Enter Keypad Serial Number 0A = Number of Wireless Sirens

  2. Press Up Arrow Key (Yes) 1K = Number of Keypads

(Sounder will sound 2 short beeps)

  1. Install Batteries in second keypad

  2. Press # key on 2nd Keypad "DSC RF Security System v2.20"

  3. Press # key again "Enter Controller Serial # [ ]"

  4. Enter Serial for 2nd keypad on 2nd keypad "Confirm Press [ Y ] Serial # [??????]"

  5. Press Up Arrow Key (Yes) on 2nd keypad (Sounder sounds 2 short beeps)

  6. Mount 2nd keypad on backplate (Sounder sounds 2 short beeps again)

  7. Display updates number of Keypads installed, press # to exit

Review Commands Display shows

Modules 1. Enter programming "Edit System Components Y/N?"

2. Press Up Arrow Key (Yes)

3. Press Down Arrow Key (No) until "List System Components Y/N?"

4. Press Up Arrow Key (Yes)

5. Display lists zones one at a time then Sounders, then Keypads

(Press * when viewing any device to show serial numbers)

6. Press # to exit

Hex Program A = * 1 * B = * 2 * C = * 3 * D = * 4 * E = * 5 * F = * 6 *

If combining press * at end of hex string (example BC would be *23*
