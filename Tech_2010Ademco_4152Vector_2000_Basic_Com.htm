Ademco 4152/Vector 2000 Basic Commands

Zones 29 Hardwired

Battery 12 volt rechargeable Gel-Cell inside main panel

AC Transformer is 18 volts, screwed into outlet somewhere in house

Wireless No

Downloadable No

Panic Buttons Police - * and #

Fire - Not available

Medical - Not available

Check Status Press * (READY)

Arm AWAY User Code + 2 (AWAY)

Arm STAY User Code + 3 (STAY)

Arm INSTANT User Code + 7 (INSTANT) (Arms doors instant but bypasses motions)

Arm MAX User Code + 4 (MAXIMUM) (Arms doors and motions instant

Quick-Arm (If enabled)

# + 2 (AWAY)

# + 3 (STAY)

Disarm User Code + 1 (OFF)

Silence Alarm User Code + 1 (OFF)

Bypass Zones 1. User Code + 6 (BYPASS) + Zone (01 to 29)

2. "bb" will appear indicating zones bypassed

To view bypassed zones

  1. Enter User Code + 6 (BYPASS)

  2. Display shows all bypasse zones

Alarm Memory 1. User Code + 0

2. System will display 10 days of previous alarms/troubles

  1. Exit with User Code + 1 (OFF)

Sensor Test 1. User Code + 5 (TEST)

2. Siren will briefly sound if a sensor is activated

3. End test by User Code + 1 (OFF)

Sensor Test 1. User Code + 0 (TEST) (Siren sounds for 3 seconds, keypad beeps every 15 seconds)

2. Open sensors, siren will sound

3. End test with User Code + 1 (OFF)

Reset Smoke User Code + 1 (OFF) twice

(If this does NOT reset smoke detector set for service, smoke is bad)

Door Chimes User Code + 9 (CHIME)
