CEI Capricorn EM300

Zones None

Battery 12 volt Gel-Cel (Non-replaceable)

Keyfob (Non-replaceable)

AC Transformer is 16 VAC 750mA

Downloadable No

Wireless Yes, keyfob only

Panics Police, Fire and Help (Medical), some units may only have Help button

Silence Alarm Press RESET button on back

Power Light OFF = No Power

  1. Blink = AC Power Failure

  2. Blinks = Backup Battery is low

  3. Blinks = Inside Fuse is blown

Telephone Light OFF = No Phone is connected

Slow Blink = Sending signal to Monitoring Station

Fast Blink = Failure to communicate

Programming Can only be done with an Arrowhead/Sentrol P4000 chip programmer
