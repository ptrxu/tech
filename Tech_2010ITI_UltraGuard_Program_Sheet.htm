ITI UltraGard/SecurityPro 5000 Program Sheet

Power Ratings 750mA Aux output maximum

750mA Siren output

Enter Program 1. Open and remove panel door, set PROGRAM/RUN switch down

2. Enter Installer Code (by default there is no Installer Code)

Exit Program Set PROGRAM/RUN switch to UP

Default Panel 1. Turn Power Switch OFF then ON

2. Hold MEMORY CLEAR button on upper left, system says "Memory Goodbye"

Panel Version Press User Code + STATUS + 7

Add Sensor 1. Press BYPASS or STATUS until LEARN SENSOR is displayed

2. Press COMMAND, keypad will display GROUP __ __

3. Enter GROUP <00 to 29, 32> + COMMAND button

4: Keypad will show SENSOR ##, enter sensor <01 to 76>

  1. Press COMMAND, keypad will read TRIP - ##, trip sensor

  2. Once sensor is learned system will move to next sensor or activate Tamper

  3. Press FIRE to exit

(Do not use SAW sensors on this system, use only older Learn Mode Sensors!)

Delete Sensor 1. Press BYPASS or STATUS until DELETE SENSOR is displayed

2. Press COMMAND button, Enter sensor to delete (01 to 76)

3. Press COMMAND button, keypad will show DEL ## OK

4. Press FIRE to exit

Add Touchpad 1. Press BYPASS or STATUS until LEARN TOUCHPADS is displayed

2. Press COMMAND button to display PRESS BYP TP-1

3. Press BYPASS on wireless touchpads and display will read One OK

4. Press FIRE to exit

Locations BYPASS scrolls forward

STATUS scrolls back

COMMAND enters Location and saves settings

FIRE exits Location

01 - Delete Text

02 - Program Sensor Text

03 - Learn Sensors

04 - Delete sensors

05 - Upper Sensors (Zones 77 and above)

06 - F Options (Functions, see bottom of page)

07 - Learn Touchpads (Learns wireless keypads)

08 - Phone Number (18 digits, press Police Panic for 1 second pause)

09 - Phone Format (ITI Format, 4/2 1400hz, 4/2 2300hz supported)

10 - Siren Time-out (01 to 30 minutes)

11 - Install Code (Default is blank)

12 - Account Number (5 digits, A to F only programmed by CS-4000)

13 - Entry Delay (008 to 120 seconds)

14 - Extended Delay (Exit) (affects Group 11 sensors, 01 to 8 minutes)

15 - Exit Delay (008 to 184 seconds)

16 - Activity Time-out (01 to 24 hours)

17 - House Code (001 (A) to 255 (P) Default = 001)

18 - Freeze Temperature (40 to 90 degrees)

19 - Set Temperature (32 to 99 degrees)

20 - Touchpad Options

21 - AVM Mode (Interrogator 2000) 01 = 1 ring callback mode

02 = 1 ring silent callback mode

03 = Instant Activation mode

09 = 1 Ring, Fire siren Shutdown

10 = One Ring Silent Fire Shutdown

11 = Instant Activation, Fire Siren Shutdown

22 - AVM Time-out (activation time) (030 to 300 seconds in 2 second increments)

23 - AVM Code (Blank/0001 to 9998, do not program this)

24 - HOM Setup

25 - Time Zone 05 = Eastern Time

06 = Central Time

07 = Mountain Time

08 = Pacific Time

10 = Hawaii Time

F13 ON = Low RF Battery Restoral Reports enabled OFF = Disabled

F14 ON = Hourly Phone Test enabled (TLM Enable) OFF = Disabled

F15 ON = Alarm Verification (2 zones must trip for report) OFF = Disabled

F16 ON = RF Trouble Beeps but allows Arming OFF = Will not allow arming

F17 ON = Sensor Tamper causes Alarm OFF = Disabled

F20 ON = AVM Interrogator Active OFF = AVM is Disabled

F21 ON = Immediate Trouble Beeps OFF = Trouble Beeps in 10 Hours

F22 ON = DTM Disaling enabled OFF = Pulse Dialing Enabled

F23* ON = Arm/Disarm & Time Change to Event Buffer OFF = All Events saved to Event Buffer

F25 ON = Keyfob arms to level 3 OFF = Keyfob arms level 1, 2 then 3

F32 ON = Keyfob * arms no delay OFF = Turns ESM On or Off

F33 ON = Downloader Access enabled OFF = Disabled

F35 ON = Phone Panics enabled (*7777, *8888, *9999) OFF = Disabled

F36 ON = Local Phone Access enabled OFF = Access disabled

F37 ON = Quick-Arm enabled OFF = Quick-Arm disabled

F40* ON = Fire Zone must trip twice for Alarm OFF = Only trips once for Alarm

F41 ON = Interior Siren Trouble shows on Keypad OFF = Interior Siren not Supervised

F42 ON = Sensor 70 to 80 Alarm tracking enabled OFF = Disabled

F43* ON = Demo Kit Enabled OFF = Disabled

F44* ON = Exterior Siren Latched when in Alarm OFF = Disabled

F45* ON = Zone 1 Keyswitch Arm/Disarm OFF = Zone 1 Normal

F46* ON = Daylight Savings Enabled OFF = Disabled

F47 ON = 4-Hour Silence on Fire Troubles OFF = Disabled

CS-4000 Commands

ACCT or ACCOUNT <XX-XX> Account Number (0 as 1st digit if 4-digit account)

ACCESS <4-digits> Programs Master Code

ANS 1 Connects CS-4000 to Panel

BATTLIFE <1 to 30> Battery Life? in Hours

BYPASS or BYP (00 to 16) Bypasses Sensors

CPUTIME (SET/STATUS) SET Sets panel time, STATUS reads panel time

DELETE or DEL (00 to 16) Deletes Sensors

DURESS <00 to 99> Hostage/Duress Code (usually 99)

ENTRY <8 to 88> Entry Delay (seconds)

EXIT <8 to 88> Exit Delay (seconds)

Event <0 to 32 / All / Clear> List All or Clear Event Buffer (v2 UltraGards only)

GROUP <zone> <zone type> Changes Zone type

Freeze GROUP <zone> 01 L

Fire GROUP <zone> 02 D

Delay GROUP <zone> 04 L

Instant GROUP <zone> 05 L

Interior Inst Follower GROUP <zone> 06 L

Interior Follower GROUP <zone> 06 I

Silent Police Panic GROUP <zone> 12 A

Audible Medical Panic GROUP <zone> 01 B

HOUSE <1 to 255> Views House Code

INIT <00 to 16> Initializes Sensors

MACCESS <2 to 6> <4 digits> Programs Secondary Codes

(Commander 8 only has 1 secondary Code)

PLOCK ???? %XXXXXXXX Changes Installer Code

???? = Old Installer Code

XXXX = New Installer Code Twice

PFORMAT <0 to 2> Report Format (0=ITI, 1=4x2, 1400hz, 2=4/2, 2300hz)

PTFREQ <1 to 255> Automatic Phone Test in Days

PHONE <7 to 4 digits> Primary Receiver (D = Pause)

PHONE2 <7 to 4 digits> Secondary Receiver (Commander 17 only)

PLEVEL <1 to 3< Changes Arming Level

PLEVEL 1 = Disarm

PLEVEL 2 = Arm Home

PLEVEL 3 = Arm Away

RECALL Has panel disconnect then call back

REL Releases Panel from CS-4000

SIREN <1 to 9> Siren Timeout in Minutes

STIME <hh:mm> Supervisory Report Time

UPD Sends updated programming to panel

ZONES Lists all Zones including Panics & Supervisory
