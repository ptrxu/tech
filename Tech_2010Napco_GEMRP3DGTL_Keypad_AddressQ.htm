Napco GEM-RP3DGTL Keypad Address/Quick Program Sheet

Keypad Address (set by back jumpers)

Keypad 1 = All Jumpers OFF

Keypad 2 = Jumpers 1 and 3 OFF, 2 ON

Keypad 3 = Jumpers 1 and 2 ON, 3 OFF

Keypad 4 = Jumbers 1 and 2 OFF, 3 ON

Keypad 5 = Jumpers 1 and 3 ON, 2 OFF

Keypad 6 = Jumper 1 OFF, 2 and 3 ON

Keypad 7 = All Jumpers ON

Misc options Cut Jumper A to disable touchpad backlighting

Cut Jumper B to disable LCD backlighting

Cut Jumper C to disable keypad sounder

Enter Program GEM-P816/1632 Installer Code by default is 456789

GEM-P3200/9600 Installer Code is printed on wiring diagram label

  1. Installer Code + FUNCTION

  2. Press INSTANT until

"17" (Standard 2-digit display Keypad)

  1. Press INTERIOR to enter "Easy Menu" Programming (See above "Quick Programming")

  2. or press RESET to enter "Advanced" Programming (see below "Direct Programming")

Exit Program Press RESET twice

Programming INTERIOR moves forwards

Keys INSTANT moves backwards

FUNCTION toggles between 4-digit location and 2-digit data

ON/OFF saves

RESET twice exits

0 Blanks a location (unless otherwise noted)

  • + 0 = 0

  • + 1 = B

  • + 2 = C

  • + 3 = D

  • + 4 = E

  • + 5 = F

Enter User Codes (enter 0 to 9, * 0 = Blank Space)

(Codes can be set for either 4 or 6 digits)

  1. Press FUNCTION to set cursor to User Code Number (01, 02 etc)

  2. Enter new User Code

  3. Press FUNCTION to move to Area 1 Options

  4. Enter new 2-digit Option

_ _ = Code Disabled (press * 0 = Blank Space)

_ 1 = Arm/Disarm

_ 2 = Arm Only

_ 3 = Service Code

_ 4 = Access Code

Add 8 to above to allow User Programming

Example: Code of 2222 with Arm/Disarm only would be 2222 BLANK 1

Press ON/OFF to save and move to next User Code or INTERIOR for next location

RF Transmitter Points

  1. Press FUNCTION to set cursor, enter Zone number

  2. Press BYPASS to enter Enroll Mode, LED's will begin flashing

  3. Enter Zone to program (09, 10 etc)

  4. Open Zone loop to program and install the battery

  5. Keypad should beep once indicating learned

  6. Move to next sensor or loop (Door sensors have 2 loops and can be 2 Zones)

  7. Press ON/OFF to save, INTERIOR for next

Keyfob Transmitters (system can use 8 keyfobs, these do not use zones)

  1. Enter Keyfob to program (01 to 08)

  2. Enter Area (Partition) of Keyfob (1, 2 or 0 to disable)

(GEM-P816 has only 1 partition)

  1. Enter 6-digit serial number on sticker of keyfob

  2. Enter 1-digit checksum on sticker of keyfob

  3. Enter Aux 1 (Bottom Left Button) 0 = None/Keyfob Disabled

(Arm/Disarm buttons are auto learned) 1 = Relay Group 1 Toggle

  1. Enter Aux 2 (Bottom Right Button) 2 = Relay Group 2 Toggle

9 = Keypad Panic

A (*0) = Keypad Aux

B (*1) = Instant

C (*2) = PGM2 Output Toggle

  1. Press ON/OFF to save, INTERIOR for next D (*3) = Access on PGM2 Output

Enter 6-digit Dealer (Installer) Code then ON/OFF to save

(ON/OFF saves, INTERIOR for next location)

Enter 6-digit Dealer Code again then ON/OFF to save

(ON/OFF saves, INTERIOR for next location)
