Linear DF-16 SSD Panel Program Sheet

1704 Dialer can only programmed with P-9 Programmer (See bottom of this page)

Formats Radionics HEX, Sescoa Standard, Sescoa Superspeed

Enter Program User Code + LEARN (1 Gong, 5 Beeps)

Exit Program Press OFF for 3 seconds

Compatible DF-10 Wireless Keypad (Linear DVS2400 Sensors likely won't work)

Wireless DF-31 Door/Window

Sensors DF-41 & DF-42 Keyfobs

DF-54 Motion Detector

DF-71 Smoke Detector

Add Sensor 1. Enter Zone (01 to 16), Zone light turns ON

2. Activate Sensor (press AUTO on keyfob)

1 Bing = Learned OK

Long Buzz = Another sensor already learned in that zone

Delete Sensor Enter Zone (01 to 16) then TEST (1 Bing, Zone Light turns OFF)

Interior Zones User Code + LEARN (1 Gong 5 Beeps)

Enter Zone Number + HOME + 7 + AWAY (1 Bing)

(Instant or Delay Zones selected by jumper under Transmitter)

Default System Hold LEARN button while plugging in AC Transformer

Defaulting the system doesn't default the dialer

(Learn Master Code again with * + New Code (1 to 5 digits) + OFF)
