DSC PC1500/PC1550 v3.0 Program Sheet Binary Programming

Power Ratings 475mA Aux output maximum

300mA PGM/Smoke Detector power

  1. A Siren output

Enter Program * 8 1500 or 1550 (READY, ARMED and SYSTEM Lights should flash)

Exit Program #

Move to location Enter 2-digit location, # may be needed to exit some locations

Battery Reset (Resets Low Battery threshold on panels with constant low battery display)

  1. Enter programming

  2. Press 00 + # + #

Default Panel 1. Power down then up, if board clicks panel is locked out (replace board)

2. If panel does not click leave power ON, short default pins and enter * 8 0000

  1. Keypad should show TROUBLE, BYPASS and MEMORY lights flashing

  2. Remove short from default pins and program normally

(Does not work on V4.0 and above panels)

[24] PGM Output

Default

04 I _0 _I _4 _I

  1. = Ground Start Pulse 07 = System Status (Arm/Disarm) Output

  2. = Utility Output no Access Code 5 sec (* 7) 08 = Strobe Output (Latched Alarm Output)

  3. = Utility Output any Access Code 5 sec (* 7) 09 = Fail to Com. Output Follows trouble #4

  4. = 5 sec Smoke Reset Pulse (* 7)

  5. = Courtesy Pulse (Follows E/E Times)

  6. = Keypad Buzzer Follow Mode

(06 follows the entry delay, door chime, auto-arm alert period and 24 hour buzzer loop)
