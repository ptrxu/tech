Eagle 1250EE Terminator

Specifications 9 to 14VDC

44mA standby (with one 2172 speaker/mic)

180mA active

210mA if used as siren

Programming 1. Short default pins for 3 seconds

2. Call panel with standard phone

3. 1250 should pick up (high/low tone)

4. Enter 5678 (3 beeps)

5. press * (1 beep)

6. Press # (1 beep)

7. Enter 1234 (3 beeps)

8. Enter slowly 22111112001234 (3 beeps)

10. Press * (1 beep) Steps 10 & 11 saves data

11. Press # (1 beep) To exit without saving press **

12. Press 9 to exit

Step 8 Data Underlined = Default

Digit ON OFF

1 1 = Direct Wired Trigger 2 = Auto Trip Enabled

2 1 = Enables required for Direct/Auto Trip 2 = Enables not required

3 1 = Trigger activates immediate Turn-On 2 = Trigger Activates 5 min. callback

4 1 = Auto Disconnect Enabled 2 = Auto Disconnect Disabled

5 1 = Enables Internal Siren Driver 2 = Disables Internal Siren Driver

6 1 = 1250 active on Multiple Alarms 2 = Multiple Alarms shut down 1250

7 1 = Must always be 1

8 1 = * accesses callback feature 2 = *55* accesses callback feature

9 1 to 9 = Rings before hangup 0 = Voice Message bypass disabled

10 1 to 9 = Rings before answer 0 = Disabled

11 Maintenance Code for Remote Access Default = 1234

12 User Access Code for Remote Access Default = 5678

(Maintenance Code cannot reprogram User Access Code)

(User Access Code cannot reprogram Maintenance Code)

LED - 1 Flash every 6 seconds = In Standby, all OK

2 Flashes every 6 seconds = 1 Ring Callback Mode waiting for Ring

3 Flashes every 6 seconds = 1 Ring Callback Mode, waiting for Acknowledgment code

4 Flashes every 6 seconds = Triggered, waiting for Station to send command

On Steady = Two Way Active

Ademco Vista panels must have *91 set to 4 if available
