Telular TelGuard TG-4 General Information

Information 1. TG-4 acts like a receiver and reads output from panel dialer

2. Can interpret 4-digit Contact ID, 4 or 6-digit SIA (see programming notes), 4/2 and 3/1 pulse

3. Will not interpret 10-digit Contact ID

4. Branded Honeywell/Ademco panels must be downloaded and changed to 4-digit account numbers if

using TelGuard as cellular backup

  1. Sends exactly the same signals as panel

  2. Can be configured to send to same phone and account as panel or a specific account and phone

(configured by web page)

  1. Billing issues go to Greg Nelson at gnelson@monitronics.com

  2. This will work with GE Simon 3 using SIA format, Simon option 32 (300 baud) must be ON

  3. TG-4 is picky about 4/2 formats, for DSC 1500/1550 use either format 01 or 09

  4. Linear DVS1200/2400/2408 pre v2.0 panels have problems sending Contact ID to the TG-1, this is due to

an incompatibility in older DVS dialers.

Programming (Usually required only if switching to SIA format)

  1. Connect handset or plug house phone into J7

  2. Connect power to the TG-1, handset/phone should sound 2 beeps

  3. Press # then * to enter program mode, handset/phone should sound 2 beeps

  4. Enter address below, handset/phone should sound 2 beeps

  5. Enter new 1 or 2 digit value, handset/phone should sound 2 beeps

  6. When finished exit programming with *, handset/phone should sound 2 beeps
