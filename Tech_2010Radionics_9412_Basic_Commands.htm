On commercial accounts the user may not have access to some commands listed below

Master Arm All COMMAND + 1 or User Code + ENT

Perimeter Instant COMMAND + 2 or COMMAND + 2 + User Code + ENT

Perimeter Delay COMMAND + 3 or COMMAND + 3 + User Code + ENT

Stay Arm COMMAND + 3 or COMMAND + 3 + User Code + ENT

Instant Arm COMMAND + 1 + 1

Partial Perimeter COMMAND 8 or COMMAND + 3 + User Code + ENT

Disarm User Code + ENT

Zone Status 1. Press MENU/ESC

(If enabled) 2. Display should show "Watch Mode ?"

3. Press NEXT/DOWN ARROW key until "View PT Status?"

  1. Press ENT to display area/partition zones (Example: AREA 1 7 = Partition 1, 7 zones)

  2. Press NEXT/DOWN ARROW to view first zone in system

  3. Press ENT to view that zones status

(Example: "1PT 2 NORML HW" = Partition 1, Zone 2, Closed, Hardwire Zone)

"2PT 1 OPEN HW" = Partition 2, Zone 1 Open, Hardwire Zone)

  1. Press NEXT/DOWN ARROW for next zone or MENU/ESC to exit

Silence Alarm User Code + ENT

Reset Alarm COMMAND + 4

Reset Smoke 1. Press COMMAND + 4 + 7 (Enter User Code + ENT if required)

2. Display will show "Sensors Resetting"

Alarm Memory COMMAND + 4 + 0

Change Partition 1. Press COMMAND + 5 + 0

(If enabled) 2. Enter User Code + ENT if required

3. Keypad will ask "Enter AREA"

4. Enter Partition 1 to 8 + ENT key

If code doesn't have authority "NO AUTHORITY" will display

Bypass 1. Press COMMAND + 0 (Enter User Code + ENT if required)

(If enabled) 2. Enter User Code if required

3. Enter Zone to bypass (1 to 246) + ENT

4. Press ENT again to confirm bypass and exit

NOT BYPASSABLE = Bypassing not allowed or arming not allowed if this zone/point is open

TOO MANY BYPASSED = Maximum number of zone allowed to be bypassed has been reached

TOO MANY FAULTED = Maximum number of open zones allowed has been reached

NOT AUTHORIZED = User is not authorized to bypass zones

Force Bypass (If zone is open)

(If enabled) 1. Enter User Code + ENT key (Display should show "CHK AREA 1")

2. Press NEXT to scroll through each open zone

3. Press COMMAND (Display should show "FORCE ARM")

4. Press ENT key

System will arm with open zones bypassed

Un-Bypass 1. Press COMMAND + 0 + 0

(If enabled) 2. Enter User Code + ENT if required

3. Enter Zone to un-bypass (1 to 246)

4. Press ENT key (Zone status light should flash)

Door Chimes COMMAND + 6 to turn on/off

(If enabled) or

  1. Press MENU/ESC key

  2. Display should show "Watch Mode ?"

  3. Press ENTER

(Display should show "Watch Mode is now ON" or "OFF"

Phone Test 1. Press COMMAND + 4 + 1 or COMMAND + 4 + 2

(If enabled) 2. If needed enter User Code + ENT

3. Display will show "Sending Report"

  1. Failure will display "SERVC COMM FAIL"

Sensor Test 1. Press COMMAND 4 + 4

(If enabled) 2. Enter User Code + ENT if required

3. Keypad beeps twice if sensor opened

4. End by pressing any number key

System Test 1. Press COMMAND + 5 + 8

(If enabled) 2. Enter User Code + ENT if required

3. Siren will sound

Keypad will go dead for 4 minutes

4. Activate fire devices, keypad will beep

  1. End by pressing any number key

Set Time/Date 1. Press COMMAND + 4 + 5

(If enabled) 2. If needed enter User Code + ENT (Display should show "ENT TIME")

3. Enter 4-digit time + ENT (Display should show "AM ?")

4. If AM press ENT or NEXT for "PM ?" then ENT (Display should show "ENT DATE")

5. Enter new 6-digit Month, Day, Year + ENT (Example July 2 2007 would be 070207)

Display Version 1. COMMAND + 59

(If enabled) 2. Keypad will display "D9412 REV x-xx"

3. After 5 seconds will exit
