Enter Program 1. While powered up plug in Programmer Keypad

  1. Enable Programmer Keypad by entering 8 + Installer Code + 02

  2. Press 8 + Installer Code + 00, keypad says "SYSTEM PROGRAMMING"

(Do not power up with Programmer connected to avoid Bus Errors later)

Move to Location Enter 4 or 5 digit "Shortcut" (you can't do this while IN a location)

or press A and B to move through options (this is the slow way to move around)

Press # to enter a location

Press # to save data within a location

Press * to exit a location

Exit Program 1. Press * until "SYSTEM PROGRAMMING" appears

2. Press A or B keys until Keypad shows "EXIT PROGRAMMING" press #

3. Remove Programmer Keypad from connector on circuit board

(Do not power up system with Programmer keypad attached!)

Enable 1. Press 9 + Master Code ("SYSTEM MENU" then "TIME/DATE")

Downloading 2. Press 2 + 0

3. Press 2 + # ("Display shows "DOWNLOADING ON")

  1. Press * + 4 + #

Sensor Test 1. Press 8 + Master Code + 3

2. Keypad shows "ARMED TO SENSOR TEST" and beeps every 60 seconds

3. Open Door or Window, move by motion

Good wireless sensor test will beep keypad and show "OK"

Bad wireless sensor test will sound long low-pitched beep and not show as tested

4. Press * to view any untested sensors

  1. Press 1 + Master Code to end

(Wireless motions require no movement for 4 minutes prior to testing)

"Call For Service" Unknown trouble, to remove see below

  1. Press D key & 6 at same time

  2. Press # + 250 + # + *

Send to 8 + Master or Installer Code + 7 + 0

Downloader

Scan Bus Devices 8 + Installer Code + 01 (Do not do this if programmer is attached)

Default Panel 1. Enter programming mode

2. At "SYSTEM PROGRAMMING" press A or B keys until "CLEAR MEMORY" then press #

  1. At "ENTER CODE TO CLEAR MEMORY" enter Installer or Dealer Code then #

  2. Panel will exit programming and default to factory (will not default if phone lock is enabled)

OR

Short 2 solder points on main board for 20 seconds (will not default if phone lock is enabled)

Delete Unit ID (If Programmer is accidentally learned into system see bottom of program sheet)
