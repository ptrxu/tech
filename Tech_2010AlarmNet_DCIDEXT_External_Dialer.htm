AlarmNet DCID-EXT Dialer Capture Module

Information 1. Provides primary reporting with Telco dialer capture, designed for any panel using Contact ID.

2. Accepts 4 or 10-digit Contact ID however AlarmNet will only send 4-digit Contact ID.

3. Program Zone as 24 hour trouble.

4. Zone is triggered if 7845GSM detects a fault or if DCID-EXT tamper is activated.

5. Honeywell/Ademco panels must delete field *40 and remove any pauses in phone number.

6. Use either 2-wire or RJ-45 jack for phone connections.

7. DCID-EXT module will not provide dial tone in the event of a 7845GSM fault.

8. Current use: 40mA standby, 80mA active, Input Voltage 9 to 15VDC.
