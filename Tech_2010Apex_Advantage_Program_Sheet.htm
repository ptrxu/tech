Zone Voice Programming

To Program

1. Press 920, keypad will say "Enter Zone" enter 2-digit zone number

2. Enter 3-digit code for Word Definition, keypad will repeat it

3. Enter next word, maximum of 4 words, if less than 4 end with H or # key to save and exit

To Hear Descriptions

Press 930, keypad will say "Enter Zone" enter 2-digit zone number

Keypad will speak current Voice Description, end with H or * key
