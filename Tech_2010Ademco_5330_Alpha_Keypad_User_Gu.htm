Ademco 5330 Alpha Keypad User Guide

Display Adjust Open side door, use screwdriver to adjust viewing angle higher or lower

Brightness and contrast cannot be adjusted

Keypad Help Each function key has a built-in help function, to display hold key down 5 seconds

Example: Holding 1 (Off) key will display the following screens, release key to return to normal

Display Zones Press # + 3 to show main options screen

Press # twice to show 1st zone description

Press C key (3rd function key) for next zone description

Press D + C keys (3rd and 4th function keys) for previous zone description

Press D + # three times to exit
