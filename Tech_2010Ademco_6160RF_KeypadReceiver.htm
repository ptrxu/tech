Ademco 6160RF/First Alert FA570RF Keypad/Receiver

Keypad will operate as 5881H Receiver

  • Warning** Pre-v3 6160RF keypads would cause panel to send sporadic panic alarms.

V3 6160RF's will say "6160RF-3" on back of keypad

Information Will not work with 4110, 4140, 4150, 4160, 4180 and Vector panels

Keypad operates as 5881H receiver with up to 64 zone capacity

Contains 5800TM module

Sensors cannot be closer than 18 inches to keypad

6160RF has no relay or tamper

Supports Encrypted keyfobs

Keyfobs cannot be learned into this receiver like the 6150RF

Backlight times out after 40 seconds

Power Ratings 80mA (Backlight On, Sounder On)

50mA (Backlight On, Sounder Off)

35mA (Backlight Off, Sounder Off)

Enter Program 1. Press 1 and 3 together for 5 seconds within 60 seconds of power up

2. Programming begins at "Keypad Address", * saves and moves to next location

Exit Program Press * at end of last program location

Location Display Options Default Description

  1. Keypad Address "CON ADDRESS = XX" 01 - 31 31 Enter keypad address then * for next

Vista-10SE/Vista-20SE = 31

Vista-40 = 01-15

Vista-50P and above = 01-30

Vista-15P/20P = 16 to 23

  1. Receiver Enable "RECEIVER ON" 0 or 1 0 Enter option then * for next

0 = Local receiver

  1. = Panel receiver

3. Receiver Address "REC ADDRESS = XX" 00 - 31 00 Enter receiver address then * for next

Vista-10SE/Vista-20SE = 00

Vista-10P/15P/20P = 00

Vista-40 = 01-15; Vista-50P and above = 01-30

4. House ID "House ID = XX" 00 - 31 10 Enter house ID then * for next

(Appears only if receiver is set for local) 00 = Disabled

5. High Security Mode "HIGH SECURITY 0 - 1 0 Enter option then * for next

OFF [ 1 = ON]"

0 = Standard and Encrypted sensors

  1. = Encrypted sensors only

6. Disable High Security "DISABLE HS DEV?" 0 - 1 0 Enter option then * for next

Devices "NO [ 1 = YES]
