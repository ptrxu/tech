ITI 60-788 Talking RF Keypad

Information Simon 2 only, does not work on Simon 3

Will only announce open sensors after pressing SYSTEM STATUS

Will not sound alarm siren or status beeps

When panel is talking touchpad is silent, when touchpad is talking panel is silent

3 AAA batteries

Mount within 100 ft of panel, may not work if mounted too close

If "Module 1" or "Module 2" failure install 47k Resistors across Output terminals

Programming 1. Enable as unsupervised sensor type 01, 03, 06, 07 (Use 01 for audible panic, 03 for silent panic)

  1. When prompted to learn press and hold both EMERGENCY buttons, keypad should say "OK"
