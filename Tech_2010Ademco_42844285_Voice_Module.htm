Ademco 4284/4285 Voice Module

Information Power 300mA if using speaker, 220mA without

This devices always occupies address 4

Will not work properly if panel does not have true line seizure

Local Access 1. Pick up any home telephone

2. Enter 2-digit "phone code", can be any number from 1 to 9 + * or #. (see panel programming)

3. Voice prompt will ask "Enter Security Code Now"

4. Enter 4-digit user code

Remote Access 1. Dial home phone

2. If answering machine or person picks up phone enter Phone Code programmed into panel within 20 seconds

3. Panel will ask "Enter System Code Now"

4. Enter User Code, module will speak status and end with 2-tone chime

To perform system commands enter 1-digit command, panel will give 1 to 3-tone chime if successful

  1. = Off/Disarm

  2. = Arm Away

  3. = Arm Stay

  4. = Arm Maximum

  5. = Test mode (only available with local access)

  6. = Bypass Zone

  7. = Instant Arm

  8. = User Code Programming

  9. = Chime On/Off

  • = Ready, speak status

# = Quick Arm

Example: to arm Away press 2, to Disarm press 1

5. To end hang up phone

Enable Access Press # + 9 + 1 from local phone or keypad to toggle remote access on/off

Panel Program Program 2-digit "phone code" for phone access

Location *26 if Vista-10SE/20SE/Via-30PSE

Location *28 if Vista-10P/15P/20P

Location 1*66 if 4140XMP

Location *20 if Vista-32FB to Vista-250FBP

(1st digit = 1 to 9, 2nd digit = * (# + 11) or # (# + 12)
