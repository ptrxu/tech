DSC PC5108 Zone Expander

Compatibility PC1616 (1 supported)

PC1832 (up to 3 supported

PC1864 (up to 7 supported)

PC5010 (up to 3 supported

PC5015 (up to 3 supported

PC5020 (up to 7 supported

Not compatible with any version of PC1555

Information None, set proper dipswitches then enable supervision in 902

V2 and below panels PC5108 acts as 2 blocks of 4 zones (use PC5108 v1 jumper settings)

V3 and above panels PC5108 acts as a single block of 8 zones (use PC5108 v2 jumper settings)

Example: if using with PC5010 v1 use v1 jumper settings, for PC5020 v3.2 use v2 jumper settings)

Troubles 1. Tamper not jumpered to keypad black (KP shows Tamper trouble)

2. Zone Expander interfering with wireless zones

3. Jumpers set incorrectly
