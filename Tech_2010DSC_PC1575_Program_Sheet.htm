DSC PC1575 Program Sheet Binary Programming

Report Format SIA, Contact ID, 4/2, 3/1

Compatibility

Power Ratings 550mA Aux output

700mA Siren output

50mA PGM 1 output

300mA PGM 2 output or 50mA PGM2 output on 50-50 models

Street Smart 1. Connect power (Black & Red) and keypad wires (Yellow & Green) like normal

Configuration 2. Connect Brown wire to PGM2 with inline 2.2k, connect Brown/White Wire to Aux +

3. Program PGM2 for momentary keyswitch (26)

Enter Program * 8 1575 (READY, ARMED and TROUBLE Lights should flash)

Exit Program #

Move to location Enter 2-digit location, some areas may require # to exit

Default Panel 1. Connect wire from Zone 1 to PGM1

2. Power up panel, wait 10 seconds

3. If Zone 1 turns ON panel is defaulted, if not panel is locked

Enable Download * + 6 + Master Code + 5 (Enables for 1 hour)

(System also enables downloading for 1 hour if powered down then up)

Hex Program A = * 1 * B = * 2 * C = * 3 * D = * 4 * E = * 5 * F = * 6 *

If combining press * at end of hex string (example BC would be *23*
