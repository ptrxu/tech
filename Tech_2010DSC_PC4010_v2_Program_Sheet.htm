DSC PC4010 v1 to v2 Program Sheet

How To Identify Has sticker that shows version

Power Ratings 500mA Aux output

300mA Switched Aux output (Fire)

700mA Siren output

50mA PGM output

Report Format SIA, Contact ID, 4/2, 3/1, SIA preferred (anything below v3.0 will NOT Auto Report SIA)

Address Keypad On power-up 1st keypad will display "Press Any Key To Continue"

Press any key which addresses the keypad, now use this keypad to

enter programming and address other keypads

Enter Program 1. Press * 8 4010 (v3.3 and above use 5555)

2. Keypad should show

(If keypad displays "Select 00" panel is v3.0 or higher)

Exit Program Press # at "System Area" prompt

Move to location Enter 1 or 2-digit location or press < > to move to location then *

# exits a location (the data entered is automatically saved) and exits to previous menu

  • toggles Y/N or moves through options within location, # saves and exits to previous menu

Default Panel You don't really want to do this

Enable Download * + 6 + Master Code + 5 (Enables for 6 hours)

(System also enables downloading for 1 hour if powered down then up)

Hex Program A = * 1 B = * 2 C = * 3 D = * 4 E = * 5 F = * 6 (Do NOT use * afterwards)

0 Enroll Module

  1. Enter this section from "0 System Area" by pressing 2 0 then *

  2. Scroll to type of module to enroll with <> (4500 Keypad, 4164 Wireless Receiver etc) then *

  3. Enroll device by pressing any key on Keypads, or press Tamper switch on modules

  4. Keypad will display device enrolled and assignment Example: "LCD4500 02 Enrolled"

  5. Press # to exit. To verify enrollment go to 2 2, press key or tamper on device, it will

display along with assignment on keypad
