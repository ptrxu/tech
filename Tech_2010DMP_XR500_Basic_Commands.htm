DMP XR-500 Basic Commands

Zones 574 hardwire or wireless

Battery 12 volt rechargeable Gel-Cell inside main panel

AC Transformer is 16.5 volts, screwed into outlet somewhere in house

Wireless Yes but usually not installed

Downloadable Yes

Panic Buttons Each pair of SELECT keys is a panic

Overview Use SELECT keys under display to enable

Arm Away LCD Keypad: Enter User Code then press SELECT button under ALL

LED Keypad: Enter User Code

Arm Stay LCD Keypad: Enter User Code then press SELECT button under PERIM

LED Keypad: Press STAY key

Arm Instant LCD Keypad: Press SELECT button under INSTNT display after arming in Stay or Away

LED Keypad: Press INSTANT key before arming

Force Arm 1. Open Zone or Zones

2. Arm system in Away or Stay

3. Display shows OKAY BYPASS STOP

4. Press SELECT button under BYPASS, if LED keypad hold last digit of User Code for 2 seconds

OKAY force arms open zone

BYPASS bypasses open zone

STOP stops arming process

Quick Arm Press COMMAND + 1

Disarming Enter User Code + COMMAND

Silence Alarm LCD Keypads 1. Enter User Code

2. Display shows DISARM SILENCE

3. Press SELECT key under DISARM

LED Keypads Enter User Code

Door Chimes Perimeter Zones: Press COMMAND + 6 + User Code + STAY key

Single Zone only: COMMAND + 6 + User Code + 3-digit Zone number

All Zones: COMMAND + 6 + User Code + SPECIAL key

Reset Smoke Press COMMAND + 4 + 7

Siren Test Press COMMAND + 4 + 1
