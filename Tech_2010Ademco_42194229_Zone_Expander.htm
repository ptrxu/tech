Ademco 4219/4229 Zone Expander Module

Information 4219 = No relay output

4229 = 2 relay outputs (2A max current 28VDC/VAC)

Power = 30mA standby, 100mA relays active (4229)

Carbon monoxide detectors cannot be wired to these expanders

Smoke detectors can be used but a relay output from 4204, or 4229 or panel must be used to reset smoke

Compatibility 4110XMP (Address 1)

4120XM (Address 2)

Vista-20SE (Address 1)

Via-30P/PSE (Address 1)

Vista-15P/20P (Address 7 to 8 for 15P, address 7 to 11 for 20P)

Cannot be used on Vista32FB or higher panels, use 4208U or 4209U instead

If zone doubling Vista-20P address 07 cannot be used, set 4219/4229 to address 08, zones start at 16
