Move to location - Location + # * Saves # Moves to next Location

(This is a Moose Z950)

There are 2 sections of programming, User and Installer sections

Installer Program (Locations 024 to 023)

  1. Hold 9 key 3 seconds until keypad beeps

  2. Enter Installer Code (9 6 3 2 1) (Keypad should flash)

OR (If Installer Code was changed)

  1. Power down then power up

  2. Hold 3 and 9 keys for 3 seconds (Keypad should flash)

(Must exit Installer Program then enter User Program to add Codes, change Entry/Exit & Siren times)

User Program (Locations 001 to 033)

  1. Hold 9 key 3 seconds until keypad beeps

  2. Enter Master Code (Keypad should flash)

Go to Location 1, 2 or 3 digit Location then #

Program Location After each digit entered press * to save then # to move to next location

(To program Account 1224 press 060# to enter location then 1*# 2*# 2*# 4*#

Exit Program Press * twice

Default Panel Enter programming then press 3 3 3 + # + 1 + *
