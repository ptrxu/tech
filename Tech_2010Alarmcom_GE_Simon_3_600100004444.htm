Alarm.com GE Simon 3 600-1048 GSM Module

Compatibility Works with Simon 3 v3.5 and above

Information If Phone Lock (Option 8) is ON Alarm.com cannot change phone and account numbers

If Secure Arming (Option 60) is ON Alarm.com cannot remotely arm system

Yellow SIM card = AT&T

White SIM card = T-Mobile

Registration 1. Open front cover, enter Installer code

2. Press OPTION butt and wait for panel to say "Option 1"

3. Close cover, panel will beep

4. Wait until panel beeps again (up to 15 seconds)

5. Press and hold 9/0 key until panel says "Remote Phone Test"

"Module OK" Low panel battery, this causes the Alarm.com module to go in and out of supervision

repeating

Mode Notes Module functions under 3 possible modes, troubleshooting should be done in Idle mode

  1. Idle = Normal, AC OK, battery above 6 volts, module is not currently in communication

  2. PowerSave = Module has just powered up or AC power is down or battery is less than 6 volts

Pressing 9/0 key for 5 to 10 seconds will switch from Connected to Idle mode

  1. Connected = Connected to Alarm.com servers, stays connected for 6 minutes after each message

Pressing 9/0 key for 5 to 10 seconds will switch from Connected to Idle mode

Diagnostic LEDs These readings depend on what mode module is in (Idle, PowerSave or Connected)

Because of the way the module is mounted the LED's are on the other side of circuit board

LED 1 (Red) Error message in Idle and Connected mode, disabled in PowerSave mode

  1. flash = Module cannot communicate with panel

  2. flashes = SIM card missing (check SIM card holder)

  3. flashes = Error registering with GSM network

  4. flashes = Registered with GSM network but cannot communicate with Alarm.com

  5. flashes = Radio portion of module not working correctly

  6. flashes = Module is fixing an unusual condition with connection to GSM network

LED 2 (Yellow) Indicates communication between module and panel in all modes, flashes every

2 seconds in Idle mode or 4 seconds in PowerSave mode

LED 3 (Green) Communication between module and its radio unit and Alarm.com servers,

flashes in unison with LED 2 in PowerSave mode

LED 4 (Green) Signal Strength, flashes 1 to 5 for lowest to highest in Idle mode

No flashes = In PowerSave mode

Just exited PowerSave mode

There is no GSM coverage in the area

2 seconds ON then OFF = in Connected mode

(Pressing 9/0 key for 5 to 10 seconds will switch from Connected to Idle mode)

Number Keys Press 5 to 10 seconds for Alarm.com module information

  1. 2 Key = 10-digit module serial number

3/4 Key = 4-digit firmware version

5/6 Key = 15-digit SIM card information

7/8 Key = Lists types of reports sent to central station

  1. = Manual and Automatic Phone Tests

5 = Burg and Fire Alarms

7 = Sensor Troubles

8 = Opening/Closings

10 = Sensor Bypasses

11 = AC Failure

13 = Panel programming report

14 = Tampers

15 = Alarm cancels

16 = Activity events (sensor opening/closings, motion activations)

18 = Alive signals

40 = Panel low battery

9/0 Key = Signal Strength (0 to 6) + Module status or error

I = Idle

C = Connected

O = Power Save

L = Connecting

A = Disconnecting

M = SIM card is missing

G = Cannot register module with GSM network

K = Cannot connect to Alarm.com

D = Radio not working correctly

Lights OFF = Reports

Remote Access OK = Able to communicate to Alarm.com

No Remote Access = Unable to communicate to wireless network

Module Failure 1 = Not registered with wireless network yet

Module Failure 2 = Registered to GSM network but signal strength is too weak

Module Failure 3 = Registered and able to send signals but not always

(Low signal strength)

Lights ON = Battery strength ("Battery 653" = 6.53 volts)
