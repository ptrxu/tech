Paradox Esprit 738 Program Sheet

Enter Program Press ENTER + 737373 (or 383838) (ENTER key should flash)

How To Program 1. Enter 3-digit location

2. Keypad displays 2-digit data saved in location

3. Enter 2 to 3-digit data

4. Press ENTER to save and exit location

Exit Program Press CLEAR

Program Keys 2ND = Clears Location

TRBL = F (15) 0 = A (10)

MEM = E (14) 11 = B

BYP = D (13) 12 = C
