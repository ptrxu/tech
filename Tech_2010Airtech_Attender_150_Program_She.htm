INSTALLER Program (Locations 034 to 410)

  1. Press Program Button on main panel

  2. At keypad press 9 + Program Code (9 8 7 6 5)

or

  1. Power down with black switch on bottom left of panel

  2. Power up while holding Program Button on main panel

Move to Location Enter location + #

Enter Data Enter 1 to 3 digits in location + * to save + # to move to next location

Exit Installer Program Press * twice

(Must exit Installer Program and enter User Program to add Codes & Entry/Exit/Siren times)

Enter USER Program (Locations 001 to 033) Press 9 + Master Code (Default is 1245)

Move to Location Enter location + #

Enter Data User Code = Enter entire User Code, system automatically exits

Locations 019 to 033 = Press * to save + # to move to next location

Exit User Program Press *
