Napco MA3000 Basic Commands

Zones 96 hardwired

Battery 12 volt rechargeable Gel-Cell inside main panel

AC Transformer is 16-volts plugged in somewhere in house

Wireless No

Downloadable Yes

Panic Buttons Fire Panic Press 7/F + * (Sends E110/FA Fire Panic)

Auxiliary Alarm Press 8/A + * (Sends E140/QA General/Emergency Alarm)

Police Panic Press 9/P + * (Sends E123/PA Audible Panic)

Arm Away User Code + ON/OFF (Armed light will turn ON)

Arm Stay INTERIOR + User Code + ON/OFF (Interior light will turn ON)

Arm Partitions 1. Press 1 to 8 for Partition 1 to 8 (Selects Partition to Arm)

(If Partitioned) 2. Press * + ON/OFF (This move you to that partition)

  1. Press User Code + ON/OFF to arm (User Code must have authority in partition)

Arm All Press 9 + * + User Code + ON/OFF (Arms all Partitions)

(If Partitioned) (User Code must have authority in partition)

Instant Arm User Code + ON/OFF + INSTANT (Armed light will flash rapidly)

Quick Arm Press ON/OFF (If Enabled)

Disarm User Code + ON/OFF (Press RESET to clear mistakes)

(Use 0 + * + User Code + ON/OFF to disarm all Partitions)

Silence Alarm User Code + ON/OFF + RESET

Reset Smoke User Code + ON/OFF + RESET twice

Bypass BYPASS + Zone (01 to 96) (Bypassed zone will display)

or

User Code + BYPASS + Zone (01 to 96)

Siren Test User Code + Function

Press INSTANT until "Bell Test" appears

Press INTERIOR to sound siren for 2 seconds

Set Time System clock can only be set by installer or by downloading

Door Chime 1. Enter User Code then press FUNCTION to enter Program Menu

2. Press INSTANT until "Activate Chime ?"

3. Press INTERIOR to Enable or Disable

  1. Press RESET to exit

Siren Test 1. Enter User Code then press FUNCTION to enter Program Menu

2. Press INSTANT until "Activate Bell Test?" appears

3. Press INTERIOR to enable and sound siren for 3 seconds

4. Press RESET to exit

Phone Test 1. Enter User Code then press FUNCTION to enter Program Menu

2. Press INSTANT until "Activate Telephone Test?" appears

3. Press INTERIOR to enable Phone Test, if Fail a trouble "3" will appear

4. Press RESET to exit

Sensor Test Arm system and activate sensors

(Wireless motions require no movement for 4 minutes prior to testing)

User Functions (This allows viewing of open Zones, Alarm History, programming of User Codes)

(Some of these options may not be viewable by customer)

  1. Press FUNCTION to enter User Menu

  2. FUNCTION again scrolls forward through options, BYPASS scrolls backwards

Press [BYPASS] to scroll backwards, [ON/OFF] to select

Press RESET or scroll to ******EXIT****** to exit

DISPLAY BYPASS? - Scrolls through all bypassed zones

DISPLAY ZONEDIRECTORY? - Scrolls through all zone descriptions

DSIPLAY FIRE TROUBLE? - Displays Fire Zones in Trouble

DISPLAY OPEN/CLOSE SCHEDULE - Displays Arming/Disarming schedules

ACTIVATE OVERVIEW? - Displays total system status of all Partitions

ACTIVATE CHIME? - Enables/Disables Door Chime

ACTIVATE WATCH MODE? - Activates all Day Zones (W displays)

ACTIVATE GUARD TOUR? - Commercial Applications only

RESET SMOKE DEVICES? - Resets Smoke Detectors

SILENCE FIRE BELL?

RESET RELAY FIRE AUX 1?

RESET RELAY FIRE AUX 2?

RESET SYSTEM TROUBLES?

RESET SENSOR MESSAGE? - Resets "Sensor Failure" Message

START EXIT TIME?

ACTIVATE DIALER TEST? - Tests Phone Communications

DISPLAY ALARM LOG? - Displays Alarm History

Use NEXT or PRIOR to scroll events

DISPLAY TOTAL LOG? - Displays Alarms, Open/Closes etc

Use NEXT or PRIOR to scroll events

DISPLAY FIRE LOG? - Displays Fire Alarm History

Use NEXT or PRIOR to scroll events

DISPLAY OPEN/CLOSE LOG? - Displays Arm/Disarm History

Use NEXT or PRIOR to scroll events

DISPLAY SYSTEM LOG? - Displays Troubles and other events

Use NEXT or PRIOR to scroll events

2 for 2 Hours etc)

ACTIVATE PROGRAM? - Enters User Programming
