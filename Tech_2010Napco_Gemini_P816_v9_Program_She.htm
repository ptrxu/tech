Napco Gemini P816 v8 Advanced Program Sheet

System can be programmed by "Easy Menu" (Click below) or "Advanced" (use the below program sheet)

Programming of system can only be done by Keypad on Address 1

Power Ratings 500mA Aux output with 16.5V 20VA transformer

650mA Aux output with 16.5V 40VA transformer

2A Siren output

5mA PGM output

Overview Programming is divided into 2 sections

  1. Easy Menu programming (Programs the basics, click above links)

or

  1. Direct Address Programming (Programs everything, shown below on this page)

Enter Program 1. Installer Code + FUNCTION (Default is 456789)

2. Press INSTANT until

"Activate Program Y/N" (Alpha Keypad)

"TURNON PROG Y/N" (Fixed English LCD Keypad)

"17" (Standard 2-digit display Keypad)

  1. Press INTERIOR to enter "Easy Menu" Programming (See above "Quick Programming")

  2. or press RESET to enter "Advanced" Programming (see below "Direct Programming")

Go to Location 1. Enter 4-digit location (Current value displays)

2. FUNCTION toggles 4-digit location and 2-digit data

3. Enter new data (New value displays)

(0 Blanks out a location)

4. Press ON/OFF to save (Display moves back to location)

  1. Press * + INTERIOR for next location or enter new 4-digit location

Exit Program Press RESET twice

Programming INTERIOR moves forwards

Keys INSTANT moves backwards

FUNCTION toggles between 4-digit location and 2-digit data

ON/OFF saves

RESET twice exits

0 Blanks a location (unless otherwise noted)

  • + 0 = 0

  • + 1 = B

  • + 2 = C

  • + 3 = D

  • + 4 = E

  • + 5 = F
