Scantronics/Arrowhead Spartan 6 Program Sheet Click HERE for Programming Keys

Programming can only be done by Arrowhead P4000 Chip programmer or at the keypad (some versions)

Most of the time the Installer Code is never programmed so keypad programming won't work

Enter Program Installer Code + * + 8 + # + 12345 (Status Light should turn ON)

(default Installer Code is 2345 but may not be programmed from factory)

Exit Program Press * and # at same time

2 Modes of programming are needed, Memory Location and Location Programming

Press * to toggle between modes, keypad will give constant beep with Location Programming

Example Program Account Number to 1234

Start in Memory Location Mode (Keypad is not beeping)

  1. Press 43 # (LED shows STATUS, Zone 1, 3 and 5 ON)

  2. Press * key to enter Location (Keypad begins const beep)

  3. Press 1 # 2 # 3 # 4 #

  4. Press * key to exit Location (Keypad stops constant beep)

(from here you can then go to any other location)
