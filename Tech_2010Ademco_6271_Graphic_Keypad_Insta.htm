Ademco 6271/First Alert FA701KP Graphic Keypad Installation Guide

Panel Info This keypad should not be only keypad on system, it is recommended to also install a standard keypad

The 6271 will always reboot after exiting installer programming , this is normal

Versions 6271/FA701KP = Monochrome

6271V/FA701VKP = Monochrome with Voice

6271C/FA701CKP = Color

6271CV/FA701CVKP = Color with Voice

Keypad keeps rebooting = Not enough power

One keypad should run okay but 2 usually requires an external power supply

Installation 1. Enable 6271 in panel programming (Vista-15P/20P section *189 enter 10 for 1 6271 or 11 if 2 6271's

Single 6271 2. Power up 6271 like normal

Installation 1. Enable 6271 in panel programming (Vista-15P/20P section *189 enter 10 for 1 6271 or 11 if 2 6271's

Multiple 6271's 2. Power up one keypad

3. At main menu press SECURITY

4. Press MORE CHOICES

5. Press SETUP

6. Press ADVANCE SETUP

7. Enter panel Installer code, if this does not work try 4140

8. Press CENTRAL STATION

9. Press ECP ADDRESS (each keypad defaults to address 1)

10. Press UP or DOWN arrow to select ECP address 2 then OK

11. Press Keypad may reboot, if not press HOME to return to main screen

12. Power panel down, connect 2nd keypad and power up panel

"ECP Error" Incompatible panel, 6271 ECP address not configured correctly or *189 not enabled in control panel
