Enter Program Press CODE key + * + 2468 + 1

Options Code + * + Installer Code + 1 = Enter Programming

Code + * + Installer Code + 2 = Not Used

Code + * + Installer Code + 3 = Unattended Download (send to downloader)

Code + * + Installer Code + 4 = Online Download (answer downloading)

Code + * + Installer Code + 5 = Walk Test Mode (reduced sensitivity)

Move to location Press * + 2 digit location while anywhere in programming

Program location Press # after each digit and move to next location

Example: To program phone number enter 1#8#0#0# etc

Exit Program Press STAY

Default Panel 1. Short "EEPROM RESET" pins on panel and power up

OR

2. Press CODE key + * + 2468 + 1 and hold 1 & 3 at same time

Zone Page if 4/2

Hex Program CODE+1 = A CODE+2 = B CODE+3 = C CODE+4 = D CODE+5 = E CODE+6 = F
